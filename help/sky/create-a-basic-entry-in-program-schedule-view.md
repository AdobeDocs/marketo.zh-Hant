# 在「方案計畫」視圖中建立基本條目

<br> 

您可以直接從程式計畫視圖建立新的基本條目。

1. 在樹中選擇程式，然後轉到&#x200B;**[!UICONTROL Overview]**&#x200B;下的&#x200B;**[!UICONTROL Schedule]**&#x200B;頁籤。

   ![影像一](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-1.png)

1. 將滑鼠指標暫留在您要新增項目的日期上，然後按一下&#x200B;**[!UICONTROL Add]**。

   ![影像2](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-2.png)

   >[!NOTE]
   >
   >您也可以依照類似步驟從「日」、「周」或「議程」檢視建立基本項目。

1. 輸入新基本條目的名稱，然後根據需要將&#x200B;**[!UICONTROL Type]**&#x200B;更改為[!UICONTROL Basic Entry]。 您也可以修改&#x200B;**[!UICONTROL Start & End Date]**，如果您想在建立條目後留在日曆中，則取消選中&#x200B;**[!UICONTROL Open In Agenda View]**。 按一下&#x200B;**[!UICONTROL Save]**。

   ![影像三](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-3.png)

1. 您現在可以在方案排程檢視中找到您的項目。 將滑鼠指標暫留在畫面上，以取得詳細資訊……

   ![影像4](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-4.png)

...或按一下以開啟編輯器窗口。

![影像5](/help/sky/assets/program-schedule-view/create-a-basic-entry-in-program-schedule-view/create-a-basic-entry-in-program-schedule-view-5.png)
