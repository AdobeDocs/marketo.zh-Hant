# 在「程式計畫」視圖中編輯條目

<br> 

1. 按兩下您要編輯的項目。 這可從任何日曆檢視進行。

   ![影像一](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-1.png)

1. 在快顯視窗中，您可以編輯項目的不同元素。 如果參加項目是智慧型促銷活動或電子郵件方案，請按一下參加項目標題以開啟促銷活動／方案概觀。

   ![影像2](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-2.png)

>[!NOTE]
>
>基本條目的名稱一旦建立就不能更改。

>[!NOTE]
>
>已執行的智慧型促銷活動和電子郵件程式項目無法移至過去。
