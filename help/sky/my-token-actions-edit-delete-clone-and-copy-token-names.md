# 我的代號動作：編輯、刪除、複製Token名稱

## 編輯Token

在促銷活動資料夾或程式群的[!UICONTROL My Tokens]標籤中，按一下您要編輯的Token的&#x200B;**[!UICONTROL Value]**。 這將開啟編輯器窗口。

![影像一](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-1.png)

>[!NOTE]
>
>文字Token沒有個別的編輯器視窗。 按一下&#x200B;**[!UICONTROL Value]**&#x200B;進行內嵌編輯。

![影像2](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-2.png)

## 刪除Token

1. 按一下右鍵標籤，然後選擇&#x200B;**[!UICONTROL Delete]**。

   ![影像三](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-3.png)

1. 按一下&#x200B;**[!UICONTROL Confirm]**。

   ![影像4](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-4.png)

>[!NOTE]
>
>無法刪除資產中目前使用的Token。 如果您嘗試刪除這些標籤，將會收到錯誤訊息，列出標籤的使用位置。 從這些資產移除Token，然後再次嘗試刪除它。

## 複製Token

1. 若要複製我的Token，請以滑鼠右鍵按一下它，然後選取&#x200B;**[!UICONTROL Clone]**。

   ![影像5](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-5.png)

1. 輸入克隆標籤的新名稱，視需要進行編輯，然後按一下&#x200B;**[!UICONTROL Save]**。

   ![影像6](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-6.png)

## 複製Token名稱

若要將Token名稱複製至剪貼簿（例如`{{my.Event Calendar File}}`），請在Token上按一下滑鼠右鍵，然後選取「複製Token名稱」**。**

![影像7](/help/sky/assets/my-tokens/my-token-actions-edit-delete-clone-and-copy-token-names/my-token-actions-edit-delete-clone-and-copy-token-names-7.png)
