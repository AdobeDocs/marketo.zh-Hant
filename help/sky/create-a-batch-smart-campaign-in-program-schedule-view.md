# 在方案計畫視圖中建立批智慧型促銷活動

<br> 

您可以直接在日曆中建立新的批次智慧型促銷活動。

>[!NOTE]
>
>在方案排程檢視中只能建立批次智慧型促銷活動。 不支援觸發促銷活動。

1. 在樹中選擇程式，然後轉到&#x200B;**[!UICONTROL Overview]**&#x200B;下的&#x200B;**[!UICONTROL Schedule]**&#x200B;頁籤。

   ![影像一](/help/sky/assets/program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view-1.png)

1. 將滑鼠指標暫留在您要新增項目的日期上，然後按一下&#x200B;**[!UICONTROL Add]**。

   ![影像2](/help/sky/assets/program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view-2.png)

   >[!NOTE]
   >
   >您也可以依照類似步驟，從「日」、「周」或「議程」檢視建立批次智慧型促銷活動。

1. 輸入新智慧型促銷活動的名稱，然後將&#x200B;**[!UICONTROL Type]**&#x200B;變更為[!UICONTROL Smart Campaign]。 您也可以修改&#x200B;**[!UICONTROL Start Date]**，如果您想在建立條目後留在日曆中，則取消選中&#x200B;**[!UICONTROL Open In Agenda View]**。 按一下&#x200B;**[!UICONTROL Save]**。

   ![影像三](/help/sky/assets/program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view-3.png)

1. 您現在可以在日曆中找到您的智慧型促銷活動。 將滑鼠指標暫留在畫面上，以取得詳細資訊……

   ![影像4](/help/sky/assets/program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view-4.png)

...或按一下以開啟編輯器窗口。

![影像5](/help/sky/assets/program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view/create-a-batch-smart-campaign-in-program-schedule-view-5.png)
