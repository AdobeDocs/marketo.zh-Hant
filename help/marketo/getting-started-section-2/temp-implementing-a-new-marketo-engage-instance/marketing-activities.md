# 新區域：行銷活動檢查清單 {#new-area-marketing-activities-checklist}

行銷活動包含構成自動化行銷方案的所有資產和內容。 當您設定新的Marketo Engage執行個體時，乾淨和清晰的組織確保所有使用者都可以輕鬆找到和管理各種計畫。 追蹤您的設定進度，並透過可下載的核取清單離線檢查基本步驟 — 請在這裡連結 —
