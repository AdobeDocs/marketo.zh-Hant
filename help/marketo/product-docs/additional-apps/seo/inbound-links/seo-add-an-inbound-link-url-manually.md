# SEO — 手動新增入站連結URL {#seo-add-an-inbound-link-url-manually}

您可以手動新增入站連結。 方法如下：

1. 前往 **傳入連結** 區段。

   ![](assets/image2014-9-18-13-3a40-3a3.png)

1. 按一下 **新增URL**.

   ![](assets/image2014-9-18-13-3a40-3a8.png)

1. 在URL中輸入。 按一下 **新增**.

   ![](assets/image2014-9-18-13-3a40-3a32.png)

   >[!TIP]
   >
   >你知道你能 [將入站連結新增至新清單或現有清單](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-managing-lists.md){target=&quot;_blank&quot;}? 看看！

   ![](assets/image2014-9-18-13-3a41-3a14.png)

滾！ 您現在應該會看到新的URL已新增至清單。 收集所有資料可能需要一些時間。
