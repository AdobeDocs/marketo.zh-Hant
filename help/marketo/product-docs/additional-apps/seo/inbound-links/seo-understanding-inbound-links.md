# SEO — 了解入站連結 {#seo-understanding-inbound-links}

傳入連結表示您的網站值得參考的搜尋引擎。 很好！

![](assets/image2014-9-18-13-3a18-3a10.png)

## 欄的定義 {#definition-of-columns}

| 欄標題 | 說明 |
|---|---|
| 傳入連結的URL | 有關網頁。 |
| 優先順序 | 此機會對您的頁面排名有多重要。 |
| 競爭者連結 | 競爭者是否在該URL中連結。 |
| 連結至我的網站 | 網站是否連結至該URL。 |
| 來自我網站的連結 | 指出您的網站是否連結至該URL。 |
| PageRank | 識別URL在搜尋中的頁面排名(1 - 10) |

真貼心！ 現在您了解傳入連結，我們就能為您的網站建議更多傳入連結機會。
