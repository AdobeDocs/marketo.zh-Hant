# SEO — 瞭解 [!UICONTROL 關鍵字] {#seo-understanding-keywords}

[!UICONTROL 關鍵字] 是你SEO戰略的支柱。 您的目標是在第一頁上，盡可能接近與您的業務相關的關鍵字#1。

的 [!UICONTROL 關鍵字] 部分將向您顯示您的站點在實現此功能方面有多成功，以及您如何累進 [您的競爭](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-add-competitors.md)。
