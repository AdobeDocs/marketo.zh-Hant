# 瞭解收入瀏覽器{#understanding-opportunity-analysis-in-revenue-explorer}中的Opportunity Analysis

Revenue Explorer中的Opportunity Analysis允許您更深入地檢查您的機會。 根據任何銷售線索／公司屬性（如銷售線索來源、行業或地理位置）切割資料。 根據名稱、階段或概率分析機會的建立和關閉。 瞭解對管道的行銷貢獻。

## 範例分析{#example-analysis}

以下是您可以在「業務機會分析」區域建立的一些報告。

1. 行銷對機會的影響

   您的行銷方案對您公司銷售管道的影響比例為何？ 這份報告給您答案。 以下圓形圖顯示了由行銷計畫獲得的所有機會數量的百分比和機會金額。

   ![](assets/image2015-7-21-16-3a21-3a4.png)

1. 營銷對已關閉和已贏機會的影響
1. 此報表依數量和金額顯示有多少收入是由行銷所獲得及影響的。

   ![](assets/image2015-7-21-16-3a41-3a55.png)

1. 由銷售線索來源關閉的業務機會
1. 此報告將按銷售線索來源列出所有關閉的業務機會，並為您提供一個清楚的概覽，瞭解哪些來源有效以及哪些來源無效。

   ![](assets/image2015-7-21-10-3a34-3a50.png)

1. 按源關閉時間

   此報告演示了關閉業務機會的平均天數與銷售線索來源之間的關係。

   ![](assets/image2015-7-21-10-3a35-3a3.png)

1. Open Opportunity and Stage

   此報表顯示每個收入週期階段中開啟的機會數。

   ![](assets/image2015-7-21-10-3a35-3a32.png)

1. 各產業機會數

   這份報告回答了「我們是否每年從某些行業獲得的機會越來越少？」

   ![](assets/image2015-7-21-10-3a35-3a45.png)

## 機會分析維度和度量{#opportunity-analysis-dimensions-and-measures}

Opportunity Analysis使您能夠訪問所有與銷售線索、公司和業務機會相關的維度以及與業務機會相關的措施。 使用這些機會分析維度和度量來回答您報表中的特定問題。

1. 公司屬性

   | 維 | 說明 |
   |---|---|
   | 年收入 | 公司的年收入 |
   | 城市 | 公司所在城市 |
   | 國家／地區 | 公司所在的縣 |
   | 產業 | 公司所處的行業 |
   | 公司名稱 | 公司名稱 |
   | 員工人數 | 公司員工人數 |
   | 郵遞區號 | 公司郵遞區號 |
   | SIC代碼 | 公司SIC代碼 |
   | 州 | 公司所在的州 |

1. 銷售線索屬性

   | 維 | 說明 |
   |---|---|
   | 封鎖清單 | 銷售線索被封鎖 |
   | 轉換為Opportunity | 銷售線索被轉換為業務機會 |
   | 電子郵件無效 | 銷售線索是否具有有效的電子郵件地址 |
   | 暫停行銷 | 銷售線索是否因行銷電子郵件而暫停 |
   | 電子郵件地址 | Lead的電子郵件地址 |
   | 職稱 | Lead的職稱 |
   | 完整名稱 | Lead的全名 |
   | 原始來源類型 | Lead的原始源類型 |
   | 註冊源類型 | Lead的註冊源類型 |
   | 銷售線索所有者電子郵件地址 | 潛在客戶擁有者的電子郵件地址 |
   | 銷售線索責任人職稱 | 銷售線索所有者的職稱 |
   | 銷售線索所有者名稱 | 銷售線索所有者名稱 |
   | 銷售線索來源 | 銷售線索來源 |
   | 銷售線索狀態 | 銷售線索狀態 |

1. 銷售線索建立時間範圍

   | 維 | 說明 |
   |---|---|
   | 銷售線索建立年份 | 建立銷售線索的年份 |
   | 銷售線索建立季度 | 建立銷售線索時的季度 |
   | 銷售線索建立月份 | 建立銷售線索的月份 |
   | 銷售線索建立周 | 建立銷售線索的周 |
   | 銷售線索建立日期 | 建立銷售線索的日期 |

1. 機會屬性

   | 維 | 說明 |
   |---|---|
   | 機會已關閉 | 機會是否已經結束 |
   | 業務機會預測類別 | 業務機會預測類別 |
   | 業務機會名稱 | 業務機會名稱 |
   | 機會階段 | 機會階段 |
   | 業務機會類型 | 機會類型 |
   | 贏得機會 | 這個機會是否已經結束，並贏得 |
   | 行銷影響的機會 | 此標幟指出任何銷售機會／聯絡人是否在任何行銷方案中獲得或成功。 只考慮定義期間成本的方案。 |

1. 銷售機會關閉時間範圍

   | 維 | 說明 |
   |---|---|
   | 銷售機會結束年份 | 機會關閉的年份 |
   | 銷售機會已結束的季度 | 機會關閉的季度 |
   | 銷售機會結束月份 | 機會關閉的月份 |
   | 銷售機會結束周 | 機會關閉的那周 |
   | 業務機會關閉日期 | 業務機會關閉的日期 |

1. 業務機會建立時間範圍

   | 維 | 說明 |
   |---|---|
   | 機會建立年 | 建立機會的年份 |
   | 業務機會建立的季度 | 建立業務機會的季度 |
   | 業務機會建立月份 | 建立業務機會的月份 |
   | Opportunity Created Week | 建立業務機會的那一週 |
   | 業務機會建立日期 | 建立業務機會的日期 |

1. 措施

   | 測量 | 說明 |
   |---|---|
   | 結束機會的平均天數 | 關閉業務機會的平均天數 |
   | 結束機會的平均天數（丟失） | 丟失機會的平均天數 |
   | 結束機會的平均天數(Won) | 成功銷售機會的平均天數 |
   | 所有機會數 | 所有機會的總數 |
   | 機會數（已關閉） | 已關閉（成功或失敗）的機會總數 |
   | 機會數（丟失） | 丟失的機會總數 |
   | 機會數（開啟） | 仍在開啟的機會總數 |
   | 機會數(Won) | 贏得的機會總數 |
   | 機會金額 | 總機會量。 如果一個業務機會關聯了多個銷售線索，則分配金額基於銷售線索分數。 |
   | 機會金額（丟失） | 丟失機會的總金額。 如果一個業務機會關聯了多個銷售線索，則分配金額基於銷售線索分數。 |
   | 業務機會金額（開啟） | 未結業務機會的總金額。 如果一個業務機會關聯了多個銷售線索，則分配金額基於銷售線索分數。 |
   | 機會金額(Won) | 贏得的機會總金額。 如果一個業務機會關聯了多個銷售線索，則分配金額基於銷售線索分數。 |
