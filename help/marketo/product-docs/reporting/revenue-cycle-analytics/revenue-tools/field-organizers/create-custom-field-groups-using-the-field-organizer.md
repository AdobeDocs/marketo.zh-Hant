# 使用「欄位管理器」建立自訂欄位群組 {#create-custom-field-groups-using-the-field-organizer}

在Revenue Cycle Explorer的「模型效能分析（銷售機會）區域」中啟用報告的自定義欄位組之前，必須將標準欄位或自定義欄位分為組，以便通過Marketo Lead Management中的「欄位管理器」進行報告。 這隻適用於銷售機會和公司屬性。
當您從「新建欄位管理器」對話框的「欄位」下拉清單中選擇標準欄位或自定義欄位時，系統會將與要分組的欄位相關聯的Marketo銷售機會管理資料類型與「欄位管理器」中三個可用編輯器之一進行映射：字串、整數或日期。

| Marketo銷售機會管理資料類型 | 欄位管理器編輯器資料類型 |
|---|---|
| 字串 | 字串 |
| 電子郵件 | 字串 |
| 整數 | 整數 |
| 文字 | 字串 |
| URL | 字串 |
| 參考 | 不支援 |
| 貨幣 | 整數 |
| 日期時間 | 日期 |
| 布林值 | 不支援 |
| 電話 | 字串 |
| 日期 | 日期 |
| 浮點數 | 整數 |
| 計算 | 不支援 |

接下來的三個章節說明如何為字串、整數或日期類型建立自訂欄位群組。

## 建立自訂欄位群組 — 字串編輯器 {#create-custom-field-group-string-editor}

1. 按一下 **銷售機會資料庫**.

   ![](assets/one.png)

1. 按一下 **新增**，然後選取 **新建欄位管理器**.

   ![](assets/two.png)

1. 按一下 **欄位** 並選取標準或自訂欄位，其資料類型會對應至字串編輯器（請參閱上一節的表格）。 此處使用國家/地區。

   ![](assets/three.png)

1. 按一下 **建立**。

   ![](assets/four.png)

   新的自定義組顯示在銷售機會資料庫樹中，以「欄位名稱」>「欄位名稱組」表示(示例：國家>國家/地區組)。

   ![](assets/4.5.png)

1. 按一下鉛筆圖示以自訂名稱。 例如，您可將「國家/地區群組」重新命名為「大陸」。 輸入所需的新名稱，然後按一下離開方塊以自動儲存。

   ![](assets/five.png)

1. 依預設，所有資料值都會放在「其他」子群組中。 若要對資料值進行分類，請按一下 **新增群組** 建立子組並為其命名。

   >[!NOTE]
   >
   >您最多可以新增10個子群組，以分類資料值。 所建立的每個子組都被分配一個ID號。

   在此範例中，已為大部分大陸建立群組。

   ![](assets/six.png)

   >[!NOTE]
   >
   >若要刪除子組，只需按一下子組名稱旁的紅色X即可。 如果組中有任何資料值，則資料值將移至預設的「其他」組。

1. 反白顯示畫布中的資料值，並將資料值拖放至適當的子群組中。

   ![](assets/seven.png)

   >[!NOTE]
   >
   >要從子組中刪除資料值，請將資料值重新分配到預設的「其他」組。

1. 使用畫布上方左上角的篩選選項，選取並檢視一或多個子群組中的資料值。 畫布中會顯示以您的篩選選擇為基礎的資料值。

   ![](assets/eight.png)

   >[!NOTE]
   >
   >定義群組後，您可以透過Marketo Lead Management中的Revenue Cycle Analytics索引標籤，啟用自訂欄位群組，以在Model Performance Analysis(Lead)中報告。

## 建立自訂欄位群組 — 整數編輯器 {#create-custom-field-group-integer-editor}

1. 按一下 **銷售機會資料庫**.

   ![](assets/one.png)

1. 按一下 **新增**，然後選取 **新建欄位管理器**.

   ![](assets/two.png)

1. 按一下 **欄位** 並選取標準或自訂欄位，其資料類型會對應至字串編輯器（請參閱上一節的表格）。 此處使用「年度收入」。

   ![](assets/nine.png)

1. 按一下 **建立**。

   ![](assets/9.5.png)

   新的自定義組顯示在銷售機會資料庫樹中，以「欄位名稱」>「欄位名稱組」表示(示例：年度收入>年度收入組)。

   ![](assets/9.6.png)

1. 按一下整數編輯器上方的預設自訂群組名稱，以自訂名稱。 例如，您可將「年度收入群組」重新命名為「按大小劃分的年度收入」。 按一下 **儲存**.

   ![](assets/eleven.png)

   整數編輯器可讓您建立多個子群組，以依大小定義每個子群組。 在此示例中，將為小型、中型和企業業務建立三個組。

1. 若要新增您的第一個群組，請在 **群組名稱** 欄位(範例：小)，並在 **群組範圍** 欄位(範例：200000)。 按一下 **新增群組**.

   ![](assets/twelve.png)

   在剛輸入的組下面顯示一個空組條目。 以下範例顯示「小型」、「中型」和「企業」業務的項目。

   >[!NOTE]
   >
   >您最多可以新增10個子群組，以分類資料值。 每個「群組範圍」項目都以先前的項目為基礎。 如果您為您建立的最後一個自訂子群組保留最後一個「群組範圍」項目空白，則不會設定最大資料值。

1. 按一下「摘要」標籤，儲存並檢閱您的設定。

   ![](assets/thirteen.png)

   >[!NOTE]
   >
   >若要刪除子組，請按一下子組名稱旁的紅色X。

1. 在「摘要」頁面上，檢閱您的設定。

   ![](assets/13.5.png)

   >[!NOTE]
   >
   >定義群組後，您可以透過Marketo Lead Management中的Revenue Cycle Analytics索引標籤，啟用自訂欄位群組，以在Model Performance Analysis(Lead)中報告。

## 建立自訂欄位群組 — 日期編輯器 {#create-custom-field-group-date-editor}

1. 按一下 **銷售機會資料庫**.

   ![](assets/one.png)

1. 按一下 **新增**，然後選取 **新建欄位管理器**.

   ![](assets/two.png)

1. 按一下 **欄位** 並選取標準或自訂欄位，其資料類型會對應至字串編輯器（請參閱上一節的表格）。 此處使用贏取日期。

   ![](assets/fourteen.png)

1. 按一下 **建立**。

   ![](assets/14.5.png)

   新的自定義組顯示在銷售機會資料庫樹中，以「欄位名稱」>「欄位名稱組」表示(示例：收購日期>收購日期組)。

   ![](assets/14.6.png)

1. 按一下日期編輯器上方的預設自訂群組名稱，以自訂名稱。 例如，您可以將「贏取日期群組」重新命名為「贏取日期類別」。 按一下 **儲存**.

   ![](assets/fifteen.png)

   日期編輯器可讓您建立多個子群組，並依日期定義每個子群組。 在此範例中，將建立三個群組：第1-15季銷售線索、第2-15季銷售線索和第3-15季銷售線索。

1. 若要新增您的第一個群組，請在 **群組名稱** 欄位(範例：Q1-15 Leads)，並在日期欄位中輸入一個日期，該日期表示銷售線索在或之前獲得的日期(例如：3/31/2015是第1-15季的最後一天)。 按一下 **新增群組**.

   ![](assets/sixteen.png)

   >[!NOTE]
   >
   >您最多可以新增10個子群組，以分類資料值。 每個「群組範圍」項目都以先前的項目為基礎。 如果您為您建立的最後一個自訂子群組保留最後一個「群組範圍」項目空白，則不會設定結束日期值。

   以下範例顯示2015年第1季到第3季的銷售線索。

   ![](assets/16.5.png)

   就這樣！ 幹得好。
