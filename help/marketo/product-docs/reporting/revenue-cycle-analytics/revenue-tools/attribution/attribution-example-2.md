# 歸因範例2 {#attribution-example}

請閱讀以下案例，並嘗試確定網格中應包含的數字。

* 4月11日 |票據由收購（展會）
* 4月15日 | Joan被收購（網路研討會）
* 4月22日 |（機會1）為6,000美元建立
* 4月24日 |（機會2）為10,000美元建立
* 4月25日 |比爾和瓊與 **兩者** 奧普蒂
* 4月29日 |（機會1）為閉門

| 方案名稱 | （展會） | （網路研討會） |
|---|---|---|
| (FT)已建立Opty | `<pre>1</pre>` | `<pre>1</pre>` |
| (FT)管道已建立 | `<pre>$8,000</pre>` | `<pre>$8,000</pre>` |
| (FT)奧蒂元 | `<pre>0.5</pre>` | `<pre>0.5</pre>` |
| (FT)收入韓元 | `<pre>$3,000</pre>` | `<pre>$3,000</pre>` |

**顯示答案**

>[!NOTE]
>
>**說明**
>
>因為比爾和瓊都和 **兩者** 機會，系統（根據規則）將評分平均分給兩者。
>
>為每個方案建立的管道（8 000美元）是可以貸記的總額（16 000美元）的一半。

>[!NOTE]
>
>**歸因規則**
>
>1. 貸項均分
>1. 你不能比掙的多給
>1. 你不能因為過去發生的事而評價


請試用所有範例，您就會成為歸因專家！
