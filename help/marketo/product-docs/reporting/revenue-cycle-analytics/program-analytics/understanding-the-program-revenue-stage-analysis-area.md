# 瞭解方案收入階段分析區域 {#understanding-the-program-revenue-stage-analysis-area}

此分析區域可讓您分析個別計畫的成效，或按管道檢視摘要結果。 它可讓您深入瞭解產生的新名稱中，有多少已達到您的收入週期模型內的特定成功路徑階段。

**您可以使用此分析區域回答的業務問題範例包括**：

指定方案中曾有多少新名稱達到我模型中的特定階段？

![](assets/one-3.png)

在模型中，目前有多少來自給定計畫的新名稱處於特定階段？

![](assets/two-3.png)

潛在客戶需要多少天才能達到目前的階段？

![](assets/three-3.png)

**方案收入階段分析Dimension與測量**

Dimension和測量會依功能分類，並以系統中的黃色或藍色點表示：黃色代表維度，藍色代表測量。 使用方案收入階段分析維度和量度來回答報表中的特定問題。

若要檢視類別中可用的維度或測量，請按一下類別名稱旁的向右箭頭，以展開類別清單。 按一下向下箭頭，收合類別清單。

>[!TIP]
>
>若要在報表中取得特定維度或量度的詳細資訊，請將滑鼠游標停留在特定維度或量度上。
