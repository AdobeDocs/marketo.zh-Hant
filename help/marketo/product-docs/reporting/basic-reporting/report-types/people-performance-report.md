# 人員績效報表 {#people-performance-report}

使用「人員效能報表」來測量資料庫在一段時間內的成長。 您可以瞭解新增了多少人員及新增時間。 進一步瞭解人員並評估您的計畫。 依任何人員或公司屬性或依下列條件將結果分組： [細分](/help/marketo/product-docs/personalization/segmentation-and-snippets/segmentation/create-a-segmentation.md).

1. [建立報告](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) 並選取 **人員績效** [報告型別](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md).

1. [設定報告的時間範圍](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/change-a-report-time-frame.md) 並按一下 **報告** 標籤。

1. 太棒了！ 現在您已準備好探索您的人員效能報表。 在下方清單中瞭解如何從中取得更多。

   >[!NOTE]
   >
   >依預設，人員績效報表會依下列專案分組： *建立時間* 月。

   ![](assets/one.png)

## 人力效能報表 {#the-power-of-people-performance-reports}

人員績效報表功能非常強大。 透過進一步分組、篩選和設定這些專案，您可以瞭解有關資料庫中人員以及方案有效性的許多資訊。

您可以：

* [依潛在客戶或公司屬性將潛在客戶分組](/help/marketo/product-docs/reporting/basic-reporting/report-activity/group-person-reports-by-attribute.md).
* [依區段將潛在客戶分組](/help/marketo/product-docs/personalization/segmentation-and-snippets/segmentation/group-person-reports-by-segment.md).
* [使用智慧列示作為自訂報表欄。](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/add-custom-columns-to-a-person-report.md)
* [將機會量度新增為報表欄。](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/add-opportunity-columns-to-a-lead-report.md)
* [深入研究以進一步瞭解特定列中的銷售機會。](/help/marketo/product-docs/reporting/basic-reporting/report-activity/drill-down-in-a-people-performance-report.md)
* [使用智慧清單篩選報表中的銷售機會。](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/filter-people-in-a-report-with-a-smart-list.md)
* [選取要包含的欄。](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/select-report-columns.md)
* [排序報表欄。](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md)

  >[!TIP]
  >
  >請記住，報告很容易做到 [建立](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md)、設定和 [刪除](/help/marketo/product-docs/reporting/basic-reporting/report-activity/delete-a-report.md). 繼續並嘗試許多操縱它們的方法，以瞭解專注於關鍵資料的最佳方法。
