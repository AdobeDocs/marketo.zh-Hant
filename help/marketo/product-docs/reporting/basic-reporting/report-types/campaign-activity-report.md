# 促銷活動報表 {#campaign-activity-report}

了解您 [智慧型行銷活動](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/understanding-batch-and-trigger-smart-campaigns.md) 執行中。

1. [建立報表](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) ，然後選取 **促銷活動** [報告類型](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md).

1. [設定報表的時間範圍](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/change-a-report-time-frame.md) 並按一下 **報表** 標籤。

1. 太棒了！ 探索您的報表，以識別依促銷活動類型分組的最有效促銷活動。

   ![](assets/image2014-9-16-16-3a8-3a45.png)

   >[!TIP]
   >
   >若要尋找處理最多人員的促銷活動，例如 [排序報表](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md) on _已處理的總人數_ 選擇 **降序排序**.

   按一下智慧型促銷活動的名稱以開啟它。  [可以選取的欄](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/select-report-columns.md) 對於「促銷活動」報表，包括：

   | 欄 | 說明 |
   |---|---|
   | 批運行計數 | 執行促銷活動的次數。 |
   | 已處理的總人數 | 已通過促銷活動流的人數。 |
   | 等待步驟中的人員 | 如果促銷活動仍在執行，則會保留 *等待* 步驟。 |
   | 從流中移除 | 已從促銷活動移除的人數。 |
   | 上次活動 | 促銷活動上次處理人員的時間。 |

   報表中的促銷活動類型：

   | 類型 | 說明 |
   |---|---|
   | 批次 | 在 **行銷活動** 標籤。 |
   | 執行動作 | 從 **資料庫**. |
   | 觸發（活動） | 中目前觸發的促銷活動 **行銷活動** 標籤。 |
   | 觸發（非作用中） | 中的非使用中觸發的促銷活動 **行銷活動** 標籤。 |
