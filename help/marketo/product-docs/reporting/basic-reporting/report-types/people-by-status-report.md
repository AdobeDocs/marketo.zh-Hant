# 按狀態列出的人員報表 {#people-by-status-report}

檢查每個月有多少人顯示在每個「人員狀態」值中，以瞭解您在整個流程中 *的調動程度* 。

1. [建立報表](../../../../product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) ，並選取「依狀 **態的人員** 」報 [表類型](report-type-overview.md)。
1. [設定報表的時間範圍](../../../../product-docs/reporting/basic-reporting/editing-reports/change-a-report-time-frame.md) ，然後按一下「報 **表** 」標籤。
1. 太好了！ 現在，您可以逐月查看每個人員狀態中有多少人。

   ` ![](assets/image2017-3-27-11-3a17-3a4.png)

   `

   >[!TIP]
   >
   >按一下加號(+)以展開每個月，並查看每個人員狀態的特定數字。
