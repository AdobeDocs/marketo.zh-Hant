# 將報表複製至群組報表{#clone-a-report-to-group-reports}

**「我的報表」**中的任何報表都只能顯示給您。 如果您想要讓所有其他人都能存取報表，該怎麼辦？ 您可以透過下列方式：

1. 在&#x200B;**Analytics**&#x200B;區域中，按一下所要的[儲存報表](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/save-a-report.md)。

   ![](assets/image2015-4-21-11-3a25-3a54.png)

   >[!NOTE]
   >
   >或者，可將儲存的報表拖曳至資料夾，將報表複製至&#x200B;**群組報表**。

1. 按一下&#x200B;**報告操作**&#x200B;並選擇&#x200B;**克隆報告**。

   ![](assets/image2015-4-21-11-3a29-3a32.png)

1. 按一下「複製至&#x200B;**」下拉式清單，然後選取您的群組。**&#x200B;選擇&#x200B;**資料夾**。

   ![](assets/image2015-4-21-11-3a32-3a0.png)

1. **為報** 表命名，然後按一 **下仿製**。

   ![](assets/image2015-4-21-11-3a33-3a11.png)

   酷！ 您的報表現在會顯示在&#x200B;**群組報表**&#x200B;中。

   ![](assets/image2015-4-21-11-3a37-3a25.png)
