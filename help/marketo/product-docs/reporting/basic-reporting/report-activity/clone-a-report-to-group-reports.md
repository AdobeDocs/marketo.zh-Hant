# 複製報告至群組報告 {#clone-a-report-to-group-reports}

**我的報告**中的任何報告僅對您可見。 如果您想要讓其他所有人都能存取報告，該怎麼做？ 以下說明如何執行此操作：

1. 在 **分析** 區域，按一下所需的 [已儲存的報告](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/save-a-report.md).

   ![](assets/image2015-4-21-11-3a25-3a54.png)

   >[!NOTE]
   >
   >或者，您也可以將已儲存的報告複製至 **群組報表** 將報表拖曳至資料夾。

1. 按一下 **報表動作** 並選取 **原地複製報告**.

   ![](assets/image2015-4-21-11-3a29-3a32.png)

1. 按一下 **複製至** 下拉式清單，然後選取您的群組。 選取 **資料夾**.

   ![](assets/image2015-4-21-11-3a32-3a0.png)

1. **名稱** 報表並按一下 **原地複製**.

   ![](assets/image2015-4-21-11-3a33-3a11.png)

   酷！ 您的報告現在將顯示在 **群組報表**.

   ![](assets/image2015-4-21-11-3a37-3a25.png)
