# 按屬性分組人員報表 {#group-person-reports-by-attribute}

您可以依任何 [人員](http://docs.marketo.com/display/docs/basic+reporting) 或公司屬性對人員報表進行分組。

1. 前往「行 **銷活動** (或 **Analytics**)」區域。

   ![](assets/image2017-3-28-10-3a22-3a53.png)

1. 從導覽樹中選擇您的人員報表，然後按一下「設 **定** 」標籤。

   ![](assets/image2017-3-28-11-3a33-3a48.png)

1. 連按兩下「群 **組人員依據」**。

   ![](assets/image2017-3-28-11-3a34-3a5.png)

   >[!NOTE]
   >
   >您也可以 [依區段將人員報表分組](../../../../product-docs/personalization/segmentation-and-snippets/segmentation/group-person-reports-by-segment.md)。

   在「人員分組依據」對話框中，選擇要用於分組的人員或公司屬性。

   ![](assets/image2017-3-28-11-3a34-3a42.png)

   >[!TIP]
   >
   >如果您選擇具有數值的屬性，例如「 *Created At* 」或「 *Annual Revenue*」，請從右側的「 **Units** 」下拉式清單中選取量度。

   就這樣！ 按一下「報表」標籤，即可檢視報表的分組。

   ![](assets/image2017-3-28-11-3a35-3a0.png)

   >[!NOTE]
   >
   >**深入探討**
   >
   >
   >在「基本報表」深入探討中，進一步瞭解人員報 [表和](http://docs.marketo.com/display/docs/basic+reporting)其他報表類型。
