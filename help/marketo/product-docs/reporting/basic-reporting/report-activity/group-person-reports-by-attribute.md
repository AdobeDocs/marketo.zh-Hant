# 按屬性分組人員報表 {#group-person-reports-by-attribute}

您可以依任何人員或公司屬性將人員報表分組。

1. 前往 **行銷活動** (或 **Analytics**)區域。

   ![](assets/image2017-3-28-10-3a22-3a53.png)

1. 從導航樹中選擇您的人員報告，然後按一下 **設定** 標籤。

   ![](assets/image2017-3-28-11-3a33-3a48.png)

1. 按兩下 **人員分組依據**.

   ![](assets/image2017-3-28-11-3a34-3a5.png)

   >[!NOTE]
   >
   >您也可以 [依區段分組人員報表](/help/marketo/product-docs/personalization/segmentation-and-snippets/segmentation/group-person-reports-by-segment.md).

   在「按人員分組」對話框中，選擇要用於分組的人員或公司屬性。

   ![](assets/image2017-3-28-11-3a34-3a42.png)

   >[!TIP]
   >
   >如果您選擇的屬性具有數值，例如 _建立於_ 或 _年收入_，請從 **件數** 右下拉。

   就這樣！ 按一下「報表」索引標籤，即可查看據以分組的報表。

   ![](assets/image2017-3-28-11-3a35-3a0.png)
