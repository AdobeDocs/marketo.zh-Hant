# 篩選行銷活動報告 {#filter-a-campaign-activity-report}

將[行銷活動報告](/help/marketo/product-docs/reporting/basic-reporting/report-types/campaign-activity-report.md)的焦點放在特定的[智慧行銷活動](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/understanding-batch-and-trigger-smart-campaigns.md)。

>[!NOTE]
>
>衛星模式不支援在報表中篩選資產（資產詳細資料頁面右側的「在新視窗中開啟」圖示）。

1. 移至&#x200B;**行銷活動** （或&#x200B;**Analytics**）並選取您的行銷活動報告。

   ![](assets/filter-a-campaign-activity-report-1.png)

1. 按一下「**[!UICONTROL Setup]**」標籤，然後按兩下「**[!UICONTROL Campaigns]**」。

   ![](assets/filter-a-campaign-activity-report-2.png)

1. 選擇要包含在報表中的資料夾和特定智慧行銷活動。 按一下「**[!UICONTROL Apply]**」。

   ![](assets/filter-a-campaign-activity-report-3.png)

   >[!TIP]
   >
   >如果您選取資料夾，報表會包含報表執行時資料夾所包含的所有內容。

1. 完成了！ 按一下「**[!UICONTROL Report]**」標籤，只會在報表中檢視&#x200B;_僅_&#x200B;選取的智慧行銷活動。

   ![](assets/filter-a-campaign-activity-report-4.png)
