# 篩選促銷活動報表 {#filter-a-campaign-activity-report}

將您的促銷活 [動報表重](../../../../product-docs/reporting/basic-reporting/report-types/campaign-activity-report.md)點放在特定 [的智慧型促銷活動](http://docs.marketo.com/display/docs/smart+campaigns)。

1. 前往「行 **銷活動** 」(或 **Analytics**)，並選取您的促銷活動報表。

   ![](assets/image2014-9-16-16-3a13-3a56.png)

1. 按一下「 **設定** 」標籤，然後按兩下「促銷 **活動」。**

   ![](assets/image2014-9-16-16-3a14-3a1.png)

1. 選擇要包含在報表中的資料夾和特定智慧型促銷活動。 按一 **下套用**。

   ![](assets/image2014-9-16-16-3a14-3a11.png)

   >[!TIP]
   >
   >如果您選取資料夾，您的報表將包含資料夾在報表執行時包含的所有項目。

1. 你完了！ 按一下「 **報表** 」標籤，只 *查看報表中選* 取的智慧型促銷活動。

   ![](assets/image2014-9-16-16-3a14-3a32.png)
