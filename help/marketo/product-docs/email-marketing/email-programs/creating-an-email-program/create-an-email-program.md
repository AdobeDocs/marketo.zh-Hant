# 建立電子郵件程式 {#create-an-email-program}

使用電子郵件程式，快速輕鬆地傳送電子郵件給群組。

1. 前往 **行銷活動**.

   ![](assets/one.png)

1. 選取您要建立程式的資料夾，按一下 **新增** 下拉式清單，然後選取 **新計畫**.

   ![](assets/two.png)

1. 輸入名稱，選取 **電子郵件** 作為計畫型別，然後按一下 **建立**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >當選取 **電子郵件** 由於是方案型別，該頻道將自動設定為 **電子郵件傳送**. 您可以視需要加以變更。

   ![](assets/four.png)

很好！ 請注意，程式現在位於樹狀結構中並可供使用。 下一步是定義您的對象。 請參閱下方的Marketo相關文章。
