# 下載電子郵件的HTML {#download-an-emails-html}

為了備份和其他用途，Marketo可讓您下載電子郵件的HTML內容。

1. 找到電子郵件並加以選取。 在&#x200B;**電子郵件動作**&#x200B;下拉式清單中，按一下&#x200B;**下載HTML**。

   ![](assets/one-4.png)

這太簡單了！
