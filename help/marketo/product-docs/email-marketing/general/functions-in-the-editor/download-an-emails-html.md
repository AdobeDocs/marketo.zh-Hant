# 下載電子郵件的HTML {#download-an-emails-html}

為了備份和其他目的，Marketo允許您下載電子郵件的HTML內容。

1. 找到電子郵件並加以選取。 在 **電子郵件動作** 下拉式點按 **下載HTML**.

   ![](assets/one-4.png)

就這麼簡單！
