# 下載電子郵件的HTML {#download-an-emails-html}

為了備份和其他目的，Marketo可讓您下載電子郵件的HTML內容。

1. 找到電子郵件並加以選取。 在 **電子郵件動作** 下拉式清單點按 **下載HTML**.

   ![](assets/one-4.png)

這太簡單了！
