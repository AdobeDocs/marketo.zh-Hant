# 系統權杖字彙表 {#system-tokens-glossary}

除了人員代號之外，您也可以使用一些特別酷的系統代號。 這些區段在這裡。

>[!NOTE]
>
>您的帳戶時區設定會影響日期和時間Token的執行時間。

## system.date {#system-date}

`{{system.date}}`權杖將在執行階段呈現目前日期，如下所示： **Aug 08， 2013**

* 電子郵件或範本的正文
