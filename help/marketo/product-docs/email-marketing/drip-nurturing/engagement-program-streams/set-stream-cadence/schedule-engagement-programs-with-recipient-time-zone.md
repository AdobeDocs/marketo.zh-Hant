# 與收件者時區排程參與方案 {#schedule-engagement-programs-with-recipient-time-zone}

當您排程參與方案串流且收件者時區處於作用中狀態時，方案轉播將在第一個時區的午夜開始執行(UTC +14:00)。 我們要求您安排第一次演出 **至少25小時** 未來，因為在全球的每個時區，都可能有人符合演員資格。 我們會在第一個時區開始處理，保證我們會在每個收件者的排程日期和時間傳送電子郵件。

1. 在您的參與計畫中，導覽至 **資料流** 標籤，然後按一下資料流的順序排程加以編輯。

   ![](assets/image2017-12-5-13-3a36-3a21.png)

1. [設定節奏設定](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/set-stream-cadence.md) 如您一般，檢查 **收件者時區** 框。 請記住，您的首播必須在未來至少25小時。 按一下 **儲存**.

   ![](assets/image2017-12-5-13-3a50-3a32.png)

1. 請注意，如果「收件者時區」處於作用中狀態，順序排程將不會顯示特定時區，因為可能會有多個時區。 它只會顯示該小時。

   ![](assets/image2017-12-5-13-3a56-3a21.png)
