# 依收件者時區排程參與方案 {#schedule-engagement-programs-with-recipient-time-zone}

當您排程參與方案串流且收件者時區處於作用中狀態時，方案轉換將在第一個時區的午夜開始執行(UTC +14:00)。 我們要求您排程第一次轉換 **至少25小時** 因為未來全球每個時區都可能有符合演員資格的人。 在第一個時區的這個時間開始處理，可保證我們會在排定的日期和時間傳送每個收件者的電子郵件。

1. 在您的參與計畫中，導覽至 **串流** 索引標籤並按一下資料流的步調排程以進行編輯。

   ![](assets/image2017-12-5-13-3a36-3a21.png)

1. [設定您的步調設定](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/set-stream-cadence.md) 如往常一樣，然後檢查 **收件者時區** 方塊。 請記住，您的第一次轉換必須在未來至少25小時進行。 按一下 **儲存**.

   ![](assets/image2017-12-5-13-3a50-3a32.png)

1. 請注意，在收件者時區啟用時，步調排程將不會顯示特定時區，因為可能會有多個時區。 它只會顯示小時。

   ![](assets/image2017-12-5-13-3a56-3a21.png)
