# 複製串流 {#clone-a-stream}

由於各種原因（包括測試不同的訂單和不同的序列）複製串流。

1. 選取您的參與計畫，並前往「串流 **」標籤** 。

   ![](assets/cloneasteam.jpg)

1. 按一下串流齒輪圖示，然後按一下「仿 **制」**。

   ` ![](assets/image2014-9-15-17-3a0-3a23.png)

   `

   >[!NOTE]
   >
   >每個參與計畫最多可以有25個串流。
