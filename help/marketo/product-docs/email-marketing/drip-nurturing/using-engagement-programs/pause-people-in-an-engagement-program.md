# 暫停參與計畫中的人員 {#pause-people-in-an-engagement-program}

當使用者是參與計畫的成員時，他們會收到內容，直到 [已耗盡所有內容](people-who-have-exhausted-content.md). 您可以使用 [更改參與計畫順序](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-engagement-program-cadence.md) 即使使用者尚未用盡內容，仍可停止其接收內容的流程步驟。

1. 選擇參與方案。

   ![](assets/image2014-9-22-14-3a49-3a27.png)

1. 選擇 **已暫停** 作為 **新值** 阻止人員接收內容。

   ![](assets/image2014-9-22-14-3a49-3a31.png)

   您可以將人員設為 **正常** 如果您希望他們重新開始接收內容。 他們會從中斷的地方恢復。

   >[!NOTE]
   >
   >暫停人員將會阻止他們接收內容，但如果符合條件，他們仍會在資料流之間轉換。
