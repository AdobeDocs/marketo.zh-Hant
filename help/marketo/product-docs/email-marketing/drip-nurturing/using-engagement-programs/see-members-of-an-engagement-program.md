# 請參閱參與方案的成員 {#see-members-of-an-engagement-program}

您可以快速查看參與計畫的成員。 這是方法。

1. 在您的參與計畫中，按一下右鍵並選擇 **查看成員**.

   ![](assets/membersofengagement.jpg)

   >[!NOTE]
   >
   >此  [已用盡的內容](/help/marketo/product-docs/email-marketing/drip-nurturing/creating-an-engagement-program/understanding-engagement-programs.md) 欄值會在每次轉換後更新。

1. 您現在可以查看方案的所有成員。
