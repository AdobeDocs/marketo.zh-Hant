# 查看參與方案的成員 {#see-members-of-an-engagement-program}

您可以快速檢視參與計畫的成員。 方法如下。

1. 在您的參與方案中，按一下滑鼠右鍵並選取&#x200B;**[!UICONTROL View Members]**。

   ![](assets/membersofengagement.jpg)

   >[!NOTE]
   >
   >[已用完的內容](/help/marketo/product-docs/email-marketing/drip-nurturing/creating-an-engagement-program/understanding-engagement-programs.md)資料行值會在每次轉換後更新。

1. 您現在可以看到方案的所有成員。

   ![](assets/image2014-9-15-17-3a17-3a26.png)
