# 設定行動應用程式Android推播存取 {#configure-mobile-app-android-push-access}

1. 按一下 **管理**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. 選擇 **行動應用程式**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. 選取所需的行動應用程式。

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. 在 **推送存取類型**，選取 **Android** 按一下 **設定。**

   ![](assets/image2016-6-15-15-3a16-3a22.png)

   >[!NOTE]
   >
   >您需要 **伺服器API金鑰** 和 **項目編號** 行動應用程式開發人員取得。 開發人員可登入Google Play開發人員控制台來註冊您的應用程式並啟用雲端訊息，即可收到這些訊息。

1. 輸入您的伺服器API金鑰和專案編號。 按一下 **儲存**.

   ![](assets/image2015-4-22-18-3a54-3a54.png)

   真貼心。 請務必使用iOS設定應用程式。
