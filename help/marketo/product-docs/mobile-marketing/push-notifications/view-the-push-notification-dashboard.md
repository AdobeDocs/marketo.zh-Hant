# 檢視推播通知控制面板 {#view-the-push-notification-dashboard}

您可輕鬆瞭解推播通知的運作方式。

1. 前往 **行銷活動** 區域。

   ![](assets/image2015-12-11-12-3a57-3a48.png)

1. 選取您的行銷活動。

   ![](assets/image2015-12-11-13-3a1-3a56.png)

1. 按一下 **檢視：摘要** 並選取 **儀表板**.

   ![](assets/image2015-12-11-13-3a4-3a23.png)

1. 您可以在圓形圖表中檢視iOS和Android的「已傳送總數」和「總點按」。 向下捲動以檢視點選長條圖中的「衰減」。

   ![](assets/image2015-12-15-15-3a23-3a47.png)

   >[!NOTE]
   >
   >此 _已傳送_ 量度所反映的傳送次數，可能會超過推送通知傳送對象的確切人數。 這是因為其計算基礎為 *裝置數* 有資格接收推播的客戶。 例如，如果單一人員擁有三部裝置，控制面板會記錄三部傳送，而非一部。
