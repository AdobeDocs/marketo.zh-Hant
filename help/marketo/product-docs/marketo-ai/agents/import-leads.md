# 匯入銷售機會 {#import-leads}

透過欄位對應協助，將潛在客戶清單匯入並刪除重複專案至Marketo Engage資料庫。

匯入銷售機會清單？ 人物？ 以csv格式套用某些資料標準化規則，並將其儲存至Marketo資料庫。

按一下此代理程式會帶您進入對話式AI。 在左窗格中有一個「AI助理」，它提供一個視窗，讓「代理程式」發佈指引、回應和選項，以執行哪些資料標準化功能。

若要啟動匯入程式，請按一下附件圖示以上傳檔案。

輸入匯入清單，然後按一下傳送

輸入要套用的商業規則，然後按一下傳送。 輸入後，中央主控台中的視窗將提供套用規則的結果。

視需要輸入其他商業規則，然後按一下「傳送」。

若要檢視資料庫對應清單欄位的清單，請按一下對應標籤。

準備將清單儲存到資料庫時，請按一下匯入到Marketo 。

選取要儲存清單的資料夾，輸入銷售機會清單的名稱並核取同意方塊，然後按一下「核准並匯入至Marketo」按鈕。

儲存清單後，系統將為驗證提供已處理、失敗或有警告的記錄摘要。
