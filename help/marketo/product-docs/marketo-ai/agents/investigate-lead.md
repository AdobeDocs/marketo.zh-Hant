# 調查銷售機會 {#investigate-lead}

探索為什麼有人沒有MQL、符合方案資格或在生命週期中前進。

若要檢視調查案例的完整清單，請按一下右側的箭頭。

熒幕擷圖

| **標題** | 說明 |
|----------|------------|
| **MQL調查** | 提供特定銷售機會不符合資格、停止參與或卡住的詳細資訊。 |
| **方案資格** | 提供潛在客戶不符合培養計畫資格的詳細資訊。 |
| **生命週期進度** | 用於探索為什麼銷售機會停滯在比預期更長的階段。 |
| **參與分析** | 當Lead停止開啟或點按電子郵件時使用。 |
| **行銷活動歷程** | 用於檢視Lead的完整路徑和Journey中的接觸點。 |
| **資料品質** | 當預期不良資料可能導致Lead發生問題時使用。 |
| **銷售移交** | 當銷售人員拒絕銷售機會且未追蹤時使用。 |
| **歸因** | 用於探索哪些因素影響了Lead的產生Opportunity。 |
| **分數衰減** | 調查Lead分數意外下降的原因。 |

>[!TIP]
>
>若要快速檢視所有九個案例的清單，請在頂端的提示視窗中輸入`/lead investigation`。
