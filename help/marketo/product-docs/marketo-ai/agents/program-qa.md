# 程式QA {#program-qa}

稽核您的方案，以獲得所有元件（例如電子郵件、登陸頁面、行銷活動等）的最佳實務。

存取此Agent有兩種方式。

從中央視窗按一下代理程式，然後按一下右上方的下拉式清單

按一下使用此代理程式按鈕

在提示對話方塊的左窗格中，將會顯示預設文字。

然後在右窗格中選取您要QA的方案

計畫的摘要將顯示在中心視窗中，提供計畫的概覽以及為其建立的內容。

或者，您也可以按一下Conversational AI主畫面中的Agent 。

AI助理將顯示預設文字，以顯示將執行的代理程式。

然後，選取右側需要QA的程式。
