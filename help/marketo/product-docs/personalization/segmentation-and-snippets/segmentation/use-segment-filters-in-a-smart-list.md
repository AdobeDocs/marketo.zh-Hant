# 在智慧型清單中使用區段篩選 {#use-segment-filters-in-a-smart-list}

>* [建立智慧清單](../../../../product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)

使用區段篩選器最佳化您的智慧型清單效能。
