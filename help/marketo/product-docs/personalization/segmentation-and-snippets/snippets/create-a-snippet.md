# 建立程式碼片段 {#create-a-snippet}

程式碼片段可作為上的動態內容區塊 **電子郵件** 和 **登陸頁面**.

1. 前往 **設計工作室。**

   ![](assets/designstudio.png)

1. 按一下 **新增** 然後 **新程式碼片段**.

   ![](assets/image2014-9-16-8-50-4.png)

1. 輸入所需的詳細資訊，然後按一下 **建立**.

   ![](assets/image2014-9-16-8-3a50-3a14.png)

幹得好！ 為動態內容建立片段，簡化您的工作。 現在你可以繼續 [將內容新增至新程式碼片段](/help/marketo/product-docs/personalization/segmentation-and-snippets/snippets/add-content-to-a-snippet.md).
