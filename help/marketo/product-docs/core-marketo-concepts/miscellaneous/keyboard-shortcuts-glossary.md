# 鍵盤快速鍵辭彙表{#keyboard-shortcuts-glossary}

你是鍵盤快速鍵的人嗎？ 我們能搞定你！

## 區域導覽：{#area-navigation}

| 鍵盤快速鍵 | 區域 |
|---|---|
| Alt/Opt + A | Analytics |
| Alt/Opt + C | 日曆 |
| Alt/Opt + D | Design Studio |
| Alt/Opt + F | 全域搜尋 |
| Alt/Opt + L | 資料庫 |
| Alt/Opt + M | 行銷活動 |
| 選擇+ N | 管理員 |
| 選擇+ O | 我的行銷人員 |
| Alt/Opt + S | 下拉式超級球功能表 |

## 日曆／計畫視圖導航： {#calendar-schedule-view-navigation}

| 鍵盤快速鍵 | 動作 |
|---|---|
| Alt/Opt +向上鍵 | 及時返回 |
| Alt/Opt +向下鍵 | 及時前進 |
| Alt/Opt + T | 前往「今天」 |

## 其他：{#miscellaneous}

| 鍵盤快速鍵 | 動作 |
|---|---|
| ESC | 特寫快顯 |
| 退貨／輸入 | 送出快顯視窗 |
| Ctrl + Alt/Opt + H | 隱藏／取消隱藏表單編輯器中的欄位 |
