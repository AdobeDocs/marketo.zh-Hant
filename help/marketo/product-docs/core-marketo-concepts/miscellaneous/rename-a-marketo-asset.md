# 更名Marketo資產{#rename-a-marketo-asset}

檔案夾、程式和本機資產都可輕鬆重新命名。

1. 選取您要重新命名的任何資產，然後按一下右側面板名稱區塊。

   ![](assets/image2015-4-10-17-19-48.png)

1. 在文字欄位中輸入新名稱。 按一下&#x200B;**保存**。

   ![](assets/image2015-4-10-17-3a19-3a33.png)

   >[!NOTE]
   >
   >您無法重新命名已上傳的檔案和影像，或目前在智慧型清單或智慧型促銷活動中參考的資產（在篩選器、觸發器、流程步驟等）。
