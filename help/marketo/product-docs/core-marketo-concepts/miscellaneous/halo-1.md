# 光暈概述 {#halo-overview}

Halo是AI原生行銷應用程式，可在數分鐘內將您從單一提示導向準備好檢閱的行銷活動。 Halo專為B2B和B2C中的精益生命週期和成長團隊所打造，將行銷活動規劃、對象建立、內容產生和歷程設計整合到一個對話工作區中，讓您在切換工具時花費更少時間，在重要工作上花費更多時間。

什麼是Halo？

大部分行銷人員會將5個或更多工具拼接在一起，以送出單一行銷活動：計畫、細分、內容、設定、分析。 這項工作通常會持續幾天或幾週時間。

Halo可將該工作流程收合為單一對話式介面。 您從提示或簡短開始，而Halo的代理程式AI會產生結構化的行銷活動計畫、建立您的對象、草擬品牌內容，並組合多步驟歷程 — 所有這些都可以在一個行銷活動展示板上以純樸的語言加以改良。

一切由您掌控。 光環會產生；您可以檢閱、調整並決定要上線的專案。

光環是屬於誰

Halo是專為精益團隊的生命週期與成長行銷人員所打造，這些團隊希望在不犧牲品質或控制的情況下，更快速地推出行銷活動。

執行重新參與、上線、電子報和季節性推播的B2C行銷人員。

執行網路研討會邀請、帳戶上線、產品教育和培養序列的B2B行銷人員。

無論您是單人團隊還是小型行銷組織的成員，Halo都能協助您完成許多人的工作，而不會造成工具無計畫無序。

如何存取Halo

Halo可免費試用至2026年9月30日。

請前往coworker-essentials.experience.adobe.com。

使用您的企業電子郵件登入或建立帳戶。

現有的Adobe客戶會自動獲得核准，並直接進入試用環境以接受條款。

系統會提示新使用者建立帳戶。 Halo團隊會稽核請求並核准商業電子郵件 — 您進來時，會收到我們的電子郵件。

接受試用條款。

在「您的東西>品牌」下設定您的品牌。 Halo會使用這個預設值，作為您產生的每個行銷活動的預設值。

（可選）在電子郵件範本底下上傳HTML電子郵件範本，以便在產生內容時Halo套用您的設計。

有光環能做什麼

Halo將通常存在於五種不同工具中的五個工作流程，整合到一個對話體驗中：

將簡報轉換為行銷活動計畫。 放入提示或貼上簡報，Halo會傳回結構化的端對端計畫。

建立和調整對象。 上傳CSV或連線資料，然後在對話中調整區段。

產生品牌上內容。 Halo會撰寫符合您品牌語調、參照和範本的電子郵件。

設計多步驟歷程。 Halo會組合培養序列，而不只是一次性的傳送。

傳送證明電子郵件。 啟動前預覽您收件匣中的所有郵件。

啟動並分析。 （即將推出）傳送行銷活動，並在不離開Halo的情況下取得下個最佳步驟的深入分析。

瞭解如何在建置和產生行銷活動中端對端將這些集合在一起。

AI的運作方式

光暈會遵循簡單的多圈對話回圈：

輸入。 您可以提供行銷活動目標、對象（已上傳或已連線）以及選用的品牌內容。

正在處理。 Halo的代理程式AI會產生行銷活動計畫、建置歷程，並草稿個人化內容。

輸出。 行銷活動草稿隨即顯示在您的行銷活動展示板上 — 可供檢閱、反複專案及最終啟動。

您可以隨時與Halo保持溝通 — 「讓第二封電子郵件更緊急」、「縮短主旨列」、「從第三季度開始將受眾交換給已經流失的客戶」 — 並且它會更新原廠的成品。

您永遠坐在駕駛席上。 光暈會產生；您可以檢閱、調整及核准。 傳送前請務必檢查輸出。

快速入門秘訣

早期使用者發現的一些事項可帶來實際差異：

從實際使用的特定案例開始。 「90天內未訂購的客戶的回饋系列」會比「讓我成為行銷活動」強。 瀏覽系列的使用案例以尋求靈感。

清除您的對象資料。 Halo只能在CSV允許的情況下進行個人化。 在上傳前，排除任何不應寄送電子郵件的連絡人。

投資您的品牌設定。 Halo會在您註冊時擷取品牌詳細資訊 — 但快速檢閱「您的資料>品牌」會在您產生的每個促銷活動中帶來回報。

傳送證明電子郵件。 啟動之前，請在您自己的收件匣中傳送校樣和檢閱。

重複，不要重新啟動。 將Halo的第一個輸出視為起點。 最快速的提升路徑是幾輪細分，而不是全新的提示。

需要紙張軌跡時匯出。 將個別電子郵件匯出為HTML，或將整個行銷活動匯出為Word檔案或PDF，以供團隊檢閱。

試用版中有哪些功能

Halo是積極開發中的產品。 以下為進入時的須知事項：

試用期：現在至2026年9月30日。

傳送限制：每月最多5,000封電子郵件。

對象限制：每個行銷活動最多10,000個連絡人。

需要接受：存取產品前，您需要先檢閱並接受試用條款。

資料指引：請勿上傳敏感或受管制的資料。 Halo目前不適用於HIPAA管制的產業。

法規遵循：您有責任遵守電子郵件法規 — 取消訂閱連結、隱私權政策等。

新功能會在試用期間逐步推出。 您的意見反應有助於塑造未來趨勢，我們很樂意聽到您的意見。
