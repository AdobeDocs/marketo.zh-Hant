# 手動建立人員 {#create-a-person-manually}

有很多方法可以讓一個人進入Marketo。 如果您需要手動建立，以下說明操作方式。

>[!CAUTION]
>
>Marketo **not** 支援包含emoji的電子郵件地址。

1. 前往 **資料庫**.

   ![](assets/db-1.png)

1. 在 **新增**，按一下 **新人員**.

   ![](assets/two-2.png)

1. 輸入人員資訊，然後按一下 **建立**.

   ![](assets/three-2.png)

這是在Marketo中新增單一人員的簡單快速方式！
