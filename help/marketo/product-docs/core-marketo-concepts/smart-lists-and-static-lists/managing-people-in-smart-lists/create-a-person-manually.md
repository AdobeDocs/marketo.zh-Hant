# 手動建立人員 {#create-a-person-manually}

有許多方法可讓一個人進入Marketo。 如果您需要手動建立，請以下說明如何進行。

1. 轉到數 **據庫**。

   ![](assets/db-1.png)

1. 在「新 **增**」下，按 **一下「新增人員」**。

   ![](assets/two-2.png)

1. 輸入人員的資訊，然後按一下「創 **建」**。

   ![](assets/three-2.png)

在Marketo中加入單一人員是一種快速又簡單的方式！
