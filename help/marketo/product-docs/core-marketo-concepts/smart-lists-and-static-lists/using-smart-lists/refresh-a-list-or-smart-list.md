# 刷新清單或智慧清單 {#refresh-a-list-or-smart-list}

如果您已執行智慧清單，且經過幾分鐘，結果現在可能會有所不同 — 重新整理以了解。

## 刷新結果 {#refresh-results}

1. 若要更新 **人員** 頁簽，按一下刷新表徵圖。

   ![](assets/refreshbutton.png)

1. 智慧清單會重新執行，並顯示更新的結果集。

   ![](assets/loadingrefresh.png)

>[!TIP]
>
>有時候，當您執行智慧清單，稍後再回到清單時，可能會在右上角的人數前面看到「關於」這個字。 這表示數字大致為 — 按一下計數本身以重新整理計數，並取得更新的精確計數。
