# 將多個值添加到智慧清單篩選器 {#add-multiple-values-to-a-smart-list-filter}

>[!PREREQUISITES]
>
>* [建立智慧清單](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)
>* [尋找篩選器並新增至智慧清單](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md)


假設你想在加州找到所有人，但你可能同時存放了 **加州** 和 **CA** 在資料庫中。 若要納入所有適用人員，您可使用兩個 **狀態** 篩選，但一個比較簡單。

1. 前往 **行銷活動**.

   ![](assets/login-marketing-activities-2.png)

1. 查找並選擇智慧清單，然後按一下 **智慧清單** 標籤。

   ![](assets/smarlist-choosefilters.png)

1. 按一下 **+** 在篩選器上。

   ![](assets/smartlist-plussignhand-.png)

1. 您可以從左側選擇值，或只在右側輸入值，然後按一下 **確定**.

   ![](assets/image2014-9-11-17-3a51-3a39.png)

快來！
