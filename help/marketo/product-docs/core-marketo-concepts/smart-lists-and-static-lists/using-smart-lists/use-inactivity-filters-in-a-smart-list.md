# 在智慧清單中使用不活動篩選器 {#use-inactivity-filters-in-a-smart-list}

你知道你可以在智慧清單中尋找 *didn&#39;t* 做什麼？ 查看這些不活動篩選器。

1. 前往 **行銷活動**.

   ![](assets/login-marketing-activities-3.png)

1. 選擇要編輯的智慧清單，然後按一下 **智慧清單** 標籤。

   ![](assets/smartlist-choose.png)

1. 尋找您選擇的閒置篩選器並拖曳至畫布中。 例如，我們會尋找尚未造訪您任何頁面的使用者。

   ![](assets/draginactivityfilter.png)

   >[!TIP]
   >
   >中有許多篩選器 **不活動篩選器** 檔案夾。 搜尋 **Not** 再看看。

1. 選取 **是** 運算元。 這會找出過去30天內未造訪過任何頁面的所有使用者。

   ![](assets/mysmartlist-people.jpg)
