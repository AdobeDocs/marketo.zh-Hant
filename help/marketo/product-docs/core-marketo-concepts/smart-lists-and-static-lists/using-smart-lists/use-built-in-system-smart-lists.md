# 使用內建/系統智慧清單 {#use-built-in-system-smart-lists}

Marketo有一些非常實用的內建智慧清單。 讓我們看看他們！

>[!NOTE]
>
>Marketo正在將產品中的黑名單和白名單等辭彙變更為封鎖名單和允許清單。 在此更新中，您可能會在UI和檔案熒幕擷取畫面中看到舊辭彙，並在檔案文字中看到新辭彙。 若有任何混淆，敬請見諒。

1. 前往 **資料庫**.

   ![](assets/db.png)

1. 開啟 **系統智慧清單** 資料夾以顯示集合。

   ![](assets/two.png)

1. 檢視一個範例：選取 **所有人員**，然後按一下 **人員** 標籤。

   ![](assets/three.png)

   >[!NOTE]
   >
   >系統智慧列示中的篩選器不需要套用至每個智慧列示/行銷活動。 系統會自動辨識它們的內容。

以下是內建智慧清單功能的摘要：

| 清單名稱 | 說明 |
|---|---|
| 所有人員 | Marketo資料庫中的所有人員 |
| 已取消訂閱的人員 | 這些人員只能收到操作電子郵件；這通常由人員自己控制。 |
| 行銷活動已暫停 | 這些人員只能收到操作電子郵件；這通常由您（行銷人員）控制。 |
| 封鎖清單 | 這些人完全不會收到任何電子郵件。 |
| 退回的電子郵件地址 | 具有無法傳遞的電子郵件地址或拒絕您電子郵件的人員。 |
| 可能的重複專案 | Marketo資料庫中可能重複的人員。 |

>[!NOTE]
>
>無法刪除系統智慧清單。 除了「可能的重複專案」清單以外，也無法對其進行編輯。
