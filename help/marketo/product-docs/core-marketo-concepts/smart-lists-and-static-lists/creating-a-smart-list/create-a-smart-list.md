# 建立智慧清單 {#create-a-smart-list}

智慧清單可讓您使用簡單篩選器尋找特定的人員群組。 以下說明如何建立一個。

1. 移至&#x200B;**[!UICONTROL Marketing Activities]**。

   ![](assets/create-a-smart-list-1.png)

1. 選取您要建立智慧列示的程式。

   ![](assets/create-a-smart-list-2.png)

1. 在&#x200B;**[!UICONTROL New]**&#x200B;底下，按一下&#x200B;**[!UICONTROL New Local Asset]**。

   ![](assets/create-a-smart-list-3.png)

1. 選取「**[!UICONTROL Smart List]**」。

   ![](assets/create-a-smart-list-4.png)

1. 輸入&#x200B;**[!UICONTROL Name]**&#x200B;並按一下&#x200B;**[!UICONTROL Create]**。

   ![](assets/create-a-smart-list-5.png)

   太棒了！ 現在來新增和定義一些篩選器。
