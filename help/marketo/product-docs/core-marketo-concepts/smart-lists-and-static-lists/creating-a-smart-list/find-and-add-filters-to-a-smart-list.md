# 尋找篩選器並新增至智慧清單 {#find-and-add-filters-to-a-smart-list}

一旦您 [建立智慧清單](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)，您需要新增和 [定義](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/define-smart-list-filters.md) 篩選。 以下說明如何尋找和新增篩選器。

在此範例中，我們會尋找加州中分數超過50的所有人。

>[!TIP]
>
>探索右側的樹 — 篩選器功能強大，且功能多樣。

1. 前往 **行銷活動**.

   ![](assets/ma.png)

1. 選取您要新增篩選器的智慧清單，然後按一下 **智慧清單** 標籤。

   ![](assets/two.png)

1. 尋找並拖曳 **狀態** 篩選至畫布。

   ![](assets/three.png)

1. 同時尋找並拖曳 **分數** 篩選。

   ![](assets/four.png)

完美！ 讓我們定義這些篩選器。
