# 標示完成 {#marking-it-done}

將電子郵件方案、事件或分析卡標示為「完成」，以將其從資料流中移除。 有兩種方法可以做到。

1. 點選動作選單。

   ![](assets/image2015-7-14-17-3a32-3a35.png)

1. 點選 **完成**.

   ![](assets/image2015-7-14-17-3a36-3a31.png)

1. 或者，往前滑動卡片。

   ![](assets/image2015-9-25-9-3a46-3a6.png)

   >[!NOTE]
   >
   >將資訊卡標示為「完成」並不會刪除電子郵件、事件或智慧型促銷活動。 它只會將它從「時刻/稍後」資料流移動至「完成」資料流。
