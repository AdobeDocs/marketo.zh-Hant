# 事件簽入總覽 {#event-check-in-overview}

適用於iPad或Android平板電腦的Marketo Events應用程式可讓您在人員到達您的活動時，輕鬆立即為其辦理登入手續。

本文會概述完整程式。 標題會連結至個別文章，而某些Marketo活動則是此大綱中的連結。

有些活動會發生在事件之前，例如建立新角色和邀請出席者。 某些情況會在活動期間發生，例如登記出席者。 在事件後，您會將您的資料與Marketo同步，並更新人員的狀態。

祝活動順利！

## 為您的活動準備

在事件之前：

* [建立新的Marketo角色](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/grant-users-access-to-the-check-in-app.md) 事件的iPad或Android使用者
* [邀請新的Tablet使用者加入活動](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/grant-users-access-to-the-check-in-app.md) 並指派新角色給他們
* [建立新的Marketo事件程式](/help/marketo/product-docs/demand-generation/events/understanding-events/create-a-new-event-program.md)
* 指定事件日期
* 新增人員至您的活動

## 設定iPad或Android平板電腦

若要準備iPad或Android平板電腦：

* 下載並安裝Marketo Events iPad或Android平板電腦應用程式
* 確認登入認證
* 同步應用程式以填入事件
* 確認簽入運作中

## 使用iPad或平板電腦簽入人員

在活動上：

* 登入應用程式
* 尋找來賓（並建立新人員）
* [簽入來賓](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/check-people-into-your-event-from-your-tablet.md) （反向簽入，簽入時編輯人員記錄）
* 將iPad或Android平板電腦應用程式與Marketo同步
