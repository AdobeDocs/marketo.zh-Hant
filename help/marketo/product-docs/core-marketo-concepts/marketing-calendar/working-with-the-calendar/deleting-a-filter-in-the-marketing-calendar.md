# 在行銷日曆中刪除篩選 {#deleting-a-filter-in-the-marketing-calendar}

如果您想要刪除「行銷行事歷」中的篩選器，您已到達正確的位置。

1. 選取您要刪除的篩選。

   ![](assets/image2014-9-24-11-3a27-3a32.png)

1. 按一下紅色 **x**。

   ![](assets/image2014-9-24-11-3a27-3a36.png)

1. 按一下 **刪除** ，以確認。

   ![](assets/image2014-9-24-11-3a27-3a42.png)

   再見。
