# 按照方案篩選方案報告 {#filter-a-program-report-by-program}

將您的[方案績效報表](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md){target="_blank"}集中於特定方案，以比較其績效。

1. 移至&#x200B;**[!UICONTROL Marketing Activities]** （或&#x200B;**[!UICONTROL Analytics]**）。

   ![](assets/login-marketing-activities-3.png)

1. 選取您的方案效能報表。

   ![](assets/image2014-9-23-16-3a4-3a4.png)

1. 按一下「**[!UICONTROL Setup]**」索引標籤並拖曳到&#x200B;**[!UICONTROL Programs]**&#x200B;上。

   ![](assets/prospect3.jpg)

1. 選擇要包含在報表中的資料夾和特定程式。

   ![](assets/image2014-9-23-16-3a5-3a5.png)

   >[!TIP]
   >
   >如果您選取資料夾，報表會包含報表執行時資料夾所包含的所有內容。

1. 僅此而已！ 按一下「**[!UICONTROL Report]**」標籤，只需&#x200B;_在報表中檢視所選的_&#x200B;個程式。

   ![](assets/image2014-9-23-16-3a5-3a41.png)
