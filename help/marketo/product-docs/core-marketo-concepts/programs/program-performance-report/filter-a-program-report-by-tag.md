# 依標籤篩選方案報表 {#filter-a-program-report-by-tag}

聚焦 [方案執行情況報告](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md) 特定 [標籤](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags.md).

1. 前往 **行銷** **活動** (或 **Analytics**)。

   ![](assets/login-marketing-activities.png)

1. 選取 **計畫績效** 報表。

   ![](assets/image2014-9-23-16-3a12-3a36.png)

1. 按一下 **設定** 標籤，並拖曳至其中一個 **標籤** 篩選。

   ![](assets/prospects.jpg)

1. 選擇要包含在報表中的標籤值。

   ![](assets/prospect1.jpg)

1. 按一下 **套用**.

   ![](assets/prospect2.jpg)

1. 完成! 按一下 **報表** 索引標籤 _just_ 符合報表中所選標籤的程式。

   ![](assets/image2014-9-23-16-3a14-3a42.png)
