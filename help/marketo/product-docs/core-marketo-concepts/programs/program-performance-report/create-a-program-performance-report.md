# 建立方案效能報表 {#create-a-program-performance-report}

透過此報表檢視您計畫的良好統計資料。

1. [在方案中建立報告](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md){target="_blank"}並選取&#x200B;**[!UICONTROL 方案效能]** [報告型別](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md){target="_blank"}。
1. 按一下「**[!UICONTROL 報表]**」標籤。
1. 您到了！ 探索您的報告以檢視您的計畫如何執行。

   ![](assets/image2014-9-18-17-3a23-3a2.png)

>[!TIP]
>
>若要尋找&#x200B;*每次成功成本最低*&#x200B;的程式，例如[在該欄排序您的報表](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md){target="_blank"}，然後選擇&#x200B;**[!UICONTROL 遞增排序]**。

「方案效能」報表中的欄包括：

 <thead> 
  <tr> 
   <th>欄</th> 
   <th>說明</th> 
  </tr> 
 </thead>

  <tr>

   <td>節目的頻道型別。</td>

   <td>程式獲得的新人員。</td> 
  </tr> 
  <tr> 
   <td>成功</td> 
   <td>取得計畫狀態所定義之成功的成員數。 </td> 
  </tr> 
  <tr> 
   <td>總成本</td> 
   <td><p>程式中所有期間成本的總和。</p></td> 
  </tr> 
 </tbody> 
</table>
