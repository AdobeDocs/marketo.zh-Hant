# 建立方案效能報表 {#create-a-program-performance-report}

透過此報表檢視有關您計畫的絕佳統計資料。

1. [在方案中建立報告](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) 並選取 **計畫績效** [報告型別](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md).
1. 按一下 **報告** 標籤。
1. 您到了！ 探索您的報告，瞭解您的方案執行狀況。

   ![](assets/image2014-9-18-17-3a23-3a2.png)

>[!TIP]
>
>若要尋找具有最低 *每次成功的成本*&#x200B;例如， [排序您的報告](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md) 在該欄上並選擇 **升序排序**.

「方案績效」報表中的欄包括：

<table> 
 <thead> 
  <tr> 
   <th>欄</th> 
   <th>說明</th> 
  </tr> 
 </thead> 
 <tbody> 
  <tr> 
   <td>管道</td> 
   <td>計畫的頻道型別。</td> 
  </tr> 
  <tr> 
   <td>新名稱</td> 
   <td>計畫獲得的新人員。</td> 
  </tr> 
  <tr> 
   <td>成功</td> 
   <td>取得計畫狀態中定義之成功的成員數量。 </td> 
  </tr> 
  <tr> 
   <td>總成本</td> 
   <td><p>程式中所有期間成本的總和。</p></td> 
  </tr> 
 </tbody> 
</table>
