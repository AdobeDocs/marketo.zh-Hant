# 建立方案 {#create-a-program}

程式是Marketo中最重要的元素之一。 您將會經常使用它們！

1. 前往 **[!UICONTROL Marketing Activities]**。

   ![](assets/login-marketing-activities.png)

1. 選取新方案的資料夾。 選取「**[!UICONTROL New]**」然後按一下「**[!UICONTROL New Program]**」。

   ![](assets/leadlifecycle.jpg)

1. 輸入&#x200B;**[!UICONTROL Name]**，在下拉式清單中選取&#x200B;**[[!UICONTROL Channel]](/help/marketo/product-docs/administration/tags/create-a-program-channel.md){target="_blank"}**，然後按一下&#x200B;**[!UICONTROL Create]**。

   ![](assets/image2015-2-5-16-3a33-3a23.png)
