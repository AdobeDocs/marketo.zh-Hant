# 建立計畫 {#create-a-program}

程式是Marketo中最重要的元素之一。 您將會經常使用它們！

1. 前往 **行銷活動。**

   ![](assets/login-marketing-activities.png)

1. 選取新方案的資料夾。 選取 **新增** 並按一下 **新計畫**.

   ![](assets/leadlifecycle.jpg)

1. 輸入 **名稱**，選取 **[頻道](https://docs.marketo.com/display/DOCS/Create+a+Program+Channel)** ，然後按一下 **建立**.

   ![](assets/image2015-2-5-16-3a33-3a23.png)
