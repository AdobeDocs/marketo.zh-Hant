# 建立方案 {#create-a-program}

方案是Marketo最重要的要素之一。 你會經常用它們！

1. 前往 **行銷活動。**

   ![](assets/login-marketing-activities.png)

1. 選擇新程式的資料夾。 選擇 **新增** 按一下 **新計畫**.

   ![](assets/leadlifecycle.jpg)

1. 輸入 **名稱**，選取 **[管道](https://docs.marketo.com/display/DOCS/Create+a+Program+Channel)** 按一下 **建立**.

   ![](assets/image2015-2-5-16-3a33-3a23.png)
