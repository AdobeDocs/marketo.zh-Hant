# 編輯Analytics行為設定 {#edit-analytics-behavior-settings}

您可以設定 [管道上管理層級的analytics行為](/help/marketo/product-docs/reporting/revenue-cycle-analytics/program-analytics/make-a-program-without-a-period-cost-available-in-revenue-explorer-and-analyzers.md)，但您也可以在方案層級進行編輯。 這是方法。

1. 前往 **行銷活動**.

   ![](assets/login-marketing-activities-2.png)

1. 尋找並選取您的方案。

   ![](assets/image2014-9-24-11-3a40-3a57.png)

1. 在「設定」標籤下，將「Analytics行為」拖曳至畫布上。

   ![](assets/image2014-9-24-11-3a41-3a2.png)

1. 選取您想要的Analytics行為。

   ![](assets/image2014-9-24-11-3a42-3a0.png)

>[!NOTE]
>
>**定義**
>
>**包含**  — 此選項可確保方案可用於收入瀏覽器和分析器中的報告，而無論您是否已包括期間成本。
>
>**操作**  — 此選項會導致收入總管或分析器中未顯示方案。

>[!NOTE]
>
>預設行為（如果未套用此設定）是只有在至少需要一個期間成本（即使指派了零美元）時，才會將程式納入Analytics中。

1. 按一下 **儲存**.

   ![](assets/image2014-9-24-11-3a42-3a6.png)

幹得好！ 現在您知道如何覆寫方案層級的分析行為。

>[!NOTE]
>
>變更將於隔天生效，而此程式將可供使用或從收入總管和分析器中提取。
