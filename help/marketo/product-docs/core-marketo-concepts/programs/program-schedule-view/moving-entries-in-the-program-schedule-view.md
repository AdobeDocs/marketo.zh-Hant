# 移動程式計畫視圖中的條目 {#moving-entries-in-the-program-schedule-view}

在計畫視圖中移動條目將自動重新計畫它們。

>[!NOTE]
>
>無法移動已執行的智慧行銷活動、參與計畫或郵件爆炸。

1. 選取您的項目。 將其拖放至不同的日期。

   ![](assets/image2014-9-18-17-3a47-3a23.png)

1. 接著，我們的資料擷取會取消核准、變更日期，以及重新核准資產。 他真酷。

   ![](assets/image2014-9-18-17-3a47-3a35.png)

   現在將重新安排您的參加項目。

   ![](assets/image2014-9-18-17-3a49-3a19.png)
