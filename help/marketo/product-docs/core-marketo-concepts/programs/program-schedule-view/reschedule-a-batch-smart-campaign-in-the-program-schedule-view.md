# 在方案計畫視圖中重新計畫批智慧型促銷活動 {#reschedule-a-batch-smart-campaign-in-the-program-schedule-view}

是否要重新計畫批智慧型促銷活動運行？ 在方案排程檢視中，將它們拖放至未來。

1. 選擇要重新計畫的智慧型促銷活動。

   ![](assets/image2015-5-19-12-3a8-3a28.png)

1. 拖放至您要執行促銷活動的新日期。

![](assets/image2015-5-19-12-3a12-3a1.png)

1. 太棒了！ 智慧型促銷活動已重新排期。 時間也可從 **日期** 項目詳細資訊中的下拉式清單。

   ![](assets/image2015-5-19-12-3a15-3a38.png)
