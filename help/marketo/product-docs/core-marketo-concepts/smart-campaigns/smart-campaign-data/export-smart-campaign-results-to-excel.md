# 將Smart Campaign結果匯出至Excel {#export-smart-campaign-results-to-excel}

將您的智慧型行銷活動結果匯出至Excel，以便在Marketo外部使用。

1. 在您的智慧行銷活動中，按一下 **結果** 然後是底部的匯出圖示。

   ![](assets/exportexcel-hands.png)

   馬上即可匯出結果，隨時使用！

   >[!NOTE]
   >
   >匯出限製為20,000列。 Excel檔案會儲存在瀏覽器的預設下載資料夾中。
