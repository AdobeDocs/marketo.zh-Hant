# 將智慧型促銷活動結果匯出至Excel {#export-smart-campaign-results-to-excel}

將智慧型促銷活動結果匯出至Excel，以便在Marketo外使用。

1. 在您的智慧行銷活動中，按一下 **結果** 然後是底部的匯出圖示。

   ![](assets/exportexcel-hands.png)

   很快就會匯出結果，供您使用！

   >[!NOTE]
   >
   >匯出上限為20,000列。 Excel檔案會儲存在瀏覽器的預設下載資料夾中。
