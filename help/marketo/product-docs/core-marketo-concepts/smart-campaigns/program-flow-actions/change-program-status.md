# 變更計畫狀態 {#change-program-status}

## 概觀 {#overview}

計畫狀態可協助您追蹤人們透過計畫或事件的進度。 如需詳細資訊，請參閱 [自訂、建立和管理頻道](/help/marketo/product-docs/administration/tags/create-a-program-channel.md).

>[!CAUTION]
>
>變更參與計畫中的計畫狀態會自動將其新增到第一個流中。 他們將會開始接收內容。

## 使用情況 {#usage}

1. 拖曳至 **變更計畫狀態** 流程步驟。

   ![](assets/image2014-9-22-14-3a43-3a34.png)

1. 選取 **新狀態** 您想要設定。 如果相關人員尚未成為計畫成員，則也會成為該計畫的成員。

   ![](assets/image2014-9-22-14-3a43-3a45.png)

選項僅限於該程式的有效狀態。

>[!NOTE]
>
>個人無法回到在管理員的管道編輯器中定義的舊版計畫狀態。

狀態是追蹤人員和報告的強大工具。 這值得您的寶貴時間。
