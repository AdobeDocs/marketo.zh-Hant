# 添加到參與計畫 {#add-to-engagement-program}

## 總覽 {#overview}

您透過此流量步驟建立的智慧型行銷活動將是參與計畫的入口。

![](assets/image2014-9-22-14-3a47-3a32.png)

## 使用情況 {#usage}

1. 選取您要新增人員的參與方案。

   ![](assets/image2014-9-22-14-3a47-3a36.png)

1. 選取要將人員放入的資料流。

   ![](assets/image2014-9-22-14-3a47-3a39.png)

   >[!NOTE]
   >
   >您不能將人員添加到同一程式內的多個流。
