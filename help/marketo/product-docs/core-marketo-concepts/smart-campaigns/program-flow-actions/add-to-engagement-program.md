# 新增至參與方案 {#add-to-engagement-program}

您透過此流程步驟建立的Smart Campaign將是您參與方案的閘道。

![](assets/add-to-engagement-program-1.png)

1. 選取您要新增人員的參與計畫。

   ![](assets/add-to-engagement-program-2.png)

1. 選取您要放置人員的資料流。

   ![](assets/add-to-engagement-program-3.png)

   >[!NOTE]
   >
   >您無法將人員新增至相同方案內的多個串流。
