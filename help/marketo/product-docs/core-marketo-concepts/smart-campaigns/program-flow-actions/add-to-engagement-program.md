# 新增至參與計畫 {#add-to-engagement-program}

## 概觀 {#overview}

您透過此流程步驟建立的智慧行銷活動將是您參與方案的閘道。

![](assets/image2014-9-22-14-3a47-3a32.png)

## 使用情況 {#usage}

1. 選取您要新增人員的參與計畫。

   ![](assets/image2014-9-22-14-3a47-3a36.png)

1. 選取您要放置人員的資料流。

   ![](assets/image2014-9-22-14-3a47-3a39.png)

   >[!NOTE]
   >
   >您無法將人員新增至相同方案內的多個串流。
