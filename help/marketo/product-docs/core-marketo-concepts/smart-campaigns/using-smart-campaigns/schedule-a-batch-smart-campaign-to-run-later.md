# 排程批次智慧型促銷活動以稍後執行 {#schedule-a-batch-smart-campaign-to-run-later}

如果您想要將批次智慧型促銷活動設定為在未來某個時間點執行，以下是方法。

>[!TIP]
>
>您也可以 [在方案計畫視圖中重新計畫批智慧型促銷活動](/help/marketo/product-docs/core-marketo-concepts/programs/program-schedule-view/reschedule-a-batch-smart-campaign-in-the-program-schedule-view.md).

1. 選取您要執行的批次智慧型促銷活動，請前往 **排程** ，然後按一下 **執行一次**.

   ![](assets/schedule-a-batch-smart-campaign-to-run-later-1.png)

1. 按一下 **稍後運行**，然後按一下日曆圖示，並選取您要智慧型促銷活動執行的日期。

   ![](assets/schedule-a-batch-smart-campaign-to-run-later-2.png)

1. 選取您希望智慧型促銷活動在執行的時間（至少提前15分鐘）。

   ![](assets/schedule-a-batch-smart-campaign-to-run-later-3.png)

1. 按一下 **儲存**.

   ![](assets/schedule-a-batch-smart-campaign-to-run-later-4.png)

1. 您可以查看 **排程** 標籤。

   ![](assets/schedule-a-batch-smart-campaign-to-run-later-5.png)

   >[!NOTE]
   >
   >[排程循環批次促銷活動](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/using-smart-campaigns/schedule-a-recurring-batch-campaign.md)
