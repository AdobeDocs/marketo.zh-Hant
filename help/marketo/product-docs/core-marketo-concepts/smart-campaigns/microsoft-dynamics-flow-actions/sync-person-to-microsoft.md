# 將人員同步至Microsoft {#sync-person-to-microsoft}

此功能僅適用於整合Marketo與Microsoft Dynamics的使用者。

## 概觀 {#overview}

此流程步驟會將行銷人員建立至您的Dynamics CRM。

![](assets/one.png)

## 使用狀況 {#usage}

您可以將Dynamics使用者設為人員擁有者。

![](assets/two.png)

>[!NOTE]
>
>使用「將人員同步至Microsoft」流動動作（僅在觸發促銷活動中）時，銷售機會／連絡人會在Dynamics中即時建立。
