# 將人員同步至Microsoft {#sync-person-to-microsoft}

此功能僅適用於整合Marketo與Microsoft Dynamics的使用者。

## 總覽 {#overview}

此流程步驟會將Marketo建立的人員插入您的Dynamics CRM中。

![](assets/one.png)

## 使用情況 {#usage}

您可以將Dynamics使用者設為人員擁有者。

![](assets/two.png)

>[!NOTE]
>
>使用「將人員同步至Microsoft」流量動作時（僅限於觸發促銷活動中），銷售機會/連絡人將在Dynamics中即時建立。
