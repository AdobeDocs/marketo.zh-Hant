# 將人員同步至Microsoft {#sync-person-to-microsoft}

此功能僅適用於整合Marketo與Microsoft Dynamics的使用者。

## 概觀 {#overview}

此流程步驟會將Marketo建立的人員插入您的Dynamics CRM。

![](assets/one.png)

## 使用情況 {#usage}

您可以將Dynamics使用者設定為人員擁有者。

![](assets/two.png)

>[!NOTE]
>
>使用「將人員同步至Microsoft」流程動作時（僅適用於觸發促銷活動），系統會在Dynamics中即時建立銷售機會/聯絡人。
