# 變更資料值 {#change-data-value}

## 總覽 {#overview}

您可以使用Marketo來更新欄位的值。 對此，您會使用 **變更資料值** 流量動作。

![](assets/image2014-9-22-11-3a15-3a34.png)

>[!NOTE]
>
>您也可以封鎖欄位以不更新。 請參閱 [塊對欄位的更新](/help/marketo/product-docs/administration/field-management/block-updates-to-a-field.md) 以取得詳細資訊。

## 使用情況 {#usage}

1. 尋找並選取您要變更的值的欄位。

   ![](assets/image2014-9-22-11-3a18-3a29.png)

1. 輸入所需值，就完成了！

   ![](assets/image2014-9-22-11-3a18-3a38.png)

   >[!NOTE]
   >
   >您也可以在 **新值**.

   >[!TIP]
   >
   >您可以在 **新值** 來清除欄位。 請參閱 [清除欄位值](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/change-data-value/clear-field-values.md) 以取得詳細資訊。
