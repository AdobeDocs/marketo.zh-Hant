# 在智慧型行銷活動中使用網頁連結 {#use-a-webhook-in-a-smart-campaign}

若要使用 [webhook](https://developers.marketo.com/documentation/webhooks/)，將其新增至 [智慧型行銷活動](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md) 作為流動動作。

>[!AVAILABILITY]
>
>並非所有客戶都購買過此功能。 如需詳細資訊，請連絡您的銷售代表。

1. [建立智慧型行銷活動](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md).

   >[!NOTE]
   >
   >Webhook只能用於觸發促銷活動。

1. 前往 **流量** 標籤並拖曳 **呼叫Webhook** 流量動作。

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. 選擇Webhook。

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. 您也可以在智慧清單中使用Webhook。

   ![](assets/2017-05-02-10-54-38.png)

1. 最後，您可以在 **添加選擇** 中。

   ![](assets/image2014-9-22-15-3a8-3a13.png)
