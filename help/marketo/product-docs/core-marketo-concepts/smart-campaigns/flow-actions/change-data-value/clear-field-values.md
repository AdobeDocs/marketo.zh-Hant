# 清除欄位值 {#clear-field-values}

[變更資料值](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/change-data-value.md) 很好，但你怎麼 _移除_ 價值？ 問得好！

1. 在流程步驟中，選擇要清除的欄位並輸入 **NULL** （全大寫） **新值**.

   ![](assets/image2015-3-19-10-3a6-3a14.png)

1. 砰！ 我打賭你不知道！ 流程步驟完成後，將清除所選欄位的值。

   ![](assets/image2015-3-19-10-3a11-3a9.png)

   >[!CAUTION]
   >
   >將新值留空或僅輸入SPACE不會真正清空欄位。 必須鍵入NULL。 另外，請記住，流程步驟在執行後無法還原。
