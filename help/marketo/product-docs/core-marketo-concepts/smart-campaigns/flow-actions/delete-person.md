# 刪除人員{#delete-person}

錯誤的人有時會進入您的資料庫。 「刪除人員」流程步驟可以刪除它們。

## 概述{#overview}

在智慧型促銷活動中使用流程步驟。

![](assets/one-4.png)

>[!CAUTION]
>
>刪除人員時，其所有歷史RCE資料也將被刪除。 它無法復原。

## 用法{#usage}

當您拖曳流程步驟時，也會自動將其設定為從CRM中刪除。

![](assets/two-4.png)

您可以從Marketo刪除，而不是從CRM刪除，例如：

![](assets/three-3.png)

>[!NOTE]
>
>從您的CRM **移除人員只適用於Salesforce**。 如果您從Marketo刪除人員並選擇將他們保留在Salesforce中，則如果他們的Salesforce記錄有更新，就會在Marketo重新建立。
