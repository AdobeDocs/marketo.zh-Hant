# 刪除人員 {#delete-person}

錯誤的人員有時會進入您的資料庫。 「刪除人員」流程步驟可將其移除。

## 總覽 {#overview}

在智慧型行銷活動中使用流量步驟。

![](assets/one-4.png)

>[!CAUTION]
>
>刪除人員時，其所有歷史RCE資料也會一併刪除。 無法還原。

## 使用情況 {#usage}

拖曳至流程步驟時，系統也會自動將其設為從CRM中刪除。

![](assets/two-4.png)

您可以從Marketo中刪除，而不是從CRM中刪除，例如：

![](assets/three-3.png)

>[!NOTE]
>
>從您的CRM中移除人員 **只能與Salesforce搭配使用**. 如果您從Marketo刪除人員，並選擇將其保留在Salesforce中，則如果其Salesforce記錄更新，這些人員會在Marketo中重新建立。
