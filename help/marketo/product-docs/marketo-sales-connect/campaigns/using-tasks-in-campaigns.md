# 在行銷活動中使用任務 {#using-tasks-in-campaigns}

任務是將行銷活動傳送給聯絡人時的強大工具，特別是如果您的銷售流程是多管道/接觸。

傳送電子郵件的任務和呼叫聯絡人的任務是「行銷活動」中「任務」的最常見用法。

您也可以將「Task to Inmail」設為LinkedIn上的連絡人（如果那是您工作流程的一部分）。 您也可以進行自訂工作，提醒您邀請他們參加歡樂時光、詢問他們的慈善活動，或是在遊戲5之後登記入住等。

>[!NOTE]
>
>行銷活動中的任務最重要的一點是，您必須完成任務才能觸發下一個步驟。

任務將一律與行銷活動中的每個人相關聯。 這表示您必須進入每個人並關閉任務，其個別行銷活動才能繼續。

您可以在下列位置管理您的任務：在Campaigns 2.0中，直接從您的命令中心或從即時摘要的Tasks區段管理Campaign的Tasks標籤。 此外，您也可以從Salesforce管理您的任務。
