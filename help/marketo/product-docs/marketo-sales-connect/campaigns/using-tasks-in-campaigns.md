# 在行銷活動中使用任務 {#using-tasks-in-campaigns}

將行銷活動傳送至聯絡人時，工作是功能強大的工具，尤其是如果您的銷售流程是多管道/接觸時。

傳送電子郵件和呼叫連絡人的任務是行銷活動中最常見的任務用途。

如果您的工作流程包含「郵件任務」，也可以將「郵件傳送任務」設為LinkedIn上的聯絡人。 您也可以做自訂工作，提醒您邀請他們來到快樂的時光、詢問他們的慈善活動，或在第5場比賽後登記等。

>[!NOTE]
>
>「行銷活動」中的「工作」最重要的一點是，您必須完成工作，才能觸發下一個步驟。

任務將始終與促銷活動中的每個個人相關聯。 這表示您必須進入每個人並關閉工作，才能讓其個別促銷活動繼續。

您可以在以下位置管理您的任務：行銷活動2.0中行銷活動的任務標籤、直接從命令中心或即時摘要的任務區段。 此外，您也可以從Salesforce管理您的任務。
