# 以其他使用者身分檢視行銷活動清單 {#view-campaigns-list-as-another-user}

身為管理員，您可以以任何使用者的身分檢視行銷活動。

>[!NOTE]
>
>**需要管理員許可權**

1. 在網頁應用程式中，按一下 **行銷活動**.

   ![](assets/one-5.png)

1. 按一下 **檢視方式** 下拉式清單，然後選取所需的使用者。

   ![](assets/two-4.png)

1. 您現在會以選取的使用者身分檢視行銷活動。

   ![](assets/three-4.png)

   >[!NOTE]
   >
   >您也可以使用篩選器或搜尋功能以及檢視身分，來檢視與您最相關的專案。
