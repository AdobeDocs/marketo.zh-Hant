# 自訂版本{#customization-versions}

以下是有關對您的Marketo Sales Connect Customizations for Salesforce進行更新的詳細資訊。

**第1版**
活動欄位、聯繫人欄位、銷售線索欄位、帳戶欄位、業務機會欄位、按鈕 [和這些行銷員欄位](http://docs.marketo.com/x/wQDh)。

**1.1版**
活動欄位、連絡人欄位、銷售機會欄位、帳戶欄位、業務機會欄位、按鈕、行銷欄位現在可在Salesforce Lightning的頁面版面中使用。
