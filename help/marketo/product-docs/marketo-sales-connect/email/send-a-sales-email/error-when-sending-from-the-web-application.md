# 從Web應用程式{#error-when-sending-from-the-web-application}發送時出錯

通常，當從[Web應用程式](https://toutapp.com/login)產生錯誤時，這是因為電子郵件或動態欄位格式發生問題。 請確定所有項目都已正確填入合成視窗，然後再試一次傳送。
