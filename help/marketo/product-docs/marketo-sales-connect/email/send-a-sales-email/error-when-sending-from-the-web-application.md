# 從Web應用程式發送時出錯 {#error-when-sending-from-the-web-application}

通常，當從Web應用程式產 [生錯誤](http://toutapp.com/login)，這是由於電子郵件或動態欄位格式問題所造成。 請確定所有項目都已正確填入合成視窗，然後再試一次傳送。
