# 如何安裝ToutApp瀏覽器擴充功能 {#how-to-install-the-toutapp-browser-extension}

1. 按一下 **設定** 於 [網頁應用程式](https://toutapp.com/next#settings).

1. 在我的帳戶底下，按一下 **電子郵件設定**.

1. 按一下 **外掛程式**.

1. 在Gmail和Google應用程式底下，按一下 **安裝**.
