# 如何安裝ToutApp瀏覽器擴充功能 {#how-to-install-the-toutapp-browser-extension}

1. 在[網頁應用程式](https://toutapp.com/next#settings)上按一下&#x200B;**設定**。

1. 在「我的帳戶」底下，按一下&#x200B;**電子郵件設定**。

1. 按一下&#x200B;**外掛程式**。

1. 在Gmail與Google Apps底下，按一下&#x200B;**安裝**。
