# 其他管理員 {#additional-admins}

要更改或刪除團隊成員，請轉至「設定」頁，然後按一下 **團隊管理** ，即可取得Advertising Cloud的說明。
身為帳戶的管理員，您可以在此管理團隊中的人員。
