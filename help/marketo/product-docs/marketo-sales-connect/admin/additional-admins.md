# 其他管理員 {#additional-admins}

若要變更或移除團隊成員，請移至[設定]頁面，然後按一下左側導覽面板中的[團隊管理]。
**&#x200B;**&#x200B;身為帳戶的管理員，您可以在這裡管理團隊中的人員。
