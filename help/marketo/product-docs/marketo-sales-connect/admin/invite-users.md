# 邀請使用者 {#invite-users}

新增使用者既快速又簡單！

1. 按一下齒輪圖示並選取&#x200B;**[!UICONTROL Settings]**。

   ![](assets/one.png)

1. 在[!UICONTROL Admin Settings]下，選取&#x200B;**[!UICONTROL User Management]**。

   ![](assets/invite-team-members-2.png)

1. 按一下「**[!UICONTROL Invite Users]**」。

   ![](assets/invite-team-members-3.png)

1. 輸入您要新增之個人的電子郵件地址，然後按一下&#x200B;**[!UICONTROL Next]**。

   ![](assets/four.png)

   >[!NOTE]
   >
   >依預設，所有新成員都會新增至「所有人」團隊。

1. 按一下「**[!UICONTROL OK]**」。

   ![](assets/five.png)
