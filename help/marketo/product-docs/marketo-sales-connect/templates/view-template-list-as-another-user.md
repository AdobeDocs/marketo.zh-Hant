# 以其他用戶身份查看模板清單 {#view-template-list-as-another-user}

作為管理員，您可以以任何用戶身份查看模板。

>[!NOTE]
>
>**需要管理權限**

1. 按一下 **模板**。

   ![](assets/one.png)

1. 按一下 **查看方式** 下拉並選擇所需用戶。

   ![](assets/two.png)

1. 您現在正以選定用戶的身份查看模板。

   ![](assets/three.png)

   >[!NOTE]
   >
   >您還可以使用過濾器或搜索功能以及「查看方式」來查看與您最相關的內容。
