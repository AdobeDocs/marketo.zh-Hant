# 自訂範本檢視 {#customize-template-view}

從**檢視**下拉式清單中，您可以選擇查看：所有範本、您的範本、共用範本或未使用的範本（過去90天未使用的範本）。

此外，您可以按一下「檢視」右側的設定圖示，在範本檢視中新增／移除 **欄。**
