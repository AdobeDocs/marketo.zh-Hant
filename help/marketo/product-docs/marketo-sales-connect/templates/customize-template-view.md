# 自訂範本視圖 {#customize-template-view}

從&#x200B;**[!UICONTROL View]**&#x200B;下拉式清單中，您可以選擇檢視：所有範本、您的範本、共用範本，或未使用的範本（過去90天內未使用的範本）。
