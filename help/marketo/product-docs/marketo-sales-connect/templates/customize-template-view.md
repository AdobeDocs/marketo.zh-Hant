# 自訂範本檢視 {#customize-template-view}

從 **檢視** 從下拉式清單中，您可以選擇檢視：所有範本、您的範本、共用範本或未使用的範本（過去90天內未使用的範本）。

此外，您可以按一下右側的設定圖示，在範本檢視中新增/移除欄 **檢視。**
