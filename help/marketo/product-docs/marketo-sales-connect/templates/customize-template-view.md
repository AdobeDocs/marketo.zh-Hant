# 自訂範本檢視 {#customize-template-view}

從 **檢視** 下拉式清單中，您可以選擇查看：所有範本、您的範本、共用範本或未使用的範本（過去90天內未使用的範本）。

此外，您可以按一下範本檢視右側的設定圖示，以新增/移除欄 **檢視。**
