# 編輯範本類別 {#edit-a-template-category}

我們目前無法刪除整個類別或大量移動範本至新類別。

您可以做的是，將類別中的每個範本編輯成現有的類別名稱或全新的名稱。 這會將您所有現有的範本移至不同的資料夾。 資料夾一旦清空，就會消失。

我們正在努力新增整體執行此動作的功能。
