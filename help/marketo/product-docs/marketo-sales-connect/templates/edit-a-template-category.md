# 編輯範本類別 {#edit-a-template-category}

目前無法刪除整個類別或將範本大量移至新類別。

您可以編輯類別中的每個範本，使其變成現有類別名稱或全新名稱。 這會將您所有現有的範本移至不同的資料夾。 資料夾一旦空白，就會消失。
