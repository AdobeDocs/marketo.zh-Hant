# 編輯模板類別{#edit-a-template-category}

我們目前無法刪除整個類別或大量將範本移至新類別。

您可以做的是，將「類別」中的每個範本編輯為現有的「類別」名稱或全新名稱。 這會將您所有現有的範本移至另一個檔案夾。 資料夾一旦空白，就會消失。

我們正在努力增加集體完成這項任務的能力。
