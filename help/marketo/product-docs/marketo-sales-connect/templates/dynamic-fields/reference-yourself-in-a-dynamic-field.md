# 在動態欄位中自我參照 {#reference-yourself-in-a-dynamic-field}

動態欄位是我們的合併列印形式。

我們可讓您使用預先定義的屬性，將電子郵件範本個人化，例如 `{{my_name}}` 或 `{{my_title}}`. 這些欄位可讓您在電子郵件範本中快速參考自己。

若要將自我參照動態欄位新增至範本，只需從以下選取您要使用的動態欄位： **mse動態欄位** 按鈕。
