# 在動態欄位中自我參照 {#reference-yourself-in-a-dynamic-field}

動態欄位是合併列印的形式。

我們允許您使用`{{my_name}}`或`{{my_title}}`等預先定義的屬性，個人化您的電子郵件範本。 這些欄位可讓您在電子郵件範本中快速參考自己。

若要將自我參照動態欄位新增至範本，只要從範本編輯器的&#x200B;**MSE動態欄位**&#x200B;按鈕中選取您要使用的欄位即可。
