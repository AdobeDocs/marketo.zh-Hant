# 在動態欄位中參考自己 {#reference-yourself-in-a-dynamic-field}

動態欄位是我們合併郵件的形式。

我們允許您使用預先定義的屬性（例如或）個人化您的 `{{my_name}}` 電子郵件范 `{{my_title}}`本。 這些欄位可讓您在電子郵件範本中快速參考自己。

要向模板添加自引用的動態欄位，只需從模板編輯器的 **** MSE動態欄位按鈕中選擇要使用的動態欄位。
