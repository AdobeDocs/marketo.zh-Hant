# 在動態欄位{#reference-yourself-in-a-dynamic-field}中參考您自己

動態欄位是我們合併郵件的形式。

我們允許您使用預先定義的屬性（例如`{{my_name}}`或`{{my_title}}`）個人化您的電子郵件範本。 這些欄位可讓您在電子郵件範本中快速參考自己。

要向模板添加自引用的動態欄位，只需從模板編輯器的&#x200B;**MSE動態欄位**&#x200B;按鈕中選擇要使用的欄位。
