# 在動態欄位中參考自己 {#reference-yourself-in-a-dynamic-field}

動態欄位是我們的郵件合併形式。

我們可讓您使用預先定義的屬性(例如 `{{my_name}}` 或 `{{my_title}}`. 這些欄位可讓您在電子郵件範本中快速參考自己。

若要將自我參考的動態欄位新增至範本，只需從 **MSE動態欄位** 按鈕。
