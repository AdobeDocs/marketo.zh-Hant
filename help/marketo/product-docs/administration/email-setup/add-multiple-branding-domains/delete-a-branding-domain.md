# 刪除品牌域 {#delete-a-branding-domain}

1. 前往 **管理** 的上界。

   ![](assets/delete-a-branding-domain-1.png)

1. 按一下 **電子郵件**.

   ![](assets/delete-a-branding-domain-2.png)

1. 在「品牌域」(Branding Domains)表格中，選擇要刪除的域，然後按一下 **刪除**.

   ![](assets/delete-a-branding-domain-3.png)

   >[!NOTE]
   >
   >如果要刪除主要品牌域，必須首先選擇其他品牌域作為主要品牌域。
