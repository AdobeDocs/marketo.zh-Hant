# 建立人員分區 {#create-a-person-partition}

按照這些步驟建立新的人員分區。

>[!NOTE]
>
>**需要管理權限**

>[!NOTE]
>
>先了解 [了解工作區和人員分區](/help/marketo/product-docs/administration/workspaces-and-person-partitions/understanding-workspaces-and-person-partitions.md).

1. 前往 **管理** 的上界。

   ![](assets/create-a-person-partition-1.png)

1. 按一下 **工作區與分區**.

   ![](assets/create-a-person-partition-2.png)

1. 前往 **人員分區** 按一下 **新人員分區**.

   ![](assets/create-a-person-partition-3.png)

1. 為分區命名，選擇 **工作區** 顯示位置，然後按一下 **建立**.

   ![](assets/create-a-person-partition-4.png)

建立分區後，應該會看到更新。

![](assets/create-a-person-partition-5.png)
