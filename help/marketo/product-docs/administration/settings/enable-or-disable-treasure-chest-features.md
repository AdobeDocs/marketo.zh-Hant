# 啟用或停用藏寶箱功能 {#enable-or-disable-treasure-chest-features}

Treasure Check包含尚未完全支援的實驗功能。

>[!NOTE]
>
>**需要管理員權限**

1. 前往「**[!UICONTROL Admin]**」區域。

   ![](assets/enable-or-disable-treasure-chest-features-1.png)

1. 按一下「**[!UICONTROL Treasure Chest]**」。

   ![](assets/enable-or-disable-treasure-chest-features-2.png)

1. 按一下您要啟用或停用功能的&#x200B;**[!UICONTROL Edit]**。

   ![](assets/enable-or-disable-treasure-chest-features-3.png)

1. 勾選&#x200B;**[!UICONTROL Enabled]**&#x200B;方塊以啟用，或取消勾選以停用，然後按一下&#x200B;**[!UICONTROL Save]**。

   ![](assets/enable-or-disable-treasure-chest-features-4.png)

   >[!TIP]
   >
   >您可能需要登出並重新登入Marketo，變更才會生效。
