# 僅為Adobe IMS啟用訂閱新增API使用者 {#add-api-only-user-for-adobe-ims-enabled-subscriptions}

雖然Marketo Engage行銷使用者和管理員是在Adobe Admin Console中進行管理，但您只能在Marketo Engage中建立和管理Marketo EngageAPI使用者。

以下步驟說明如何在Marketo Engage中新增「僅限API使用者」。 在這之前，你必須 [建立僅API角色](/help/marketo/product-docs/administration/users-and-roles/create-an-api-only-user-role.md).

1. 在Marketo中，按一下 **管理** 選取 **使用者與角色**.

   ![](assets/add-api-only-user-for-adobe-ims-1.png)

1. 按一下 **僅建立API用戶**.

   ![](assets/add-api-only-user-for-adobe-ims-2.png)

1. 僅為API使用者輸入電子郵件、名字和姓氏。  選取您要指派給使用者的API僅限角色。 按一下 **僅建立API用戶** 時才能使用。

   ![](assets/add-api-only-user-for-adobe-ims-3.png)

>[!NOTE]
>
>動作成功後，僅建立API使用者強制回應將會關閉，且使用者清單會重新整理，新使用者就會顯示。
