# User與Adobe ID登錄 {#user-sign-in-with-adobe-id}

當具有Adobe身份的用戶需要登錄到Marketo Engage應用程式時，他們必須通過Adobe ID登錄連結登錄，而不是通過Marketo登錄頁上的典型登錄。 按一下該連結後，用戶將被定向到Marketo Engage應用程式。

1. 按一下 **使用Adobe ID登錄** 連結。

   ![](assets/user-sign-in-with-adobe-id-1.png)

1. 輸入Adobe憑據並按一下 **繼續**。

   ![](assets/user-sign-in-with-adobe-id-2.png)

成功登錄後，您將被定向到Marketo Engage應用程式。
