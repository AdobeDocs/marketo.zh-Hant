# 最低系統需求 {#minimum-system-requirements}

如果您想知道登入和使用Marketo需要什麼，以下是您想要的資訊。

>[!NOTE]
>
>iOS和Android尚不受支援。

* 運行Windows、OS X或Linux的電腦或筆記型電腦
* 鍵盤和滑鼠
* 最低解析度為1280 x 768的顯示器
* 最新瀏覽器，請參閱 [受支援的瀏覽器](/help/marketo/product-docs/administration/setup-administration/supported-browsers.md)詳細資訊{target=&quot;_blank&quot;}
* 網際網路連線

>[!NOTE]
>
>如果您的IT團隊使用允許清單限制Web存取，請要求他們新增 `*.marketo.com` 允許所有Marketo資源和網路通訊端。
