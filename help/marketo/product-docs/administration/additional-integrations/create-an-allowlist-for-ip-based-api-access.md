# 建立IP型API存取允許清單 {#create-an-allowlist-for-ip-based-api-access}

有時候，您只想授予API特定IP位址或位址範圍的存取權。 若要這麼做，請先啟用限制，然後指定允許使用API的IP位址。

>[!NOTE]
>
>**需要管理權限**

1. 前往 **管理** 的上界。

   ![](assets/create-an-allowlist-for-ip-based-api-access-1.png)

1. 按一下 **網站服務**.

   ![](assets/create-an-allowlist-for-ip-based-api-access-2.png)

1. 在「IP限制」區域中，按一下 **編輯、** 或按一下 **編輯IP限制** 左上角。

   ![](assets/create-an-allowlist-for-ip-based-api-access-3.png)

1. 檢查 **啟用IP限制** 框中，輸入要允許清單的IP地址。

   ![](assets/create-an-allowlist-for-ip-based-api-access-4.png)

   >[!NOTE]
   >
   >您可以輸入單一IP位址或其範圍，或使用萬用字元。

1. 按一下 **新增** 開啟其他欄位以輸入更多IP位址。

   ![](assets/create-an-allowlist-for-ip-based-api-access-5.png)

1. 按一下 **儲存**.

   ![](assets/create-an-allowlist-for-ip-based-api-access-6.png)
