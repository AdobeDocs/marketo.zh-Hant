# 將[!DNL Level 3 Web Meeting]新增為[!DNL LaunchPoint]服務 {#add-level-three-web-meeting-as-a-launchpoint-service}

Marketo會管理您的[!DNL Level 3 Web Meeting]註冊和出席情況。

>[!NOTE]
>
>**需要管理員權限**

>[!NOTE]
>
>此步驟需要[!DNL Level 3 Web Meeting]的現有訂閱和管理許可權。 準備好您的存取號碼、存取碼和密碼。

1. 前往「**[!UICONTROL Admin]**」區域。

   ![](assets/add-level-three-web-meeting-as-a-launchpoint-service-1.png)

1. 按一下「**[!UICONTROL LaunchPoint]**」。

   ![](assets/add-level-three-web-meeting-as-a-launchpoint-service-2.png)

1. 選取&#x200B;**[!UICONTROL New]**，然後選取&#x200B;**[!UICONTROL New Service]**。

   ![](assets/add-level-three-web-meeting-as-a-launchpoint-service-3.png)

1. 輸入&#x200B;**[!UICONTROL Display Name]**。 在&#x200B;**[!UICONTROL Service]**&#x200B;下，選取&#x200B;**[!UICONTROL Level 3 Web Meeting]**。

   ![](assets/add-level-three-web-meeting-as-a-launchpoint-service-4.png)

1. 輸入您的&#x200B;**[!UICONTROL Access Number]**、**[!UICONTROL Access Code]**&#x200B;和&#x200B;**[!UICONTROL Passcode]**，然後按一下&#x200B;**[!UICONTROL Create]**。

   ![](assets/add-level-three-web-meeting-as-a-launchpoint-service-5.png)

您的[!DNL Level 3 Web Meeting]帳戶現已與Marketo同步！

![](assets/add-level-three-web-meeting-as-a-launchpoint-service-6.png)
