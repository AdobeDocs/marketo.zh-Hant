# 建立指名帳戶 {#create-a-named-account}

請依照下列步驟手動建立指名帳戶。

1. 在「命名帳戶」中，按一 **下「新** 」下拉式清單，然後選 **取「建立命名帳戶」**。

   ![](assets/two-1.png)

1. 填寫所需欄位，然後按一下「 **建立**」。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接按一下指名帳戶，即可查看其控制面板。
