# 將現有的命名帳戶新增至帳戶清單 {#add-an-existing-named-account-to-an-account-list}

將指名帳戶新增至帳戶清單很簡單。

>[!NOTE]
>
>這僅適用於「帳戶清單」，而 **非** 「動態帳戶清單」。

1. 選擇要添加到的已命名帳戶的行。

   ![](assets/four-1.png)

1. 按一下「 **命名帳戶動作** 」下拉式清單，並選 **取「新增至帳戶清單」**。

   ![](assets/five-1.png)

1. 按一下「 **帳戶清單** 」下拉式清單，選取所需的帳戶清單，然後按一下「 **新增」**。

   ![](assets/six-1.png)

   就這樣！
