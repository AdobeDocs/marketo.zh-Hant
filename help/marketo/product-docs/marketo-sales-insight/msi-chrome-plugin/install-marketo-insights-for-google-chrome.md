# 安裝Google Chrome適用的Marketo Insights {#install-marketo-insights-for-google-chrome}

請依照下列步驟，開始使用功能強大的Chrome附加元件。

>[!NOTE]
>
>您不需要是Marketo管理員即可安裝擴充功能。

1. 安裝 [Marketo Insights for Google Chrome Add擴充功能](https://chrome.google.com/webstore/detail/marketo-for-google-mail/jjkfbhajlmoeegbjgjipliamplidmbjb) 從Chrome網站商店取得。

   ![](assets/image2015-10-5-10-3a24-3a7.png)

1. 以滑鼠右鍵按一下Chrome中的Marketo標誌，然後選取 **選項**.

   ![](assets/two.png)

1. 輸入 **登錄代碼**, **電子郵件地址**, **名字**，和 **姓氏**. 按一下 **驗證並儲存**.

   ![](assets/three.png)

   >[!CAUTION]
   >
   >請務必使用 **主要電子郵件帳戶** 註冊時，因為我們不支援對此插件使用別名。

   >[!NOTE]
   >
   >登錄程式碼會在Marketo管理員之後傳送的電子郵件中 [向您發放Marketo電子郵件增益集授權](/help/marketo/product-docs/marketo-sales-insight/msi-outlook-plugin/issue-a-marketo-email-add-in-license.md) [.](https://docs.marketo.com/pages/viewpage.action?pageid=7510848) **14天後過期**.

1. 按一下 **允許** 允許離線存取。

   ![](assets/image2015-10-5-10-3a34-3a1.png)
