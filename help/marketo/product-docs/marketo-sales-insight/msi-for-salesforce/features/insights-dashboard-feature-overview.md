# 前瞻分析控制面板功能概述{#insights-dashboard-feature-overview}

進一步瞭解Sales Insights儀表板中的可用功能。

>[!PREREQUISITES]
>
>您必須擁有最新的MSI SFDC軟體包和[configuration](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/configuration-for-existing-customers.md)。

![](assets/one.png)

## 連絡版面{#contact-layout}

**參與速度格線**

* 此智慧型網格包含過去90天中的「有趣時刻」、「電子郵件」和「網頁活動」
* 使用者可選擇「顯示帳戶」活動，這會在連絡人檢視中，將所有帳戶層級的「有趣時刻」、「電子郵件」和「Web活動」層級分層
* 使用者可以反白標示特定周，以檢視該周內的活動
* 預設檢視：選取目前的周

**參與追溯和摘要**

* 深入瞭解活動資訊卡中的有趣時刻、電子郵件和網頁活動
* 有趣的瞬間活動卡片——包含訂閱選項
* 電子郵件活動卡——包含預覽選項
* Web活動卡——包含點按連結的功能
* 「每週摘要」列會顯示該周的「有趣時刻」、「電子郵件」和「網頁活動」。 每個圖示都可點按，可用作顯示特定活動的篩選器
* 預設檢視：這是當前視圖的活動清單

**近期的電子郵件促銷活動和活動**

電子郵件促銷活動標籤：

* 包含屬於未來90天內排程之電子郵件方案或預設方案的促銷活動
* 非特定於連絡人／潛在客戶（亦即，促銷活動清單是Marketo例項中排程之所有電子郵件促銷活動的一般清單）。 所有銷售機會、連絡人、帳戶和銷售機會面板的促銷活動清單將相同
* 如果促銷活動週期在接下來的90天內執行超過3次，則在任何指定時間點（類似於Marketo中的行為）只會顯示接下來的3次執行
* 此區段中的活動詳細資料卡片將提供預覽選項。 如果流程包含多個傳送電子郵件步驟，所有電子郵件都可供預覽。 在傳送電子郵件流程步驟中，如果有多個「電子郵件選項」，預設選項將可供預覽
* 篩選條件包括「搜尋」和「日期範圍」

![](assets/three.png)

事件標籤：

* 包含排程在未來90天內的活動計畫
* 使用篩選選項可查看所有事件／受邀事件（根據「管理員」設定）
* 選擇邀請的事件將顯示已邀請特定聯繫人的事件以及成員狀態
* 選取所有事件時，會顯示未來90天內排程的事件清單
* 此區段中的活動詳細資料卡片具有預覽選項
* 篩選條件包括「搜尋」、「僅顯示受邀事件」和「日期範圍」

![](assets/two.png)
