# 前瞻分析控制面板功能概觀 {#insights-dashboard-feature-overview}

進一步了解Sales Insights控制面板中可用的功能。

>[!PREREQUISITES]
>
>您必須擁有最新的MSI SFDC包和[configuration](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/configuration/configuration-for-existing-customers.md)。

![](assets/one.png)

## 連絡人版面 {#contact-layout}

**參與速度網格**

* 此智慧電網包括過去90天內的「有趣時刻」、「電子郵件」和「Web活動」
* 使用者可以選擇「顯示帳戶」活動，這會在連絡人檢視中分層所有帳戶層級的「有趣時刻」、「電子郵件」和「Web活動」
* 使用者可以反白標示特定周，以查看該周內的活動
* 預設視圖：已選取目前的周

**參與細化和摘要**

* 深入研究活動卡片以找出有趣的時刻、電子郵件和Web活動
* 「有趣時刻」活動卡 — 包括訂閱選項
* 電子郵件活動卡片 — 包含預覽選項
* Web活動卡 — 包含點按連結的功能
* 每週摘要列會顯示該周的有趣時刻、電子郵件和網路活動。 每個圖示都可點按，並可作為篩選器來顯示特定活動
* 預設視圖：這是目前檢視的活動清單

**近期的電子郵件行銷活動和事件**

電子郵件行銷活動標籤：

* 包括屬於電子郵件方案或預設方案，且已在未來90天內排程的促銷活動
* 不特定於連絡人/銷售機會(亦即，促銷活動清單是Marketo例項中排程之所有電子郵件促銷活動的一般清單)。 所有銷售機會、聯絡人、帳戶和銷售機會面板的促銷活動清單將相同
* 如果促銷活動重複發生在接下來90天內執行超過三次，則在任何指定時間點(類似於Marketo中的行為)都只會顯示接下來的三次執行
* 此區段中的活動詳細資料卡會有預覽選項。 如果流程包含多個傳送電子郵件步驟，則所有電子郵件都可預覽。 在傳送電子郵件流程步驟中，如果有多個「電子郵件選擇」，則預設選項將可供預覽
* 篩選器包括「搜尋」和「日期範圍」

![](assets/three.png)

事件標籤：

* 包括未來90天內排程的事件方案
* 使用篩選選項可查看所有事件/受邀事件（根據管理設定）
* 選擇邀請的事件將顯示特定聯繫人已被邀請的事件以及成員狀態
* 選取所有事件會顯示未來90天內排程的事件清單
* 此區段中的活動詳細資訊卡有預覽選項
* 篩選條件包括「搜尋」、「僅顯示受邀事件」和「日期範圍」

![](assets/two.png)
