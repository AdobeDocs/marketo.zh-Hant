# 最佳下注{#best-bets}

「最佳下注」標籤包含您所有熱門客源的清單，這些潛在客源會根據其優先順序（使用緊急程度和相對分數計算）列出。

按一下「動作」欄下的資料選單，即可使用參與選項，例如：

* 傳送行銷人員至電子郵件
* 新增至Marketor Campaign
* 在Marketo中檢視
* 新增至監視清單

您也可以從「最佳下注」標籤中選擇多個潛在客戶，然後選擇「傳送行銷人員電子郵件」_或「新增至行銷人員促銷活動」_。__
