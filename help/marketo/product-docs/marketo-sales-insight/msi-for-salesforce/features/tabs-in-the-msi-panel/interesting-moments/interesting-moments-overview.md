# 有趣的時刻概述 {#interesting-moments-overview}

您可以利用有趣的時間流步驟，讓您的銷售團隊能夠洞察您的主管在智慧行銷活動中執行的很酷的事情。

![](assets/image2016-1-27-11-3a1-3a53.png)

1. 選擇要使用的有趣時刻類型。

   ![](assets/image2014-9-23-16-3a30-3a33.png)

1. 定義您希望銷售團隊查看的文本。

   ![](assets/image2014-9-23-16-3a30-3a53.png)

>[!TIP]
>
>**越少越多**. 與您的銷售團隊合作，確保有趣的時刻真的有趣。

您也可以在有趣的時刻使用代號，以製作真正有用的動態說明。
