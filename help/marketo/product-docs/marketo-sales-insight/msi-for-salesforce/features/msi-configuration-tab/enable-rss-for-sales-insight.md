# 啟用Sales Insight的RSS {#enable-rss-for-sales-insight}

>[!NOTE]
>
>**需要管理員許可權**

如果Marketo使用者不僅想要在Salesforce中、也想要在RSS摘要中檢視其銷售機會摘要，Marketo管理員必須先將其啟用。 操作很簡單。

1. 在「我的Marketo」中，按一下&#x200B;**管理員**，然後按&#x200B;**銷售分析**。

   ![](assets/set-up-rss-1-hands.png)

1. 在[設定]上，按一下[編輯]。**** 請注意，RSS摘要會顯示為&#x200B;**已停用**。

   ![](assets/rss-settings-tab.png)

1. 在[編輯設定]對話方塊中，勾選&#x200B;**RSS摘要**&#x200B;核取方塊，然後按一下[儲存]。****

   ![](assets/rss-edit-settings-2-hands.png)

   RSS摘要現在顯示為&#x200B;**已啟用**。

   ![](assets/rss-final-box.png)

   小菜一碟！
