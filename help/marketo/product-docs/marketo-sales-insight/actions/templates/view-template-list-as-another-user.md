# 以其他使用者身分檢視範本清單 {#view-template-list-as-another-user}

身為管理員，您可以以任何使用者的身分檢視範本。

>[!NOTE]
>
>**需要管理員許可權**

1. 按一下 **範本**.

   ![](assets/view-template-list-as-another-user-1.png)

1. 按一下 **檢視方式** 下拉式清單，然後選取所需的使用者。

   ![](assets/view-template-list-as-another-user-2.png)

1. 您現在正在以選取的使用者身分檢視範本。

   ![](assets/view-template-list-as-another-user-3.png)

   >[!NOTE]
   >
   >您也可以使用篩選器或搜尋功能以及 _檢視方式_ 以檢視與您最相關的內容。
