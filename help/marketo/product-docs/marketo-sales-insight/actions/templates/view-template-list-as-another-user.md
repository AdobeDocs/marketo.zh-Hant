# 以其他用戶身份查看模板清單 {#view-template-list-as-another-user}

作為管理員，您可以以任何用戶身份查看模板。

>[!NOTE]
>
>**需要管理權限**

1. 按一下 **模板**。

   ![](assets/view-template-list-as-another-user-1.png)

1. 按一下 **查看方式** 下拉並選擇所需用戶。

   ![](assets/view-template-list-as-another-user-2.png)

1. 您現在正以選定用戶的身份查看模板。

   ![](assets/view-template-list-as-another-user-3.png)

   >[!NOTE]
   >
   >還可以使用篩選器或搜索函式以及 _查看方式_ 來查看與你最相關的內容。
