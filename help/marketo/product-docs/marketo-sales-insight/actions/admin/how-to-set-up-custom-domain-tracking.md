# 如何設定自訂網域追蹤 {#how-to-set-up-custom-domain-tracking}

我們提供為您的連結設定自訂網域追蹤的選項，因此當有人將游標暫留在連結上時，系統會顯示 `go.yourcompany.com` 而非 `go.toutapp.com`.

若要設定此設定，您的IT管理員需遵循 [設定頁面](https://toutapp.com/custom_tracking_domain).

完成此設定後，我們會允許將您電子郵件中的任何連結顯示為 `go.yourcompany.com`.
