# 位置跟蹤是如何工作的？ {#how-does-location-tracking-work}

如果我們對您的收件人的位置錯誤，則可能發生的情況有三種可能：

 — 如果您的收件人已登錄到無線網路，並且公司的IT總部位於其他城市或國家/地區，則我們將記錄該位置，而不是其當前區域設定。

 — 如果您的電子郵件是由「抄送」或「密件抄送」欄位中的某人轉發或開啟的，我們會嘗試準確顯示這些人的位置。

總的來說，我們會根據IP跟蹤盡可能準確地確定位置，但不幸的是，這並不是一門精確的科學。
