# 位置追蹤的運作原理是什麼？ {#how-does-location-tracking-work}

如果我們誤解收件者的位置，以下是可能發生事件的三種可能性：

 — 如果您的收件者登入的IT總部位於其他城市或國家的公司的無線網路，我們會記錄該位置，而非其目前的地區設定。

 — 如果您的電子郵件是由「副本抄送」或「密件抄送」欄位中的某人轉寄或開啟，我們會嘗試精確顯示這些人員的位置。

整體而言，我們會根據IP追蹤儘可能準確地判斷位置，但遺憾的是，這並不是精確的科學。
