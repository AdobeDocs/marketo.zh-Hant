# 位置追蹤如何運作？ {#how-does-location-tracking-work}

如果我們對您的收件者找錯位置，以下是可能發生的情況的三種可能性：

 — 如果您的收件人已登錄到位於其他城市或國家/地區的公司的無線網路，我們將登錄該位置，而不是其當前地區。

 — 如果電子郵件被「抄送」或「密件抄送」欄位中的某人轉發或開啟，我們會嘗試準確顯示這些人的位置。

總的來說，我們會根據IP追蹤盡可能精確地判斷位置，但可惜這並非精確的科學。
