# 如何允許在我的伺服器上列出Sales Insight操作？ {#how-can-i-allowlist-sales-insight-actions-on-my-server}

很遺憾，無法提供一個靜態IP地址。 Sales Insight Actions部署到雲中，利用200多台伺服器處理電子郵件和用戶請求。 而且提供一個非常廣闊的範圍是不安全的。
