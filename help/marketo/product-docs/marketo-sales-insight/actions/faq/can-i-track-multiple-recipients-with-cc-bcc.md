# 我可以使用CC/BCC追蹤多個收件者嗎？ {#can-i-track-multiple-recipients-with-cc-bcc}

您可以像一般在電子郵件中一樣，透過CC和BCC個人。

但是，在追蹤方面，我們的服務無法區分開啟您電子郵件的使用者（無論是原始收件者或複製的人員）。 您偶爾可以根據使用者的位置和瀏覽器來猜測，但這並非精確的科學。
