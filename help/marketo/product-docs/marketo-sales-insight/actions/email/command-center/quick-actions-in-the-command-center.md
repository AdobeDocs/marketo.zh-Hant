# 命令中心中的快速操作 {#quick-actions-in-the-command-center}

電子郵件網格中存在兩種類型的快速操作列。 「電子郵件操作」（允許您對電子郵件採取操作）和「後續操作」（允許您通過幾次快速按一下來採取接洽操作）。

## 快速操作 {#quick-actions}

電子郵件快速操作是動態的，根據您正在查看的電子郵件狀態的最相關操作進行更新。 每個電子郵件狀態最多可以看到兩個電子郵件快速操作。 您可以查看下面每種狀態都可以使用哪些電子郵件快速操作。

**已交付**

| 狀態 | 說明 |
|---|---|
| 存檔 | 將電子郵件添加到存檔資料夾並停止所有禁用視圖，然後按一下該電子郵件的跟蹤。 |
| 成功 | 將在模板分析中將電子郵件標籤為成功報告。 |

**後續操作**

![](assets/quick-actions-in-the-command-center-1.png)

| 函式 | 說明 |
|---|---|
| 發送跟進電子郵件 | 開啟合成窗口，並添加選定的內聯電子郵件正文，並準備發送給收件人。 |
| 撥打 | 開啟銷售電話以呼叫電子郵件收件人。 |
| 發送InMail | 重定向到LinkedIn，以向此人發送InMail郵件。 |
| 建立任務 | 開啟建立任務彈出窗口以建立提醒任務。 |
