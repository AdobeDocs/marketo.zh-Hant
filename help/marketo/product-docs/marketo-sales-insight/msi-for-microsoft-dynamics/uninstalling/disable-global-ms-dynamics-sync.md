# 禁用全局MS Dynamics同步 {#disable-global-ms-dynamics-sync}

請按照以下簡單步驟禁用MS Dynamics同步。

1. 在Marketo中，按一下 **管理**.

   ![](assets/one.png)

1. 在整合下方，按一下 **Microsoft Dynamics**.

   ![](assets/two.png)

1. 按一下 **禁用同步**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >如果您的執行個體中未顯示「停用同步」按鈕，請聯絡 [Marketo支援](https://nation.marketo.com/t5/Support/ct-p/Support).
