# 停用全域MS Dynamics同步 {#disable-global-ms-dynamics-sync}

請依照下列簡單步驟停用MS Dynamics同步處理。

1. 在Marketo中，按一下 **管理員**.

   ![](assets/one.png)

1. 在整合底下，按一下 **Microsoft Dynamics**.

   ![](assets/two.png)

1. 按一下 **停用同步**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >如果您的執行個體中沒有顯示「停用同步」按鈕，請聯絡 [Marketo支援](https://nation.marketo.com/t5/Support/ct-p/Support).
