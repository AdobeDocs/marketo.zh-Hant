# 網站活動 {#web-activities}

「Web活動」頁簽顯示您的銷售機會/聯繫人的Web活動。
參考頁面造訪計數和個別帳戶，檢閱您潛在客戶的最新Web活動。 您可以篩選結果以限制為指定的頁數。

![](assets/web-activities-1.png)

## 匿名Web活動 {#anonymous-web-activities}

「匿名Web活動」頁簽顯示 **匿名** 網頁訪客網站活動。 檢閱引用頁面造訪計數的最新網頁活動。
