# 網站活動 {#web-activities}

Web活動標籤會顯示您的潛在客戶/聯絡人的網頁活動。
檢視潛在客戶的最新網頁活動，說明頁面瀏覽計數和個別帳戶。 您可以篩選結果，以限製為指定的頁數。

![](assets/web-activities-1.png)

## 匿名網路活動 {#anonymous-web-activities}

「匿名網路活動」標籤會顯示所有 **匿名** 網頁訪客網站活動。 檢閱引證頁面瀏覽計數的最新網頁活動。
