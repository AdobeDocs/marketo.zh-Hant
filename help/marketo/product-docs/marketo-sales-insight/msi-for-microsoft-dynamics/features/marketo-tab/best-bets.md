# 最佳下注 {#best-bets}

「最佳下注」索引標籤包含根據優先順序計算的所有熱門潛在客戶清單，這些優先順序是使用緊急程度和相對分數計算的。

按一下「動作」欄下的點功能表，即可使用下列參與選項：
* 傳送Marketo電子郵件
* 新增至Marketo Campaign

您也可以從「最佳下注」索引標籤中選取多個銷售機會，然後選擇 _傳送Marketo電子郵件_ 或 _新增至Marketo Campaign_.
