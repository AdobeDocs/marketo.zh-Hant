# 最佳比對 {#best-bets}

「最佳選擇」索引標籤會根據優先順序包含您所有熱門潛在客戶的清單（使用緊急性和相對分數計算）。

按一下「動作」欄下方的點選單，您可以使用參與選項，例如：
* 傳送Marketo電子郵件
* 新增至Marketo Campaign

您也可以從「最佳選擇」標籤中選取多個銷售機會，然後選擇 _傳送Marketo電子郵件_ 或 _新增至Marketo Campaign_.
