# 建立[!UICONTROL Named Account] {#create-a-named-account}

請依照下列步驟，手動建立具名帳戶。

1. 在[!UICONTROL Named Accounts]中，按一下&#x200B;**[!UICONTROL New]**&#x200B;下拉式清單並選取&#x200B;**[!UICONTROL Create Named Account]**。

   ![](assets/two-1.png)

1. 填寫所需欄位，然後按一下&#x200B;**[!UICONTROL Create]**。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接按一下具名帳戶以檢視其控制面板。
