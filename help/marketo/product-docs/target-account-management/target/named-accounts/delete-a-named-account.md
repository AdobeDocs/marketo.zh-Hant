# 刪除具名帳戶 {#delete-a-named-account}

請依照下列快速步驟刪除已命名的帳戶。

1. 選取要刪除之具名帳戶的列。

   ![](assets/seven-1.png)

   >[!NOTE]
   >
   >按住Ctrl鍵並按一下(Windows)或按住Cmd鍵並按一下(Mac)以選取多個具名帳戶。

1. 按一下 **具名帳戶動作** 下拉式清單並選取 **刪除具名帳戶**.

   ![](assets/eight-1.png)

1. 按一下 **刪除**.

   ![](assets/nine-1.png)

   >[!NOTE]
   >
   >在TAM中無法刪除已同步至您CRM的帳戶。 如果刪除選項無法使用，或您收到「由於選取一個或多個CRM帳戶而無法刪除這些帳戶」訊息，則必須直接在CRM中刪除這些帳戶。
