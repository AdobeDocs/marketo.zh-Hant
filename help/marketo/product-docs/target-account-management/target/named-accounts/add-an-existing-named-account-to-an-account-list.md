# 將現有的具名帳戶新增至帳戶清單 {#add-an-existing-named-account-to-an-account-list}

將具名帳戶新增至帳戶清單相當簡單。

>[!NOTE]
>
>這僅適用於帳戶清單， **非** 動態帳戶清單。

1. 選取要新增至的具名帳戶列。

   ![](assets/four-1.png)

1. 按一下 **具名帳戶動作** 下拉式清單並選取 **新增至帳戶清單**.

   ![](assets/five-1.png)

1. 按一下 **帳戶清單** 從下拉式清單中選取所需的帳戶清單，然後按一下 **新增**.

   ![](assets/six-1.png)

   完成了！
