# 將現有的指定帳戶新增至帳戶清單 {#add-an-existing-named-account-to-an-account-list}

將指定帳戶新增至帳戶清單很簡單。

>[!NOTE]
>
>這僅適用於帳戶清單， **not** 動態帳戶清單。

1. 選擇要添加到的已命名帳戶的行。

   ![](assets/four-1.png)

1. 按一下 **已命名的帳戶操作** 下拉式清單並選取 **添加到帳戶清單**.

   ![](assets/five-1.png)

1. 按一下 **帳戶清單** 下拉式清單，選取所需的帳戶清單，然後按一下 **新增**.

   ![](assets/six-1.png)

   就這樣！
