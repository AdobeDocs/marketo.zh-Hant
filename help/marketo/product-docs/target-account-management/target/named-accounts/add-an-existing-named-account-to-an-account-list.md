# 將現有命名帳戶添加到帳戶清單{#add-an-existing-named-account-to-an-account-list}

將指名帳戶新增至帳戶清單很簡單。

>[!NOTE]
>
>這僅適用於帳戶清單，**not**&#x200B;動態帳戶清單。

1. 選擇要添加到的已命名帳戶的行。

   ![](assets/four-1.png)

1. 按一下&#x200B;**命名帳戶操作**&#x200B;下拉式清單並選擇&#x200B;**添加到帳戶清單**。

   ![](assets/five-1.png)

1. 按一下「**帳戶清單**」下拉式清單，選取所要的帳戶清單，然後按一下「新增&#x200B;**a3/>」。**

   ![](assets/six-1.png)

   就這樣！
