# 新增現有的[!UICONTROL Named Account]至帳戶清單 {#add-an-existing-named-account-to-an-account-list}

將具名帳戶新增至帳戶清單相當簡單。

>[!NOTE]
>
>這僅適用於帳戶清單，**不**&#x200B;動態帳戶清單。

1. 選取要新增至的具名帳戶列。

   ![](assets/four-1.png)

1. 按一下&#x200B;**[!UICONTROL Named Account Actions]**&#x200B;下拉式清單，然後選取&#x200B;**[!UICONTROL Add to Account List]**。

   ![](assets/five-1.png)

1. 按一下&#x200B;**[!UICONTROL Account List]**&#x200B;下拉式清單，選取所需的帳戶清單，然後按一下&#x200B;**[!UICONTROL Add]**。

   ![](assets/six-1.png)

   完成了！
