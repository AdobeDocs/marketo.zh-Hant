# 具名帳戶總覽 {#named-account-overview}

具名帳戶內含您鎖定目標的公司人員。 儀表板會提供您所有具名帳戶中每個屬性的目前資料。

## 具名帳戶儀表板 {#named-accounts-dashboard}

![](assets/one.png)

>[!TIP]
>
>已命名的帳戶預設會依「建立日期」排序，但您可以依任何標題中有排序圖示的欄排序。

>[!NOTE]
>
>Marketo會將同步至Marketo的所有CRM使用者顯示為「帳戶所有者」或「帳戶團隊成員」的篩選值。
