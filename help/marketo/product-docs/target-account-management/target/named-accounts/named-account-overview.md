# 命名帳戶概述 {#named-account-overview}

已命名帳戶會保留您所定位公司的人員。 控制面板會提供所有指定帳戶之每個屬性的目前資料。

## 指定帳戶控制面板 {#named-accounts-dashboard}

![](assets/one.png)

>[!TIP]
>
>根據預設，命名帳戶會依「建立日期」排序，不過，您可以依標題中有排序圖示的任何欄來排序。

>[!NOTE]
>
>Marketo會以「帳戶擁有者」或「帳戶團隊成員」的篩選值形式，顯示同步至Marketo的所有CRM使用者。
