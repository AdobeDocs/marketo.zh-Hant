# 帳戶清單深入分析 {#account-list-insights}

帳戶清單儀表板提供該清單中所有已命名帳戶的彙總見解檢視。

>[!NOTE]
>
>Marketo TAM會根據管道產生的或帳戶參與分數，自動決定帳戶清單中排名最前的指定帳戶。

## 帳戶清單儀表板 {#account-list-dashboard}

若要檢視帳戶清單的控制面板，只需按一下其名稱……

![](assets/one-new.png)

...且出現控制面板。

![](assets/two-new-1.png)

 <tbody> 
  <tr>

   <td colspan="1">檢視一段時間內的管道。 若要逐周判斷管道在一段時間內的變化，我們會在最後一天取得管道。</td> 
  </tr> 
  <tr> 
   <td><strong>收入</strong></td>

  </tr> 
 </tbody> 
</table>

## 已命名的帳戶頁簽 {#named-accounts-tab}

按一下 **具名帳戶** 標籤來檢視哪些具名帳戶屬於該帳戶清單。

![](assets/three-1.png)

>[!NOTE]
>
>您可以選取並點選「 」，在此索引標籤中移除已命名的帳戶 **移除具名帳戶**.

## 「使用者」索引標籤 {#used-by-tab}

按一下 **使用者** 索引標籤以檢視引用該帳戶清單的資產。

![](assets/four-2.png)
