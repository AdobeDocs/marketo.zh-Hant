# RCA中的具名帳戶Dimension {#named-account-dimension-in-rca}

使用Revenue Cycle Analytics中TAM特定的具名帳戶維度建立收入型報表。

>[!NOTE]
>
>**Dimension**  — 屬性（以黃色點表示），提供量值的不同檢視。

![](assets/one-2.png)

>[!NOTE]
>
>RCA中的具名帳戶維度可用來測量目標帳戶帶來的淨利影響（例如獲得的收入、產生的管道或銷售週期中的加速）。 此維度也可用來識別哪些程式對已命名的帳戶執行良好和沒有執行良好。

下列報表可存取「具名帳戶」維度：

* 電子郵件分析
* 銷售機會分析
* 機會分析
* 計畫會籍分析
