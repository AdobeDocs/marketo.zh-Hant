# 變更著陸頁面URL {#change-the-landing-page-url}

您可以修改著陸頁面的URL**。**這可協助URL更容易記住，並改善SEO。

1. 選取您的登陸頁面。 按一下「著陸頁面動作&#x200B;**」下拉式清單，捲動至「URL工具**」並選取「編輯URL設定」。********

   ![](assets/one.png)

1. 輸入&#x200B;**新URL**，選擇捨棄或重新導向舊URL，然後按一下「儲存」。****

   ![](assets/two.png)

   >[!NOTE]
   >
   >如果您決定要同時保留兩個URL，則會自動建立重新導向規則。 進一步瞭解[URL重新導向](http://docs.marketo.com/display/public/DOCS/Redirect+a+URL+Path)。
