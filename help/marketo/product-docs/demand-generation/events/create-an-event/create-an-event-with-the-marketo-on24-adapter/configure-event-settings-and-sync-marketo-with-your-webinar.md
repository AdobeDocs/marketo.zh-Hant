# 設定事件設定並將Marketo與您的網路研討會同步 {#configure-event-settings-and-sync-marketo-with-your-webinar}

請依照下列步驟設定Marketo事件設定，並連線Marketo和ON24。

## 設定事件 {#set-the-event}

1. 選擇您要與ON24網路研討會關聯的事件，然後按一下&#x200B;**[!UICONTROL Event Actions]**&#x200B;下拉式清單，並選取&#x200B;**[!UICONTROL Event Settings]**。

   ![](assets/one.png)

1. 選取ON24作為[!UICONTROL Event Partner]。

   ![](assets/two.png)

1. 選取[!UICONTROL Login]帳戶（例如顯示名稱）。

   ![](assets/three.png)

1. 輸入[!UICONTROL Event Id] （從ON24取得）。 按一下「**[!UICONTROL Save]**」。

   ![](assets/four.png)

   >[!NOTE]
   >
   >在高峰期，ON24可能需要15到20分鐘的時間才能將事件資訊提供給Marketo。 如果您收到「無效工作階段ID」訊息，請稍後再試。

## 設定排程 {#set-the-schedule}

當您設定與ON24網路研討會關聯的事件時，事件排程會填入來自ON24的資料。 若要存取[!UICONTROL Event Schedule]對話方塊，請遵循下列步驟。

1. 選取事件。 按一下&#x200B;**[!UICONTROL Event Actions]**&#x200B;下拉式清單，然後選取&#x200B;**[!UICONTROL Schedule].**

   ![](assets/five.png)

1. 選擇您的&#x200B;**[!UICONTROL Start Date]**、**[!UICONTROL End Date]**&#x200B;和&#x200B;**[!UICONTROL Time Zone]**。 按一下「**[!UICONTROL Save]**」。

   ![](assets/six-1.png)

   >[!NOTE]
   >
   >如果您在ON24中更新任何事件資訊，則必須按一下&#x200B;**[!UICONTROL Refresh from Webinar Provider]**&#x200B;功能表中的[!UICONTROL Event Actions]以檢視新資料填入。

現在您可以繼續進行下一個步驟：[建立子行銷活動和本機資產](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-child-campaigns-and-local-assets.md){target="_blank"}。
