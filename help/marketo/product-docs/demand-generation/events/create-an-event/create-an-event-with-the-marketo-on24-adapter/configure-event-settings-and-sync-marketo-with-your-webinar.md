# 設定事件設定並將Marketo與您的網路研討會同步 {#configure-event-settings-and-sync-marketo-with-your-webinar}

>[!IMPORTANT]
>
>自2022年8月起，ON24不再支援新的Marketo整合。 本文中的資訊僅適用於現有用戶。

請依照下列步驟來設定Marketo事件設定，並連線Marketo和ON24。

## 設定事件 {#set-the-event}

1. 選擇您要與ON24網路研討會建立關聯的事件，然後按一下 **事件動作** 下拉式清單並選取 **事件設定**.

   ![](assets/one.png)

1. 選擇ON24作為事件合作夥伴。

   ![](assets/two.png)

1. 選取登入帳戶（例如顯示名稱）。

   ![](assets/three.png)

1. 輸入事件ID（從ON24獲取）。 按一下 **儲存**.

   ![](assets/four.png)

   >[!NOTE]
   >
   >在高峰時段，ON24可能需要15到20分鐘，才能將事件資訊提供給Marketo。 如果您收到「Invalid Session Id」（會話ID無效）消息，請稍後再試。

## 設定排程 {#set-the-schedule}

當您設定與ON24網路研討會相關聯的事件時，事件排程會填入來自ON24的資料。 要訪問「事件調度」對話框，請執行以下步驟。

1. 選取事件。 按一下 **事件動作** 下拉式清單並選取 **排程。**

   ![](assets/five.png)

1. 選擇您的 **開始日期、結束日期**，和 **時區**. 按一下 **儲存**.

   ![](assets/six-1.png)

   >[!NOTE]
   >
   >如果更新ON24中的任何事件資訊，必須按一下 **從網路研討會提供者重新整理** 在「事件動作」功能表中，查看新資料填入。

現在您可以繼續下一步： [建立子行銷活動和本機資產](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-child-campaigns-and-local-assets.md).
