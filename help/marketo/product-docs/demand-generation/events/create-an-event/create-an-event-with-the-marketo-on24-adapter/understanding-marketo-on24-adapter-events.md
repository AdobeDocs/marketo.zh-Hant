# 了解 Marketo On24 Adapter 事件 {#understanding-marketo-on-adapter-events}

如果您的ON24網路研討會未連線至Marketo，您必須取得已在Marketo中的與會者資訊，然後將其輸入ON24。 網路研討會之後，您必須取得ON24中已存在的出席資訊，然後將其輸入回Marketo。

ON24介面卡會為您傳輸所有資訊。 它會將Marketo登陸頁面上擷取的註冊資訊推送到ON24，並將出席資訊自動提取到Marketo事件中。

這些文章將逐步引導您完成在ON24中建立網路研討會、在Marketo中建立事件以及將它們建立關聯的程式。
