# LaunchPoint活動合作夥伴{#launchpoint-event-partners}

將Market與LaunchPoint活動合作夥伴連結，以自動化您的線上活動。 當您連接它們時，您可以：

* 管理您的網路研討會活動
* 在Marketo登陸頁面上擷取的推播註冊資訊會傳送至已設定的LaunchPoint活動合作夥伴
* 自動將出席資訊納入Marketo活動
* 捕獲人員作為方案成員的狀態
