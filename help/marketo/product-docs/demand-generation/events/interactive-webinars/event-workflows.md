# 事件工作流程 {#event-workflows}

網路研討會完成後，請前往「控制面板」標籤檢視彙總效能。

![](assets/event-workflows-1.png)

>[!NOTE]
>
>如果網路研討會尚未完成或由於任何原因尚未收到資料，儀表板索引標籤將無法使用。

## 儀表板Widget {#dashboard-widgets}

「儀表板」由四個小工具組成。

**事件計畫狀態**：顯示該事件的各種方案狀態，以及目前持有該方案狀態值的方案成員數量。

**出席**：以百分比的形式，著重說明已參加網路研討會的不同成員的出席持續時間，並與整個網路研討會持續時間比較。

**輪詢效能** Adobe Connect ：顯示在網路研討會期間啟動的各種輪詢，並顯示在各個答案中參加並回應輪詢的成員分佈。

**點按和檔案下載活動**：提供已參加網路研討會並已採取特定行動的不同成員在Adobe Connect支援的網路研討會期間所點按的檔案和網路連結下載計數。

## 觸發器和篩選器 {#triggers-and-filters}

使用互動式網路研討會相關的觸發器和篩選器，找出事件程式中的特定人員群組。

![](assets/event-workflows-2.png)

每個觸發器/篩選器都附帶其他限制，可讓您擷取該群組的重點子集。
