# 建立互動式網路研討會 {#create-an-interactive-webinar}

簡介。

1. 前往 **行銷活動**.

   ![](assets/create-an-interactive-webinar-1.png)

1. 以滑鼠右鍵按一下所需的資料夾，然後選取 **新計畫**.

PICC

1. 為事件命名。 在方案類型下，選擇 **事件**.

PICC

1. 按一下「管道」下拉式清單，然後選取 **網路研討會**.

PICC

1. 選擇 **互動式網路研討會** 按一下 **建立**.

PICC

>[!NOTE]
>
>合作夥伴網路研討會的資訊可在此處找到。
