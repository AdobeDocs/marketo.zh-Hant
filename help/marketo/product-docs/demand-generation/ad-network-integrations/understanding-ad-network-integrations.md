# 了解廣告網路整合 {#understanding-ad-network-integrations}

![](assets/hith-golden-gate-144833144-e.jpeg)

## 廣告網路整合如何運作 {#how-ad-network-integrations-work}

廣告網路整合將Marketo的獨特受眾資料連結至您的付費媒體系統，讓您可以執行更具針對性的個人化數位廣告行銷活動。 這些整合目前可與Facebook、LinkedIn和Google搭配使用。

功能包括：

* 將受眾傳送至付費媒體系統，以便透過個人化顯示廣告鎖定已知潛在客戶，以培育或重新啟用這些潛在客戶

   * 依購買者階段、參與層級等劃分
   * 擴大受眾以改善贏取
   * 具有廣告的Target電子郵件訂閱者和非回應者
   * 在各個管道（例如電子郵件和數位廣告）中培養

* 從付費媒體系統擷取銷售機會或資料至Marketo

   * 例如，Facebook使用者按一下「訂閱」按鈕並提供名稱和電子郵件地址

* 將離線轉換資料上傳至付費媒體系統，最佳化銷售機會贏取並培養成果

   * 例如，某人搜尋、看見並點按廣告，然後線上填寫表單。 之後，他會離線收到示範，成為CRM機會。 廣告商就是在這裡找他的

* 重新鎖定網站訪客，使用 [網頁個人化](/help/marketo/product-docs/web-personalization/understanding-web-personalization/web-personalization-overview.md)，將其驅回您的網站

   * 依區段或垂直進行個人化
