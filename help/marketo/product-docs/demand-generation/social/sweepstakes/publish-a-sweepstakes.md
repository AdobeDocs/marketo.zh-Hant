# 發佈抽獎 {#publish-a-sweepstakes}

在您之後 [建立抽獎活動](/help/marketo/product-docs/demand-generation/social/sweepstakes/create-sweepstakes.md)，您可以將它發佈在Marketo登陸頁面、網站和Facebook上。

>[!AVAILABILITY]
>
>並非所有Marketo Engage使用者都已購買此功能。 如需詳細資訊，請聯絡Adobe客戶團隊（您的客戶經理）。

>[!NOTE]
>
>在發佈優惠方案之前，請確認其符合所有適用法律、規則和法規。

1. 找到您的登入頁面，然後按一下 **編輯草稿**.

   ![](assets/image2014-9-25-17-3a41-3a27.png)

1. 找出頁面右側的插入元素區段。 拖移到 **抽獎**.

   ![](assets/image2014-9-25-17-3a41-3a31.png)

1. 選取您的抽獎活動，然後按一下 **插入**.

   ![](assets/image2014-9-25-17-3a41-3a35.png)

   >[!NOTE]
   >
   >您需要核准您的抽獎活動，才能在這裡使用。

   恭喜！核准登陸頁面，您的抽獎活動即刻開始。 您也可以 [將登入頁面發佈至Facebook](/help/marketo/product-docs/demand-generation/facebook/publish-landing-pages-to-facebook.md) 或 [將抽獎活動放在您的網站上](/help/marketo/product-docs/demand-generation/social/social-functions/deploy-social-on-your-website.md).

   >[!TIP]
   >
   >測試並調整您的抽獎方式。 前往網頁上的抽獎活動，並註冊測試資訊。 記下您想要變更的所有使用者體驗相關內容。

   >[!NOTE]
   >
   >當使用者開始註冊後，請記得返回應用程式至 [選取抽獎的獲勝者](/help/marketo/product-docs/demand-generation/social/sweepstakes/select-sweepstakes-winners.md)！
