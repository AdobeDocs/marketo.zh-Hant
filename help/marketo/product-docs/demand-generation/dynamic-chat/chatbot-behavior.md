# 查特機器人行為 {#chatbot-behavior}

在訪客每次的預期行為之外，有下列不同的可能情況。

<table>
  <tbody>
    <tr>
      <th>縮寫</th>
      <th>詳細資料</th>
    </tr>
    <tr>
      <td>D1、D2、D3等</td>
      <td>表示多個對話框，其中D1為對話框</td>
    </tr>
    <tr>
      <td>P1、P2、P3等</td>
      <td>代表P1為最高優先順序的對話優先順序</td>
    </tr>
    <tr>
      <td>WP1、WP2、WP3等</td>
      <td>表示多個網頁，其中WP1是第一個網頁</td>
    </tr>
    <tr>
      <td>V1、V2、V3等</td>
      <td>代表多個V1為訪客的網頁訪客</td>
    </tr>
   </tbody>
</table>
