# 查特博特行為 {#chatbot-behavior}

以下是查特博在每個訪問者的預期行為之外的不同可能情形。

<table>
  <tbody>
    <tr>
      <th>縮寫</th>
      <th>詳細資料</th>
    </tr>
    <tr>
      <td>D1、D2、D3等</td>
      <td>表示多個對話框，其中D1是對話框1</td>
    </tr>
    <tr>
      <td>P1、P2、P3等</td>
      <td>表示P1為最高優先順序的對話優先順序</td>
    </tr>
    <tr>
      <td>WP1、WP2、WP3等</td>
      <td>表示WP1是第一個網頁的多個網頁</td>
    </tr>
    <tr>
      <td>V1 、 V2 、 V3等</td>
      <td>表示V1為訪問者的多個網頁訪問者</td>
    </tr>
   </tbody>
</table>
