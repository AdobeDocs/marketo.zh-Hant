# 使用量限制 {#usage-limits}

請造訪使用限制頁面以檢視重要資訊，例如套件資訊和使用限制狀態。

1. 在[設定]下，按一下[**[!UICONTROL Usage limits]**]。

   ![](assets/usage-limits-1.png)

1. 檢視您目前的套件、參與交談的每月限制狀態，以及行事曆/即時聊天席位的期限限制狀態。

   ![](assets/usage-limits-2.png)

>[!NOTE]
>
>每月上限會在每月1日自動重設。
