# 使用文檔卡 {#using-the-document-card}

「文檔」卡允許您在對話框中嵌入PDF文檔，並跟蹤訪客的文檔參與活動。 下面是如何設定它。

1. 導覽至 [Adobe PDF Embed API](https://udp.adobe.io/document-services/apis/pdf-embed/){target=&quot;_blank&quot;}。

1. 按一下 **獲取憑據**.

   ![](assets/using-the-document-card-1.png)

1. 登入您的Adobe帳戶。

   ![](assets/using-the-document-card-2.png)

1. 輸入您的憑據，接受條款，然後按一下 **建立憑證**.

   ![](assets/using-the-document-card-3.png)

   >[!IMPORTANT]
   >
   >您將需要使用將托管該聊天機器人的網域（例如，如果您要在mycompany.com上托管該聊天機器人，請務必在步驟4中輸入）。

1. 按一下 **複製** 來複製您的用戶端ID。

   ![](assets/using-the-document-card-4.png)

1. 返回動態聊天，按一下 **整合**. 在「 Adobe PDF Embed API」卡片中，按一下 **啟動**.

   ![](assets/using-the-document-card-5.png)

1. 貼上用戶端ID，然後按一下 **儲存**.

   ![](assets/using-the-document-card-6.png)

您現在可以在對話方塊中使用檔案卡 [串流設計工具](/help/marketo/product-docs/demand-generation/dynamic-chat/dialogues/stream-designer.md)!
