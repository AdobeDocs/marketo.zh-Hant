# 會議 {#meetings}

您可以在此處檢視網站訪客透過各種對話方塊排程的所有約會。 您可以在這裡找到預約預訂者的電子郵件地址、預約預訂的代理商、預約排定發生的時間，以及排定的會議時間是否已過。

![](assets/meetings-1.png)

>[!NOTE]
>
>在代理程式的行事曆上預約會議時，代理程式會收到關於預約的電子郵件通知，包括訪客Dynamic Chat參與的詳細資訊。
