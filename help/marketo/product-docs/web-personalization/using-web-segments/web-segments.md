# 網頁區段 {#web-segments}

## 檢視區段 {#view-segment}

![](assets/image2014-11-11-20-3a24-3a5.png)

「區段」標籤會顯示您根據各種屬性所設定的所有自訂已定義區段。  **區段是符合「設定區段」頁面中定義之指定條件的訪客集合。**  區段可以是來自特定產業、位置的訪客，或根據訪客的現場活動而定。

在Web個人化中，訪客可比對多個區段。 例如，如果有美國訪客的區段和金融公司的區段，則美國銀行的Web訪客會相符 **兩者** 美國訪客的區段和金融公司的區段。

**圖表：**  「區段」頁面會根據區段的訪客數（y軸）和區段名稱（x軸），顯示選定區段的長條圖。

**區段 — 右側面板**

![](assets/image2014-11-12-10-3a46-3a32.png)

在表格中選取區段，會在右側面板中顯示該區段的其他詳細資訊。

這些詳細資料包括：

* 區段名稱
* 區段的建立日期
* 顯示使用該區段之促銷活動的相關促銷活動。 按一下反應數量，您就會前往行銷活動頁面，顯示區段的行銷活動CTA （行動號召）
* 區段的符合數量（符合區段條件的訪客數量）以及符合區段的相異（不重複）訪客數量。 按一下「不重複訪客」連結，您會前往顯示區段結果的訪客頁面
* 區段的所有者/使用者建立者
* 與區段相關聯的網域網站
* 區段所選條件的簡短摘要

## 啟用或停用區段 {#enable-or-disable-a-segment}

![](assets/image2014-11-12-10-3a48-3a9.png)

若要啟用或停用區段，請選取表格中該區段的核取方塊，然後在表格底部的「選擇動作」下拉式方塊中，選取「啟用」或「停用」動作。 區段停用時，「狀態」欄下會顯示「停用」一詞。

## 建立區段 {#create-segments}

您建立的區段符合您在 **設定區段** 頁面。 您也可以根據條件組合來自訂區段，在您的行銷活動中定位特定對象。

建立新區段的方式

從 **區段** 頁面，按一下 **新建** 在圖表底下。 下列畫面隨即顯示。

![](assets/four.png)

定義區段的一般引數：

* **名稱：**  為您的區段命名。
* **說明：**  提供更詳細的區段條件說明。
* **網域：**  選取您要納入區段中的網域。
* **區段規則邏輯：**  選取AND / OR邏輯以建置每個細分屬性
* **時間：** 定義您想在行銷活動中使用的訪客參與層級

   * **進入時**：當訪客造訪網站時吸引訪客
   * **第1次到第9次點按後**：在網站上進行特定次數的點按後，與訪客互動

>[!TIP]
>
>**區段規則邏輯**
>
>篩選器選項共有三種：
>
>1. 使用所有篩選器（1和2及3...）
>1. 使用任何篩選器（1或2或3...）
>1. 進階篩選（使用和/或運算式）
>
>    進階篩選器可讓您控制區段條件。 輸入以「and」和「or」分隔的篩選編號。
>
>    * 1和2及3
>    * 1或2或3
>
>    混合「and」和「or」需要括弧來釐清邏輯意圖。 例如，「1或2和3」必須寫成下列其中一項：
>
>    * 1和（2或3）
>    * （1和2）或3
>
>    較複雜的邏輯可接受巢狀括弧，例如：
>
>    * （1和2）或（3和4）
>    * 1和(2或（3和4）)
>
>    在插入、刪除或重新排序之後，檢查您的邏輯。

### 第一圖形 {#firmographics}

**位置**

拖放 **位置** 放入區段編輯器中。

* 從下列引數中選取：

   * **包含**  — 選取您要讓行銷活動包含或排除位置。
   * **選取要新增的國家**  — 從下拉式方塊中，選取要納入區段中的國家/地區。 國家/地區名稱會顯示在右側。 您可以選擇多個國家/地區。

新增國家/地區後，您就可以指定區段的州/省、城市和郵遞區號。

* **選取要新增的州或省**  — 從下拉式方塊中，選取要納入的美國州或加拿大省份。 您可以進行多個選取。
* **郵遞區號**  — 輸入您要納入區段的郵遞區號。
* **城市**  — 輸入您要納入的城市。 在城市之間使用分號。

>[!TIP]
>
>**我該選擇哪些區段條件？ &#39;AND&#39;或&#39;OR&#39;？** OR會在每個欄位中當作額外選項運作。 潛在客戶只需滿足每個欄位中所選多個條件中的一個條件，即可符合區段的資格。 (例如，潛在客戶可能來自美國。 *或* 國防工業的)。 AND會當作此區段必須滿足的附加必要引數。 （例如，潛在客戶必須同時來自美國和國防工業）。 在每個區段設定檔中，每個個別欄位都可作為兩者使用，其功能可為「AND」或「OR」（視選取的區段條件而定）。

**產業** 在 **設定檔分段** 區段，勾選旁的方塊 **產業**.

* 從下列引數中選取：

   * **包含**  — 選取您要讓區段包含或排除某個產業。
   * **選取要新增的產業**  — 選取您要納入區段的產業。 產業會顯示在下拉式方塊下方。 您可以選擇多個產業。

**組織群組**

在 **設定檔分段** 區段，勾選旁的方塊 **組織群組。**

* 從下拉式方塊中，選取下列選項：

   * 《財富》500強 — 在此區段僅包括《財富》500強公司
   * Fortune 1000 — 僅包括Fortune 1000在此區段中的公司
   * Global 2000 — 在此區段中包含Global 2000公司
   * 企業 — 包括員工超過1,000人且收入超過2.5億美元的組織
   * 中小型企業 — 僅包括此區段的中小企業

* **來自這些公司（特定名稱）**

   * 從「選取要新增的公司」下拉式清單中選取要定位的公司。
   * 您可以輸入想要鎖定的確切組織名稱。 *它是 _一律_ 建議使用具名帳戶清單，而非手動輸入名稱以符合條件（請參閱下文）。

### 已知人員 {#known-people}

**資料庫**

Web個人化會與您的Marketo資料庫整合，可讓您依據已知的個人屬性和資料來細分和個人化行銷活動。

選取資料庫，然後從下拉式清單中選取人員資料欄位。 選取 **+** 以從下拉式清單新增欄位。

![](assets/seven.png)

您可以從「帳戶設定>資料庫」新增或移除人員資料欄位

>[!TIP]
>
>根據Marketo人員的所有人員資料欄位（例如職稱、分數、角色等）建立您的區段條件。
>
>例如： 「職稱等於CMO」和「分數小於或等於50」

**Marketo電子郵件行銷活動** 透過訪客點選Marketo電子郵件並到達網站時的電子郵件反向連結，對行銷活動進行細分和個人化。 依Marketo方案名稱或行銷活動名稱劃分割槽段，並繼續從電子郵件到網頁的對話。 從下拉式清單中選取+以新增欄位。

![](assets/image2015-5-27-17-3a20-3a34.png)

**狀態**

根據潛在客戶狀態定義您的區段：已知或匿名。

* 已知 — 從已知訪客的下拉式方塊中選取此選項。 訪客在您的網站上提交表單並出現在「網頁個人化人員」頁面時，即可得知訪客。
* 匿名 — 從匿名訪客的下拉式方塊中選取此選項。

![](assets/image2015-5-27-17-3a23-3a2.png)

### 行為 {#behavioral}

**造訪 —** 根據訪客行為或身分識別來定義您的區段。

* 造訪次數 — 從下拉式方塊中選取此選項，可指定潛在客戶在網站上的造訪次數。

   * 從下拉式方塊中選取等於、等於或大於或等於或小於。

* 特定造訪 — 從下拉式方塊中選取此選項，以指定特定訪客。

   * 在右側的文字方塊中，輸入您要追蹤的訪客編號。 按一下訪客（在訪客頁面中）和右側面板的「設定行銷活動」時，可找到唯一的Web Personalization訪客識別碼。 訪客ID位於進階設定區段中。 訪客ID也可在URL中找到（例如VISITOR=JZZJIFJNUI60PZ8Y97BHTY9BL8PKWS）。

**搜尋字詞**  — 根據潛在客戶的搜尋辭彙定義區段。

* 搜尋的訪客 — 從下拉式清單中，從訪客搜尋中選取您要追蹤的字詞，或新增您自己的搜尋字詞。 (不需要 &#42; 搜尋字詞上的萬用字元（設為預設值，以包含搜尋字詞的片語）。

**轉介**  — 新增訪客參考的URL。

* 選取要新增的轉介 — 從下拉式清單中，選取您要追蹤的轉介網站或新增您自己的轉介。 選取後，反向連結將顯示在下面的方塊中。 (使用 &#42; （允許為萬用字元）

**包含頁面**  — 追蹤潛在客戶在您網站上瀏覽的特定頁面。

* URL相符 — 新增您想要追蹤的特定網頁的URL。 您可以新增多個URL，方法是以分號分隔。 (使用 &#42; （允許為萬用字元）。

**排除頁面**  — 排除您不想要在區段中相符的特定頁面。 (使用 &#42; （允許為萬用字元）。

* URL不符合 — 新增您想從追蹤中排除的特定網頁URL。 您可用分號區隔多個URL，以新增多個URL

## 編輯區段 {#edit-segments}

您可以編輯已建立的區段。

1. 若要編輯區段，請前往 **區段**.

   ![](assets/image2014-11-12-11-3a38-3a22.png)

1. 在 **區段** 表格中，按一下編輯圖示( ![](assets/segment-edit.png))。 此 **設定區段** 頁面隨即開啟，並顯示選取的區段。
1. 套用任何您想要對區段進行的編輯或變更。
1. 按一下 **儲存**.

## 刪除區段 {#delete-segments}

您可以刪除已建立的區段。

1. 從 **區段** 頁面上，選取區段。
1. 按一下刪除圖示( ![](assets/segment-delete.png) )。
1. 確認訊息隨即出現，確認您即將刪除 **區段**.

>[!NOTE]
>
>您無法刪除與行銷活動相關聯的區段。 首先，您需要刪除行銷活動，然後刪除區段。
