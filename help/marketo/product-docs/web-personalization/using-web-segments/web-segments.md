# 網頁區段 {#web-segments}

## 檢視區段 {#view-segment}

![](assets/image2014-11-11-20-3a24-3a5.png)

「區段」索引標籤會顯示您根據各種屬性所設定的所有自訂定義區段。  **區段是符合「設定區段」頁面中定義之指定條件的訪客集合。**  區段可以是特定產業、位置或根據訪客的站上活動的訪客。

在網站個人化中，訪客可以符合多個區段。 例如，如果有美國訪客的區段和金融公司的區段，則來自美國銀行的網站訪客會符合 **both** 美國訪客的區段和金融公司的區段。

**圖表：**  「區段」頁面會根據區段的訪客數（y軸）和區段名稱（x軸），顯示所選區段的長條圖圖。

**區段 — 右側面板**

![](assets/image2014-11-12-10-3a46-3a32.png)

在表格中選取區段會顯示右側面板中該區段的其他詳細資訊。

這些詳細資訊包括：

* 區段名稱
* 區段的建立日期
* 顯示與區段一起運作之促銷活動的相關促銷活動。 按一下反應次數後，您會前往顯示區段之促銷活動CTA（動作呼叫）的促銷活動頁面
* 區段的符合次數（符合區段條件的訪客量），以及符合區段的不重複（不重複）訪客數量。 按一下獨特訪客連結時，您會前往顯示區段結果的訪客頁面
* 區段的擁有者/使用者建立者
* 與區段相關聯的網域網站
* 區段所選條件的簡短摘要

## 啟用或停用區段 {#enable-or-disable-a-segment}

![](assets/image2014-11-12-10-3a48-3a9.png)

若要啟用或停用區段，請在表格中選取該區段的核取方塊，並在表格底部的「選擇動作」下拉式方塊中，選取「啟用」或「停用」動作。 停用區段時，「狀態」欄下方會顯示「停用」一詞。

## 建立區段 {#create-segments}

您建立的區段符合您在 **設定區段** 頁面。 您也可以根據條件的組合來自訂區段，以定位促銷活動中的特定對象。

若要建立新區段

從 **區段** 頁面，按一下 **新建** 下。 畫面隨即顯示。

![](assets/four.png)

定義區段的一般參數：

* **名稱：**  為區段命名。
* **說明：**  提供區段條件的更詳細說明。
* **網域：**  選取您要納入區段的網域。
* **區段規則邏輯：**  選取AND/OR邏輯以建立每個分段屬性
* **時間：** 定義您要在行銷活動中的訪客參與程度

   * **進入時**:訪客的參與到達網站
   * **第1次至第9次點按後**:在網站上點按特定次數後與訪客互動

>[!TIP]
>
>**區段規則邏輯**
>
>有三個篩選選項：
>
>1. 使用所有篩選器（1和2以及3...）
>1. 使用任何篩選器（1或2或3...）
>1. 進階篩選器（使用和/或運算式）

   >
   >    進階篩選器可讓您控制區段條件。 輸入以「和」和「或」分隔的篩選數字。
   >
   >    * 1和2及3
   >    * 1或2或3

   >
   >    混合&quot;and&quot;和&quot;or&quot;需要括弧來釐清邏輯意圖。 例如「1」或「2」和「3」必須寫成下列其中一項：
   >
   >    * 1和（2或3）
   >    * （1和2）或3

   >
   >    對於較複雜的邏輯，例如，
   >
   >    * （1和2）或（3和4）
   >    * 1和(2或（3和4）)

   >
   >    在插入、刪除或重新排序任何項目後，請檢查您的邏輯。

### Firmographics {#firmographics}

**位置**

拖放 **位置** 填入區段編輯器中。

* 從下列參數中選取：

   * **包括**  — 選取您要讓促銷活動包含或排除位置。
   * **選擇要添加的國家**  — 從下拉式方塊中，選取您要納入區段的國家/地區。 國家/地區名稱會顯示在右側。 您可以選擇多個國家/地區。

新增國家後，您也可以指定區段的州、市和郵遞區號。

* **選擇要添加的州或省**  — 從下拉框中，選擇要包括的美國州或加拿大省。 您可以進行多個選取。
* **郵遞區號**  — 輸入您要納入區段的郵遞區號。
* **城市**  — 輸入要包括的城市。 在城市之間使用分號。

>[!TIP]
>
>**我選擇哪些區段條件？ 「AND」或「OR」？** OR在每個欄位內當作其他選項運作。 潛在客戶只需滿足每個欄位內所選多個條件中的一個條件，即可符合區段資格。 (例如，潛在客戶可以來自美國。 *或* 來自國防工業)。 AND函式作為此區段必須符合的其他強制參數。 （例如，前景必須來自美國和國防工業）。 在每個分段設定檔中，每個個別欄位可以同時作為「AND」或「OR」，視選取的「區段條件」而定。

**產業** 在 **設定檔區段** 區段，核取旁邊的方塊 **產業**.

* 從下列參數中選取：

   * **包含**  — 選取您要讓區段包含或排除產業。
   * **選擇要添加的行業**  — 選取您要納入區段的產業。 產業顯示在下拉式方塊下方。 您可以選擇多個產業。

**組織組**

在 **設定檔區段** 區段，核取旁邊的方塊 **組織組。**

* 從下拉式方塊中，選取下列選項：

   * 《財富》500強 — 此部分僅包括《財富》500強公司
   * 《財富》1000強 — 此部分僅包括《財富》1000強公司
   * 2000年全球 — 包括2000年全球公司
   * 企業 — 包括員工人數超過1,000、收入超過2.5億美元的組織
   * 中小型企業 — 此領域僅包括中小型企業

**— 指定帳戶 —**

**組織**

* **來自這些公司（特定名稱）**

   * 從「選擇要新增的公司」下拉式清單中選取要定位的公司。
   * 您可以輸入要定位的確切組織名稱。 *是 _always_ 建議您使用「指定帳戶清單」，而不要手動輸入名稱，以獲得更理想的符合項目（請參閱下文）。

**指定帳戶清單**

從 [指定帳戶清單](/help/marketo/product-docs/web-personalization/account-based-web-marketing/create-a-new-account-list.md) 劃分關鍵目標帳戶。

### 已知人員 {#known-people}

**資料庫**

Web個人化與您的Marketo資料庫整合，可讓您依已知人員屬性和資料來劃分及個人化行銷活動。

選擇「資料庫」，然後從下拉清單中選擇人員資料欄位。 選取 **+** 從下拉式清單新增欄位。

![](assets/seven.png)

您可以從「帳戶設定>資料庫」中新增或移除人員資料欄位

>[!TIP]
>
>根據Marketo人員（例如職銜）的所有人員資料欄位，建立您的區段條件；分數；角色；等等。
>
>例如 「職稱等於CMO」和「分數小於或等於50」

**Marketo電子郵件行銷活動** 按一下Marketo電子郵件並到達網站的訪客所寄送的電子郵件反向連結，以劃分及個人化促銷活動。 依Marketo方案名稱或促銷活動名稱分段，並繼續從電子郵件進行對話。 從下拉式清單中選取「+ 」以新增欄位。

![](assets/image2015-5-27-17-3a20-3a34.png)

**狀態**

根據潛在客戶的狀態定義您的區段：已知或匿名。

* 已知 — 從已知訪客的下拉式方塊中選取此選項。 訪客在您的網站上提交表單並出現在「網站個人化人員」頁面時即為已知。
* 匿名 — 從用於匿名訪客的下拉式方塊中選取此選項。

![](assets/image2015-5-27-17-3a23-3a2.png)

### 行為 {#behavioral}

**瀏覽 —** 根據訪客行為或識別定義區段。

* 瀏覽次數 — 從下拉式方塊中選取此選項，以指定網站上潛在客戶的瀏覽次數。

   * 從下拉式方塊中選取「等於」、「等於」或「大於」、「等於」或「小於」。

* 特定瀏覽 — 從下拉式方塊中選取此選項，以指定特定訪客。

   * 在右側的文字方塊中，輸入您要追蹤的訪客編號。 按一下訪客（在訪客頁面中）和右側面板的「設定促銷活動」時，可找到不重複的Web個人化訪客識別號。 訪客ID位於「進階設定」區段。 您也可以在URL中找到訪客ID(例如VISITOR=JZJIFJNUI60PZ8Y97BHTY9BL8PKWS)。

**搜尋詞**  — 根據潛在客戶的搜尋詞定義區段。

* 搜尋的訪客 — 從下拉式清單中，選取您要從訪客搜尋中追蹤的詞語，或新增您自己的搜尋詞。 (不需要 &#42; 萬用字元，因為此萬用字元預設為包含包含搜尋詞的片語)。

**轉介**  — 新增訪客所參考的URL。

* 選取要新增的轉介 — 從下拉式清單中，選取您要追蹤或新增您自己的轉介的轉介網站。 選取後，反向連結會顯示在下方的方塊中。 (使用 &#42; 為允許的通配符)

**包含頁面**  — 追蹤訪客在您網站上瀏覽的特定頁面。

* URL符合 — 新增您要追蹤之特定網頁的URL。 您可以使用分號來分隔多個URL，以新增這些URL。 (使用 &#42; 為萬用字元)。

**排除頁面**  — 排除您不想在區段中比對的特定頁面。 (使用 &#42; 為萬用字元)。

* URL不符 — 新增您要從追蹤中排除之特定網頁的URL。 您可以使用分號分隔多個URL，以新增這些URL

## 編輯區段 {#edit-segments}

您可以編輯已建立的區段。

1. 若要編輯區段，請前往 **區段**.

   ![](assets/image2014-11-12-11-3a38-3a22.png)

1. 在 **區段** 表格，按一下「編輯」圖示( ![](assets/segment-edit.png))的區段。 此 **設定區段** 頁面隨所選區段開啟。
1. 套用您要對區段進行的任何編輯或變更。
1. 按一下 **儲存**.

## 刪除區段 {#delete-segments}

您可以刪除已建立的區段。

1. 從 **區段** 頁面上方選取區段。
1. 按一下刪除圖示( ![](assets/segment-delete.png) )的區段。
1. 畫面會顯示確認訊息，確認您即將刪除 **區段**.

>[!NOTE]
>
>您無法刪除與促銷活動相關聯的群體。 首先，您需要刪除促銷活動，然後刪除群體。
