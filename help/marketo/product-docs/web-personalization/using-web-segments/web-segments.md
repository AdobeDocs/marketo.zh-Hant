# 網頁區段{#web-segments}

## 檢視區段{#view-segment}

![](assets/image2014-11-11-20-3a24-3a5.png)

「區段」標籤會顯示您根據各種屬性所設定的所有自訂定義區段。  **區段是符合「設定區段」頁面中所定義之指定條件的訪客集合。**  區段可以是來自特定產業、位置或根據訪客上門活動的訪客。

在「網頁個人化」中，訪客可以比對多個區段。 例如，如果有美國訪客的區段和金融公司的區段，來自美國銀行的Web訪客會比對&#x200B;**兩個**&#x200B;美國訪客的區段和金融公司的區段。

**圖表：**  「區段」頁面會根據區段的訪客數（y軸）和區段名稱（x軸），顯示所選區段的長條圖。

**區段——右側面板**

![](assets/image2014-11-12-10-3a46-3a32.png)

在表格中選取區段時，會在右側面板中顯示區段的其他詳細資訊。

這些詳細資訊包括：

* 區段名稱
* 區段的建立日期
* 顯示與區段一起運作之促銷活動的相關促銷活動。 按一下反應數，會帶您前往顯示區段之促銷活動CTA（行動要求）的促銷活動頁面
* 區段的符合次數（符合區段條件的訪客數量），以及符合區段的不同（獨特）訪客數量。 按一下獨特訪客連結會帶您前往顯示區段結果的訪客頁面
* 區段的擁有者／使用者建立者
* 與區段關聯的網域網站
* 區段選取標準的簡短摘要

## 啟用或停用區段{#enable-or-disable-a-segment}

![](assets/image2014-11-12-10-3a48-3a9.png)

若要啟用或停用區段，請選取表格中該區段的核取方塊，並在表格底部的「選擇動作」下拉式方塊中，選取「啟用」或「停用」動作。 當區段停用時，「狀態」欄下會顯示「停用」字詞。

## 建立區段{#create-segments}

您建立的區段符合您在&#x200B;**設定區段**&#x200B;頁面中定義的特定條件。 您也可以根據條件組合來自訂細分，鎖定促銷活動中的特定對象。

若要建立新區段

在&#x200B;**區段**&#x200B;頁面中，按一下圖表下的&#x200B;**建立新**。 出現以下螢幕。

![](assets/four.png)

定義區段的一般參數：

* **名稱：**  命名區段。
* **說明：**  提供區段標準的更詳細說明。
* **網域：**  選取要包含在區段中的網域。
* **區段規則邏輯：**  選取AND/OR邏輯以建立每個區段屬性
* **時間：定** 義您要在促銷活動中的訪客參與程度

   * **進入時**:從訪客進入網站時進行互動
   * **第1次到第9次按一下後**:在網站上點按特定數量後，吸引訪客

>[!TIP]
>
>**區段規則邏輯**
>
>有三個篩選選項：
>
>1. 使用所有篩選器（1和2及3...）
>1. 使用任何篩選器（1或2或3...）
>1. 進階篩選器（使用和／或運算式）

   >
   >    
   進階篩選可讓您控制區段條件。 輸入由&quot;and&quot;和&quot;or&quot;分隔的篩選編號。
   >
   >    
   * 1和2和3
   >    * 1或2或3

   >
   >    混合&quot;and&quot;和&quot;or&quot;需要括弧來釐清邏輯意圖。 例如，「1或2及3」必須以下列其中一種方式撰寫：
   >
   >    
   * 1和（2或3）
   >    * （1和2）或3

   >
   >    對於更複雜的邏輯，例如，
   >
   >    
   * （1和2）或（3和4）
   >    * 1和(2或（3和4）)

   >
   >    在插入、刪除或重新排序後，請檢查您的邏輯。

### Firmographics {#firmographics}

**位置**

將&#x200B;**Location**&#x200B;拖放至區段編輯器。

* 從以下參數中選擇：

   * **包含** -選擇您要讓促銷活動包含或排除位置。
   * **選取要新增的國家** -從下拉式方塊中，選取您要納入區段的國家。國家／地區名稱會出現在右側。 您可以選擇多個國家。

新增國家後，您也可以指定區段的州、市和郵遞區號。

* **選擇州或省以新增** -從下拉式方塊中，選取您要包含的美國州或加拿大省。您可以進行多項選擇。
* **郵遞區號** -輸入您要納入區段的郵遞區號。
* **城市** -輸入您要納入的城市。在城市之間使用分號。

>[!TIP]
>
>**我選擇哪些區段條件？「AND」或「OR」?** OR在每個欄位中當作其他選項運作。潛在客戶只需滿足每個欄位中選取之多個標準中的一個標準，即可符合區段資格。 (例如，潛在客戶可能來自美國。*或*)。 AND函式是此區段必須符合的額外必要參數。 （例如，前景必須來自美國和國防工業）。 在每個區段設定檔中，每個個別欄位都可以同時使用「AND」或「OR」，視選取的區段條件而定。

**Industries** 在「 **Profile** Segmentations」區段下，核取「 **Industry**」旁邊的方塊。

* 從以下參數中選擇：

   * **包含** -選擇您要讓區段包含或排除產業。
   * **選擇要添加的行業** -選擇要包括在該領域中的行業。產業會顯示在下拉式方塊下方。 您可以選擇多個產業。

**組織群組**

在&#x200B;**描述檔劃分**&#x200B;區段下，核取&#x200B;**組織群組旁的方塊。**

* 從下拉式方塊中，從下列選項中選擇：

   * 財富500強——此領域僅包括財富500強公司
   * 財星1000大企業——僅包括財星1000大企業
   * Global 2000 —— 包括這一領域的Global 2000公司
   * 企業——包括員工人數超過1,000人且收入超過2.5億美元的組織
   * 中小型企業——此領域僅包括中小型企業

### 已知人員{#known-people}

**資料庫**

Web Personalization與您的Marketo Database整合，讓您依據已知人員屬性和資料細分並個人化宣傳活動。

選擇資料庫，然後從下拉清單中選擇人員資料欄位。 選擇&#x200B;**+**&#x200B;以從下拉式清單中新增欄位。

![](assets/seven.png)

您可以從「帳戶設定>資料庫」新增或移除人員資料欄位

>[!TIP]
>
>根據Marketing中所有人員資料欄位（例如職銜）建立您的區段標準；得分；角色；等等……
>
>例如 「職稱等於CMO」和「分數小於或等於50」

**Marketo Email** CampaignSegment透過來自訪客點按行銷人員電子郵件並到達網站的電子郵件反向連結，將行銷活動個人化。依行銷人員方案名稱或行銷活動名稱細分，並繼續從電子郵件到網路的對話。 從下拉式清單中選取「+」以新增欄位。

![](assets/image2015-5-27-17-3a20-3a34.png)

**狀態**

根據潛在客戶的狀態定義您的細分：已知或匿名。

* 已知——從已知訪客的下拉式方塊中選取此選項。 訪客在您的網站上送出表單並出現在「網頁個人化人員」頁面時即為已知。
* 匿名——從匿名訪客的下拉式方塊中選取此選項。

![](assets/image2015-5-27-17-3a23-3a2.png)

### 行為{#behavioral}

**瀏覽——根** 據訪客行為或識別定義您的區段。

* 瀏覽次數——從下拉式方塊中選取此選項，以指定網站上潛在客戶的瀏覽次數。

   * 從下拉式方塊中選取「等於」、「等於」或「大於」，或「等於」或「小於」。

* 特定瀏覽——從下拉式方塊中選取此選項，以指定特定訪客。

   * 在右側的文字方塊中，輸入您要追蹤的訪客編號。 按一下訪客（在訪客頁面中）和右側面板上的「設定促銷活動」時，可找到唯一的「網頁個人化」訪客識別碼。 訪客ID位於「進階設定」區段。 訪客ID也可在URL中找到（例如VISITOR=JZZJIFJNUI60PZ8Y97BHTY9BL8PKWS）。

**搜尋詞** -根據潛在客戶的搜尋詞定義區段。

* 訪客搜尋——從下拉式清單中，選取您要從訪客搜尋中追蹤的詞語，或新增您自己的搜尋詞。 （搜尋詞上不需要*萬用字元，因為它設定為預設值，以包含包含搜尋詞的片語）。

**反向連結** -新增訪客被反向連結的URL。

* 選取要新增的轉介——從下拉式清單中，選取您要追蹤的轉介網站，或新增您自己的轉介。 選取後，轉介會顯示在下方的方塊中。 （允許使用*作為萬用字元）

**包含頁面** -追蹤在您網站上瀏覽的潛在客戶特定頁面。

* URL符合——新增您要追蹤之特定網頁的URL。 您可以使用分號分隔多個URL來新增這些URL。 （允許使用*作為萬用字元）。

**排除頁面** -排除您不想在區段中符合的特定頁面。（允許使用*作為萬用字元）。

* URL不符合——新增您要排除在追蹤之外之特定網頁的URL。 您可以新增多個URL，方法是以分號分隔URL

## 編輯區段{#edit-segments}

您可以編輯已建立的區段。

1. 若要編輯區段，請前往&#x200B;**區段**。

   ![](assets/image2014-11-12-11-3a38-3a22.png)

1. 在&#x200B;**區段**&#x200B;表格中，按一下您要編輯之區段的編輯圖示(![](assets/segment-edit.png))。 **設定區段**&#x200B;頁面隨即開啟，其中顯示選取的區段。
1. 套用您想要對區段進行的任何編輯或變更。
1. 按一下&#x200B;**保存**。

## 刪除區段{#delete-segments}

您可以刪除已建立的區段。

1. 從上述的&#x200B;**區段**&#x200B;頁面中，選取區段。
1. 按一下您要刪除之區段的刪除圖示(![](assets/segment-delete.png))。
1. 此時會出現確認訊息，確認您即將刪除&#x200B;**Segment**。

>[!NOTE]
您無法刪除與促銷活動關聯的區段。 首先，您需要先刪除促銷活動，再刪除群體。
