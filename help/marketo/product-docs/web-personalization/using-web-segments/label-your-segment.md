# 為區段加上標籤 {#label-your-segment}

您是否擁有如此多的區段，以至於捲動變得繁瑣起來？ 使用標籤來標籤區段，以便快速尋找。

## 標籤區段 {#tag-a-segment}

1. 登入Web Personalization並前往 **區段**.

   ![](assets/new-dropdown-segments-hand.jpg)

1. 選取您要使用標籤標籤的區段。

   ![](assets/image2015-10-14-15-3a26-3a28.png)

1. 若要使用現有標籤，請按一下 **設定標籤**，勾選方塊，然後按一下 **套用**.

   ![](assets/image2015-10-14-15-3a34-3a42.png)

1. 或者，若要建立新標籤，請按一下 **設定標籤**，輸入新標簽名稱，然後按一下 **新建**.

   ![](assets/image2015-10-14-15-3a38-3a30.png)

   >[!NOTE]
   >
   >「建立新標籤」按鈕會顯示新標籤的名稱。 如果標籤太長，可能不會顯示「新建」。

酷！ 您現在知道如何指派及建立區段的標籤。
