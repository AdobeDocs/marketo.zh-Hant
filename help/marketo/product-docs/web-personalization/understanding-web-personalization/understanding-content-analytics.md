# 了解Content Analytics {#understanding-content-analytics}

「內容分析」頁面會顯示您從網站探索到的現有內容（案例研究、部落格文章、影片、新聞稿等）。 它也會顯示內容的效能，以及訪客與其互動時產生的人員。

## 檢視內容分析 {#view-content-analytics}

前往 **內容分析**.

![](assets/one.png)

在「內容分析」頁面上，您可以：

* 依時間範圍（日、周和月）篩選
* 依內容標題和內容URL搜尋
* 按一下檢視、直接轉換和轉換率的欄標題，以降序或升序排序。

![](assets/content-analytics-8-29-16-blur.png)

您也可以按一下圖示，將檔案匯出為CSV格式。

![](assets/image2016-8-29-13-3a51-3a49.png)

## 檢視其他內容分析 {#view-additional-content-analytics}

按一下內容片段的Analytics圖示。

![](assets/four.png)

隨即開啟對話方塊，其中包含該特定內容片段的其他「內容分析」。

![](assets/five.png)

其他內容分析包括：

**Analytics**

* **檢視**:所選時間範圍內該內容片段的視圖
* **直接轉換**:在同次瀏覽中檢視內容並填寫表單的網站訪客。
* **轉換率****:**以「直接轉換」除以「點按」計算的百分比轉換率

**趨勢**

* A **趨勢** 圖表顯示特定內容片段最近30天的檢視。 將滑鼠指標暫留在折線圖上，即可查看特定日期的內容檢視次數

## 刪除內容 {#delete-content}

![](assets/six.png)

在「內容分析」頁面中，按一下您要刪除之內容的刪除圖示。 系統會顯示訊息，確認您要刪除「內容」。
