# 網頁個人化概述 {#web-personalization-overview}

## 簡介 {#introduction}

針對Marketo的新客戶，即時個人化分為四個應用程式，使用這兩個圖磚存取：Web個人化和ContentAI。

![](assets/pasted-image-at-2016-03-23-02-45-pm.png) ![](assets/mlm-homepage-content-ai-281-29.png)

>[!NOTE]
>
>Web個人化和ContentAI提供所有Marketo支援的語言（英文、法文、德文、日文、葡萄牙文、西班牙文）。

現有客戶會繼續透過「即時個人化」圖磚存取應用程式，直到續約為止，屆時他們將使用上述登入。

![](assets/image2016-2-9-8-3a52-3a32.png)

按一下「Web個人化」圖磚時，會顯示「控制面板」頁面。 按一下ContentAI圖磚，然後直接前往「內容Recommendations」頁面。

這四個應用是：

* 網頁個人化
* 帳戶型網路行銷
* 網站重新定位
* ContentAI

此 **網頁個人化** 圖磚是您建立個人化網頁行銷活動的存取點。 帳戶型網頁行銷和網頁重新定位。 您可以在「內容分析」頁面上新增內容。

**帳戶型網路行銷** 可搭配「網頁個人化」和「指定帳戶」功能使用。 「指定帳戶」頁提供有關現有客戶、總參與和組織趨勢的圖形資訊，以及帳戶清單。 您也可以在那裡建立新的指定帳戶。

**網站重新定位** 可讓您向細分的受眾進行再行銷。 您可以設定區段，但無法在篩選器中使用指定帳戶。

**ContentAI** 是提供內容建議的位置。 您可以輕鬆將內容新增至「內容Recommendations」頁面。

>[!NOTE]
>
>若要存取帳戶型網頁行銷或網站重新鎖定目標，您必須按一下「網頁個人化」圖磚，即使您沒有網頁個人化應用程式亦然。

## 組織 {#organizations}

「組織」索引標籤會顯示指定期間造訪您網站之組織的所有詳細資訊（名稱、位置、活動和時間戳記）。 表格可依時間、位置、網域，以及透過自由文字搜尋來排序和組織。

![](assets/image2014-11-10-19-3a23-3a18.png)

>[!TIP]
>
>「最活躍」與&#39;Latest&#39; — 表格依序由已識別的組織和Internet服務提供者組織（以ISP圖示表示）。 可依下列項目排序：
>
>* 最活躍：表格中最活躍的組織（根據頁面檢視次數）
>* 最新：表格中最近的組織（預設）
>* 如需寶貴的深入分析，請根據最活躍的

**共用組織**

想要與同事共用您網站訪客的資訊？ 按一下組織，然後按一下右側面板上的「共用」連結。

這會啟動「共用此組織」覆蓋圖，您可在此傳送電子郵件，分享造訪網站之組織的名稱和詳細資訊。

![](assets/image2014-11-10-19-3a25-3a42.png)
