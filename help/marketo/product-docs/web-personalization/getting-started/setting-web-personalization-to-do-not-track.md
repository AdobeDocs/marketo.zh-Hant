# 將[!UICONTROL Web Personalization]設定為Do Not Track {#setting-web-personalization-to-do-not-track}

網頁訪客可透過選擇「不要追蹤」(DNT)來設定瀏覽器，以防止任何網站進行追蹤。 如此可防止追蹤該特定瀏覽器和裝置。

在[!UICONTROL Web Personalization]和[!UICONTROL Predictive Content]中，行銷人員可以設定切換按鈕，指出是否支援或忽略瀏覽器的Do Not Track (DNT)設定。 帳戶的切換已設定為預設關閉，表示應用程式不會接受DNT。

## 啟用或停用切換 {#enable-or-disable-the-toggle}

1. 前往 **[!UICONTROL Account Settings]**。

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. 在[!UICONTROL Domain]和[!UICONTROL Domain Configuration]中，選取&#x200B;**[!UICONTROL On]**&#x200B;以啟用[!UICONTROL Honor DNT]切換。

   ![](assets/two-1.png)

   當切換設定為[!UICONTROL On]時，Web Personalization將遵循並支援瀏覽器的Do Not Track (DNT)設定，且不會追蹤任何網路活動或在您的網站上執行任何行銷活動或內容推薦。

   >[!NOTE]
   >
   >將切換設定為[!UICONTROL On]可能會影響Marketo在特定區域中的值和功能。

1. 若要停用[!UICONTROL Honor DNT]切換並忽略瀏覽器的Do No Track (DNT)設定，請在[!UICONTROL Honor DNT]下選取&#x200B;**[!UICONTROL Off]**。
