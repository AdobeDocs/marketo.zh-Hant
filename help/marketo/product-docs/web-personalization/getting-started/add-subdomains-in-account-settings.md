# 在帳戶設定{#add-subdomains-in-account-settings}中新增子網域

以下說明如何在「帳戶設定」中新增子網域至您的主要網域。 這可讓您管理與主網域的特定RTP Javascript相關的子網域。 建議將Javascript標籤部署至任何新增的子網域。

1. 在「Web個人化」中，前往「帳戶設定」**。**

   ![](assets/image2014-12-1-23-3-12.png)

1. 在「網域設定」頁面上，您會看到與您的帳戶相關聯之所有主要網域的清單。 每個部分首先列出主域（在下面突出顯示），然後列出所有子域。 按一下&#x200B;**添加子域**。

   ![](assets/highlightprimary2.png)

1. 按一下&#x200B;**添加**。

   ![](assets/add.png)

1. 輸入子網域URL。 按一下&#x200B;**添加**&#x200B;或&#x200B;**刪除**&#x200B;管理子域清單，然後在完成時按一下&#x200B;**確定**。

   ![](assets/newsubdomain.png)

1. 現在會列出您新新增的子網域。

   ![](assets/finalnew.png)

   >[!NOTE]
   >
   >如果您想要將&#x200B;*primary*&#x200B;網域新增至您的帳戶，請聯絡[行銷人員支援](https://docs.marketo.com/cdn-cgi/l/email-protection#5e2d2b2e2e312c2a1e333f2c353b2a31703d3133)。
