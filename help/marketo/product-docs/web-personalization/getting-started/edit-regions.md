# 編輯地區 {#edit-regions}

想要變更使用者的地區設定，僅查看您特定地區的資料嗎？

1. 前往 **使用者設定**.

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 按一下 **編輯地區**.

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 檢查與您所在地區相關的國家/地區或州。

>[!NOTE]
>
>選取美國時，會在頁面底部開啟所有要選取的美國選項。
