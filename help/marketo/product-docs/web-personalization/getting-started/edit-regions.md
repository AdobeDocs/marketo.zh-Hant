# 編輯區域 {#edit-regions}

想要變更您的使用者地區設定，以僅檢視您特定地區的資料嗎？

1. 前往 **使用者設定**.

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 按一下 **編輯區域**.

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 檢查與您地區相關的國家或州。

>[!NOTE]
>
>選取美國，將在頁面底部開啟所有美國州選項以供選取。
