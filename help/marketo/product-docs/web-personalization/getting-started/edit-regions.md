# 編輯區域{#edit-regions}

想要變更使用者的地區設定，只查看您特定地區的資料？

1. 前往&#x200B;**使用者設定**。

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 按一下「編輯區域」(**Edit Regions)**。

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 檢查您的國家／地區或與您的地區相關的州。

>[!NOTE]
>
>選擇美國時，會在頁面底部開啟所有要選擇的美國選項。
