# Web Personalization中的工作區 {#workspaces-in-web-personalization}

Web Personalization支援Web促銷活動和Web區段的多個工作區。

## 切換工作區 {#switch-workspaces}

若要在Web個人化中的工作區之間切換，請按一下左上方的球形圖示，然後從下拉式選單中選擇不同的工作區。

![](assets/ss7.png)

## 變更區段的Workspace {#change-a-segments-workspace}

1. 移至&#x200B;**區段**&#x200B;頁面，選取區段，然後按一下編輯圖示。

   ![](assets/ss4.png)

1. 從&#x200B;**Workspace**&#x200B;下拉式清單中選取其他工作區。

   ![](assets/ss6.png)

   ![](assets/ss5.png)

>[!NOTE]
>
>使用者將只能檢視與其有權存取的工作區相關聯的網頁行銷活動和區段。 以下說明如何[讓使用者存取一或多個工作區](/help/marketo/product-docs/administration/workspaces-and-person-partitions/allow-user-access-to-a-workspace.md)。
