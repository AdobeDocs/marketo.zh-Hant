# 使用Web個人化RTF編輯器 {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

在「設定促銷活動」頁面中找到的Web個人化RTF編輯器可允許文字格式、連結和影像插入。 它包括與標準字處理器類似的格式選項。

>網路代號使用 [人員資料](/help/marketo/product-docs/web-personalization/using-web-segments/manage-person-data.md) 從Marketo資料庫，該資料庫在「帳戶設定>資料庫」下的「Web個人化」中管理。 如果Web Token是帳戶設定>資料庫中尚未新增的新資料庫欄位，則會自動新增該Token，且最多需要24小時才能啟動。
