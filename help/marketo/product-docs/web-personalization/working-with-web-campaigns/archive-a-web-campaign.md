# 封存Web促銷活動{#archive-a-web-campaign}

1. 前往&#x200B;**Web促銷活動**。

   ![](assets/one.jpg)

1. 按一下所要Web促銷活動的狀態下拉式清單，然後選取「封存」**。**

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >封存的Web促銷活動不會顯示在預設篩選中。 要查看它們，請按一下「過濾器」表徵圖，並在&#x200B;**Status**&#x200B;下，選擇&#x200B;**Archived**&#x200B;複選框，然後按一下&#x200B;**Apply**。
