# 封存Web行銷活動 {#archive-a-web-campaign}

1. 前往 **網路行銷活動**.

   ![](assets/one.jpg)

1. 按一下所需網頁促銷活動的狀態下拉式清單，然後選取 **封存**.

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >封存的網頁促銷活動不會顯示在預設篩選器中。 若要查看，請按一下「篩選」圖示，然後在 **狀態**，請選取 **已封存** 核取方塊和按一下 **套用**.
