# 封存Web促銷活動 {#archive-a-web-campaign}

1. 前往「網 **頁促銷活動**」。

   ![](assets/one.jpg)

1. 按一下所要Web促銷活動的狀態下拉式清單，然後選取「封 **存」**。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >封存的Web促銷活動不會顯示在預設篩選中。 要查看它們，請按一下「篩選」表徵圖，然後在「狀 **態**」下，選中「已存 **檔** 」複選框，然 **後按一下**「應用」。
