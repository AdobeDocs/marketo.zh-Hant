# 優先處理Web行銷活動 {#prioritizing-web-campaigns}

當兩個或多個網路行銷活動重疊時，透過設定優先順序分數來排定您的網路行銷活動的優先順序。

>[!NOTE]
>
>**重疊的行銷活動**
>
>發生網路行銷活動重疊的情況如下：
>
>* 兩個或多個Widget和/或對話方塊行銷活動會在相同頁面上同時回應
>* 具有相同區域ID的兩個或兩個以上的In Zones會在相同網頁上同時反應
>
>In Zone行銷活動和（Widget或對話方塊）行銷活動可在相同頁面上回應。

1. 前往Web Campaigns。

   ![](assets/web-campaigns-hand-6.jpg)

   >[!NOTE]
   >
   >為了更方便找到您想要的網頁行銷活動，請使用 [篩選功能](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

1. 在「編輯行銷活動」頁面中，設定優先順序分數（9999 =最高優先順序1 =最低優先順序）。

   ![](assets/image2015-7-9-20-3a20-3a58.png)

   >[!TIP]
   >
   >我們建議您僅在可能存在行銷活動重疊時才使用行銷活動優先順序分數，且其中一個行銷活動具有較高重要性。 您不需要為每個行銷活動設定優先順序。

1. 儲存或啟動行銷活動。

1. 請參閱網頁行銷活動頁面中顯示的優先順序分數。

![](assets/web-campaign-priority-score.jpg)
