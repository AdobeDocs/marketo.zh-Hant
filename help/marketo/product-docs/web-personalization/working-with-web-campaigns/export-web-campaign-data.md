# 匯出網頁促銷活動資料 {#export-web-campaign-data}

請依照下列簡單步驟，匯出您的網頁促銷活動資料。

1. 前往 **網路行銷活動**.

   ![](assets/one-2.png)

1. 在頁面右上角，按一下「匯出CSV」圖示。

   ![](assets/two-2.png)

1. 開啟或儲存您的檔案。

   ![](assets/three-2.png)

1. 檢視您的檔案，以查看有用的統計資料。

   ![](assets/four-1.png)
