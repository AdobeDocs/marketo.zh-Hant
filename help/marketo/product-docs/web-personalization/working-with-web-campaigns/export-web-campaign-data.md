# 匯出網站行銷活動資料 {#export-web-campaign-data}

請依照下列簡單步驟匯出您的網頁行銷活動資料。

1. 前往 **網路行銷活動**.

   ![](assets/one-2.png)

1. 在頁面的右上方，按一下「匯出CSV」圖示。

   ![](assets/two-2.png)

1. 開啟或儲存您的檔案。

   ![](assets/three-2.png)

1. 檢視您的檔案以檢視有用的統計資料。

   ![](assets/four-1.png)
