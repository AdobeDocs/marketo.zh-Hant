# 複製網站行銷活動 {#clone-a-web-campaign}

使用「網站行銷活動」頁面上的複製功能，可複製行銷活動設定並變更內容以最佳化分割測試，或複製具有相同內容的行銷活動，並將其鎖定在不同的區段。 數秒內建立網站行銷活動！

## 建立複製行銷活動 {#create-a-clone-campaign}

1. 前往 **網路行銷活動**.

   ![](assets/image2016-8-18-16-3a44-3a0.png)

   >[!NOTE]
   >
   >為了更方便找到您想要的網頁行銷活動，請使用 [篩選功能](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

1. 針對網頁行銷活動，按一下 **原地複製**.

   ![](assets/web-campaigns-1-clone-hand.png)

1. 行銷活動復製程式會將現有網頁行銷活動的所有內容複製到新的複製行銷活動中。 新的複製網路促銷活動名為「[行銷活動名稱] 複製」。

   ![](assets/image2016-8-18-17-3a8-3a27.png)

   >[!TIP]
   >
   >除了分割測試外，網路促銷活動中的所有內容都會複製到複製促銷活動中，因此，如果您想要針對其他促銷活動進行測試，請勿忘記將分割測試群組新增至複製促銷活動。
