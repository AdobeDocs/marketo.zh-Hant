# 複製網頁促銷活動 {#clone-a-web-campaign}

使用「網頁促銷活動」頁面上的複製功能來複製促銷活動設定並變更內容以進行分割測試最佳化，或複製具有相同內容的促銷活動並將其定位至不同的群體。 在數秒內建立Web行銷活動！

## 建立原地複製促銷活動 {#create-a-clone-campaign}

1. 前往 **網路行銷活動**.

   ![](assets/image2016-8-18-16-3a44-3a0.png)

   >[!NOTE]
   >
   >若要更輕鬆找到您想要的網頁行銷活動，請使用 [篩選功能](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

1. 若為網路促銷活動，請按一下 **原地複製**.

   ![](assets/web-campaigns-1-clone-hand.png)

1. 促銷活動復製程式會將現有網頁促銷活動的所有內容複製到新的複製促銷活動。 新的克隆網路促銷活動的名稱為「[行銷活動名稱] 複製」。

   ![](assets/image2016-8-18-17-3a8-3a27.png)

   >[!TIP]
   >
   >Web促銷活動中的所有內容都會複製到原地複製促銷活動中，除了分割測試之外，因此，如果您想針對其他人測試，請別忘了將分割測試群組新增至原地複製促銷活動。
