# 啟動/暫停網頁行銷活動 {#launch-pause-a-web-campaign}

網路行銷活動是與特定區段關聯的自訂反應，可以是您網站上的[對話方塊](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-dialog-web-campaign.md)、區域取代[中的](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-in-zone-web-campaign.md)、Widget功能或電子郵件警示。

您可以從[!UICONTROL Web Campaigns]頁面或[!UICONTROL Set Web Campaigns]頁面以兩種方式啟動或暫停網路行銷活動。

1. 按一下「**[!UICONTROL Web Personalization]**」。

   ![](assets/one-1.png)

1. 移至&#x200B;**[!UICONTROL Web Campaigns]**。

   ![](assets/two-1.png)

1. 如果促銷活動已啟動，請按一下&#x200B;**[!UICONTROL Launched]**&#x200B;將促銷活動的狀態變更為&#x200B;**[!UICONTROL Paused]**，以暫停促銷活動。

   ![](assets/image2014-11-26-17-3a26-3a38.png)

1. 如果行銷活動已暫停，請按一下&#x200B;**[!UICONTROL Paused]**&#x200B;將行銷活動的狀態變更為&#x200B;**[!UICONTROL Launched]**。 這將會啟動並啟動行銷活動。

   ![](assets/image2014-11-26-17-3a28-3a59.png)

   >[!NOTE]
   >
   >您也可以在設定行銷活動頁面中變更行銷活動狀態。

1. 在行銷活動中，按一下&#x200B;**[!UICONTROL Edit]**。

   ![](assets/image2014-11-26-17-3a31-3a37.png)

1. 按一下&#x200B;**[!UICONTROL Save]**&#x200B;以儲存並暫停行銷活動。 按一下&#x200B;**[!UICONTROL Launch]**&#x200B;以啟動並啟動行銷活動。

   ![](assets/image2014-11-26-17-3a32-3a48.png)
