# 啟動/暫停網路促銷活動 {#launch-pause-a-web-campaign}

網路促銷活動是與特定區段相關聯的自訂反應，可以是 [對話框](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-dialog-web-campaign.md) 在您的網站上， [區域替換](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-in-zone-web-campaign.md)、介面工具集功能或電子郵件警報。

您可以從「網路促銷活動」頁面或「設定網路促銷活動」頁面，以兩種方式啟動或暫停網路促銷活動。

1. 按一下 **網頁個人化**.

   ![](assets/one-1.png)

1. 前往 **網路行銷活動**.

   ![](assets/two-1.png)

1. 如果促銷活動已啟動，請按一下 **已啟動** 將促銷活動的狀態變更為 **已暫停**，以暫停促銷活動。

   ![](assets/image2014-11-26-17-3a26-3a38.png)

1. 如果促銷活動已暫停，請按一下 **已暫停** 將促銷活動的狀態變更為 **已啟動**. 這會啟動行銷活動。

   ![](assets/image2014-11-26-17-3a28-3a59.png)

   >[!NOTE]
   >
   >您也可以在「設定促銷活動頁面」中變更促銷活動狀態。

1. 在促銷活動中，按一下 **編輯**.

   ![](assets/image2014-11-26-17-3a31-3a37.png)

1. 按一下 **儲存** 儲存及暫停促銷活動。 按一下 **Launch** 來啟動和啟動促銷活動。

   ![](assets/image2014-11-26-17-3a32-3a48.png)
