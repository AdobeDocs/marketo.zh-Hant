# 啟動/暫停網站行銷活動 {#launch-pause-a-web-campaign}

網路行銷活動是與特定區段關聯的自訂反應，可以是您網站上的[對話方塊](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-dialog-web-campaign.md)、區域取代[&#128279;](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-in-zone-web-campaign.md)中的、Widget功能或電子郵件警示。

您可以從網站行銷活動頁面或設定網站行銷活動頁面，以兩種方式啟動或暫停網站行銷活動。

1. 按一下&#x200B;**網頁Personalization**。

   ![](assets/one-1.png)

1. 前往&#x200B;**網路行銷活動**。

   ![](assets/two-1.png)

1. 如果行銷活動已啟動，請按一下&#x200B;**已啟動**，將行銷活動的狀態變更為&#x200B;**已暫停**，以暫停行銷活動。

   ![](assets/image2014-11-26-17-3a26-3a38.png)

1. 如果行銷活動已暫停，請按一下&#x200B;**已暫停**&#x200B;以將行銷活動的狀態變更為&#x200B;**已啟動**。 這將會啟動並啟動行銷活動。

   ![](assets/image2014-11-26-17-3a28-3a59.png)

   >[!NOTE]
   >
   >您也可以在設定行銷活動頁面中變更行銷活動狀態。

1. 在行銷活動中，按一下&#x200B;**編輯**。

   ![](assets/image2014-11-26-17-3a31-3a37.png)

1. 按一下&#x200B;**儲存**&#x200B;以儲存並暫停行銷活動。 按一下&#x200B;**啟動**&#x200B;以啟動並啟動行銷活動。

   ![](assets/image2014-11-26-17-3a32-3a48.png)
