# 依最新或最佳成效排序Web促銷活動 {#sort-web-campaigns-by-latest-or-top-performing}

您可以用幾種不同的方式來排序您的網頁促銷活動。

1. 前往 **網路行銷活動**.

   ![](assets/web-campaigns-hand-1.jpg)

1. 選擇排序類型。

   >[!NOTE]
   >
   >**定義**
   >
   >**最新**  — 會根據促銷活動建立日期排序。 最新的促銷活動。
   >
   >**表現最佳**  — 會根據點進率來排序促銷活動。 頂端的最高點進次數。

   ![](assets/image2016-11-4-13-3a34-3a59.png)

   是的，這真的很簡單。
