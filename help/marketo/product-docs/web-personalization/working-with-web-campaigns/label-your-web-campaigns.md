# 標籤您的Web行銷活動 {#label-your-web-campaigns}

您有那麼多的促銷活動，捲動變得麻煩了嗎？ 使用標籤來標籤促銷活動，以便您對促銷活動進行排序並快速找到。

## 新增標籤至網頁促銷活動 {#add-a-label-to-a-web-campaign}

1. 登入「Web個人化」並前往「Web促銷活動」區域。

   ![](assets/web-campaigns-hand.jpg)

   >[!NOTE]
   >
   >若要更輕鬆找到您想要的促銷活動，請使用 [篩選功能](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

1. 選取您要以標籤標籤的促銷活動。

   ![](assets/web-campaigns-label.jpg)

1. 輸入所需的標籤名稱，然後按一下「新建」。

   >[!TIP]
   >
   >如果標籤已存在，請選取該標籤，不要建立新標籤。

   ![](assets/web-campaigns-set-label.jpg)

酷！ 您現在知道如何建立標籤，並將標籤指派給行銷活動。

## 依現有標籤篩選 {#filter-by-existing-labels}

1. 在「標籤」下拉式清單中，選取您要作為篩選的標籤。

   ![](assets/web-campaigns-my-campaigns-dropdown.jpg)

1. 現在，我們只會顯示與所選標籤相關聯的促銷活動。

   ![](assets/web-campaigns-label-showing.jpg)
