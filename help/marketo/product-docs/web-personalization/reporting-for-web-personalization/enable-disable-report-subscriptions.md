# 啟用／停用報表訂閱{#enable-disable-report-subscriptions}

Web Personalization有幾份有用的報表，會透過電子郵件傳送。 以下是訂閱方式。

1. 登入Web個人化。 在您的登入名稱下，按一下「使用者設定」**。**

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. 選擇您要訂閱的報表及其頻率。 按一下&#x200B;**保存**。

   ![](assets/email-settings.png)

   就這樣！ 若要取消訂閱報表，只要取消勾選並按一下「儲存&#x200B;**」。**
