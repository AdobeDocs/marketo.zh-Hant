# 電子郵件報表 {#email-reports}

若要自訂使用者收到的自動電子郵件報表，請參閱  [「使用者設定」區段](/help/marketo/product-docs/web-personalization/getting-started/user-settings.md).

## 網站個人化電子郵件報表 {#web-personalization-email-reports}

每日或每週電子郵件報告會傳送至使用者的電子郵件地址，提供最新的組織、人員、行銷活動和資產績效資料。

可使用下列報表：

## 每日/每週組織報告 {#daily-weekly-organizations-report}

![](assets/image2014-12-6-13-3a32-3a8.png)

每日/每週報表會傳送前50個造訪組織的電子郵件給使用者，包括組織名稱、位置、造訪次數、頁面檢視以及使用的反向連結網站或搜尋詞。

## 每日/每週人員報告 {#daily-weekly-people-report}

![](assets/two.png)

每日/每週「人員報表」會根據下列資訊，從網站傳送最近100人的電子郵件給使用者：人員的電子郵件地址、組織名稱、位置、狀態、人員擷取日期，以及該人員所屬的區段。

## 表現最佳的即時行銷活動 {#top-performing-real-time-campaigns}

![](assets/image2014-12-6-13-3a32-3a31.png)

「表現最佳的即時促銷活動報表」會傳送表現最佳即時促銷活動的電子郵件給使用者，顯示促銷活動名稱、曝光數、點按次數、促銷活動反應的群體以及促銷活動的轉換率。

## 表現最佳的資產報表 {#top-performing-assets-report}

![](assets/image2014-12-6-13-3a29-3a5.png)

表現最佳的資產報表會傳送表現最佳內容資產的電子郵件給使用者，顯示資產名稱和與其他資產相比的相符百分比。

## 建議的資產報表 {#recommended-assets-report}

![](assets/image2014-12-6-13-3a28-3a43.png)

建議的資產報表會傳送電子郵件給使用者，內含內容建議引擎中顯示的內容所產生的所有內容和點按次數。

## 摘要報表 {#summary-report}

![](assets/six.png)

「摘要」報表會根據點按次數和參與個人化促銷活動或建議內容的人數（直接或協助），傳送所有促銷活動和建議內容績效的電子郵件（每月或每季）給使用者，然後成為已知人員。 報告會將結果與上個月或季度比較。

>[!NOTE]
>
>**定義**
>
>**直接轉換**:在同一次瀏覽工作階段中點按個人化促銷活動或建議內容資產的網站訪客，會繼續使用其電子郵件地址填寫網站上的任何表單。
>
>**輔助轉換**:在上次造訪期間（過去6個月內），填寫網站上的任何表單並離開其電子郵件地址的網站訪客點按了個人化促銷活動或建議的內容資產。

>[!NOTE]
>
>Marketo網站個人化會針對網站上完成的任何表單擷取網站訪客的電子郵件地址。 這會在「網站個人化人員」頁面中檢視，且是「摘要」報表中使用的人員。

若要自訂使用者收到的自動電子郵件報表，請參閱 [「使用者設定」區段](/help/marketo/product-docs/web-personalization/getting-started/user-settings.md).
