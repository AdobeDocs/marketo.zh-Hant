# 使用Google Tag Manager {#implementing-rtp-using-google-tag-manager}實作RTP

要實施RTP標籤，請遵循以下安裝說明。

1. 登入您的Google標籤管理員帳戶。

1. 新增「標籤>標籤設定>自訂HTML標籤**」。**呼叫它&#x200B;**RTP**。

1. 登入您的RTP帳戶**。**

1. 前往&#x200B;**帳戶設定**。

   a.如果您已從「支援」收到JavaScript標籤，請繼續步驟6。

   ![](assets/image2014-11-30-15-3a19-3a21.png)

1. 在「Domain（域）」下，找到相關域，然後按一下「Generate Tag **（生成標籤**）」。

   ![](assets/image2014-11-30-15-3a20-3a17.png)

1. 複製RTP JavaScript標籤並貼至您建立的新&#x200B;**自訂HTML標籤**（步驟1）。

1. 按一下「**+新增規則至觸發標籤」。**&#x200B;選擇&#x200B;**所有頁面**。

1. 按一下「儲存」**「儲存」**&#x200B;和「發佈新版本」](https://support.google.com/tagmanager/answer/2699097?hl=en)。[

1. 確認它出現在所有頁面上，包括著陸頁面和子網域。

   a.您可以在網站頁面上按一下滑鼠右鍵來完成此動作。 前往&#x200B;**檢查元素**，搜尋&#x200B;**RTP**&#x200B;以找出標籤。
