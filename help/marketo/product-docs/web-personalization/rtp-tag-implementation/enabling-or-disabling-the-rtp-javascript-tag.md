# 啟用或停用RTP JavaScript標籤 {#enabling-or-disabling-the-rtp-javascript-tag}

RTP JavaScript標籤可控制網站Personalization是否追蹤網站活動，或在您的網站上執行行銷活動或內容建議。

>[!NOTE]
>
>**不需要從您網站上的HTML程式碼中移除標籤。**&#x200B;透過帳戶設定控制它。

## 啟用或停用標籤 {#enable-or-disable-the-tag}

1. 移至&#x200B;**帳戶設定**。

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. 在「網域和網域設定」中的「標籤」下方，選取&#x200B;**關閉**&#x200B;以停用JavaScript標籤。

   ![](assets/account-settings-domain-tag.jpg)

   當標籤設為關閉時，Javascript程式碼會停用，且Web Personalization將不會追蹤任何網站活動，或在您的網站上執行任何促銷活動或內容建議。

1. 若要啟用RTP標籤，請在[網域和網域設定]中的[標籤]下，選取&#x200B;**開啟**&#x200B;切換。

   簡單。
