# Microsoft Dynamics Sync:機會同步{#microsoft-dynamics-sync-opportunity-sync}

Marketo to Dynamics sync功能強大。 以下是機會同步的所有詳細資訊：

## 如何在兩個系統之間保持機會詳細資訊的同步？{#how-are-opportunity-details-kept-in-sync-between-the-two-systems}

機會同步是Marketo的一種方式- Dynamics。 如果您在Dynamics中變更業務機會，您的更新將反映在Marketo中。

## 我是否可使用Marketo在Dynamics中創造商機？{#can-i-create-an-opportunity-in-dynamics-using-marketo}

否，您必須在Dynamics中建立商機，它會自動同步至Marketo。

## 哪些欄位將與Marketo同步？{#what-fields-will-sync-to-marketo}

您可以在設定期間選取要同步的欄位。[](https://docs.marketo.com/pages/viewpage.action?pageId=3571830#Step3of3:ConnectMicrosoftDynamicswithMarketo(Online)-SelectFieldstoSync)
