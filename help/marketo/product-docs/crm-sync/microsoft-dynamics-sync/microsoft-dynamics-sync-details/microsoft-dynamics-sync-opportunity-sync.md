# Microsoft Dynamics Sync:機會同步{#microsoft-dynamics-sync-opportunity-sync}

Marketo到動態同步的功能非常強大。 以下是機會同步的所有詳細資訊：

## 如何在兩個系統之間保持機會詳細資訊的同步？{#how-are-opportunity-details-kept-in-sync-between-the-two-systems}

機會同步是一種方式——動力與Marketo。 如果您在Dynamics中變更商機，您的更新將反映在Marketo。

## 我可以使用Marketo在Dynamics中創造商機嗎？{#can-i-create-an-opportunity-in-dynamics-using-marketo}

不，您必須在Dynamics中創造機會，並自動與Marketo同步。

## 哪些欄位會與Marketo同步？{#what-fields-will-sync-to-marketo}

您可以在設定期間選取要同步的欄位。[](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/microsoft-dynamics-365/step-3-of-3-connect.md#select-fields-to-sync)
