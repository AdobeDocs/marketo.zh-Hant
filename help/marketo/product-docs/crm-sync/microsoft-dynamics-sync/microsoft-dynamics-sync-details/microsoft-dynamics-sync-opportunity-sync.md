# Microsoft Dynamics同步：機會同步 {#microsoft-dynamics-sync-opportunity-sync}

Marketo到Dynamics同步功能超強。 以下是機會同步的所有詳細資訊：

## 如何在兩個系統之間保持機會詳細資訊的同步？ {#how-are-opportunity-details-kept-in-sync-between-the-two-systems}

機會同步是一種方式 — Dynamics到Marketo。 如果您在Dynamics中變更商機，您的更新將會反映在Marketo中。

## 我可以使用Marketo在Dynamics中建立商機嗎？ {#can-i-create-an-opportunity-in-dynamics-using-marketo}

否，您必須在Dynamics中建立商機，該商機將自動同步至Marketo。

## 哪些欄位會同步至Marketo? {#what-fields-will-sync-to-marketo}

您可以 [選取要同步的欄位](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/microsoft-dynamics-365/step-4-of-4-connect.md#select-fields-to-sync) 進行設定。
