# 促銷活動同步概述 {#campaign-sync-overview}

作為促銷活動同步的一部分，使用者可以設定自動化，以便運作為Dynamics促銷活動的成員。 符合特定行為或人口統計資格的銷售機會，可以成為Microsoft促銷活動的一部分。

>[!PREREQUISITES]
>
>更新至最新版的Dynamics外掛程式for Marketo。

可使用下列流動動作：

* 新增至Microsoft Campaign
* 從Microsoft Campaign中移除

>[!NOTE]
>
>**從Microsoft Campaign中移除** 只會移除在Marketo中使用「流量」動作新增的人員。

還有兩個觸發因素：

* 新增至Microsoft Campaign
* 從Microsoft Campaign移除

觸發器將根據來自Marketo流量動作「新增至Microsoft促銷活動」和「從Microsoft促銷活動移除」的動作執行。

>[!NOTE]
>
>確認促銷活動同步功能正常，且促銷活動在Microsoft Dynamics中未處於非作用中狀態。
