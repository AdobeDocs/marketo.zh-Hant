# Microsoft Dynamics Sync:用戶同步{#microsoft-dynamics-sync-user-sync}

您知道Marketo將您的整個資料庫與Dynamics同步嗎？ 它會同步，然後等待5分鐘，然後每天再次同步。 以下是有關Marketo如何具體處理動態帳戶的一些詳細資訊。

您將需要專屬的Microsoft Dynamics CRM使用者，以進行整合。 我們稱此用戶為同步用戶。

## 用戶詳細資訊如何在兩個系統之間保持同步？{#how-are-user-details-kept-in-sync-between-the-two-systems}

使用者同步是一種方式- Dynamics到Marketo。 如果您在Dynamics中變更使用者，變更將反映在Marketo。

## 我可以使用Marketo來建立使用者嗎？{#can-i-create-an-user-using-marketo}

不。 Marketo無法在Dynamics中建立使用者。

## 哪些欄位會與Marketo同步？{#which-fields-will-sync-to-marketo}

您可以在設定期間選取要同步的欄位。 [](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/microsoft-dynamics-365/step-3-of-3-connect.md#select-fields-to-sync)但Marketo將只同步您的Dynamics同步使用者可存取的欄位。
