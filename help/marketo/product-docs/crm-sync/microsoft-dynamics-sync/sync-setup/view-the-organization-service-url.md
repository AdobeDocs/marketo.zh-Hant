# 檢視組織服務URL {#view-the-organization-service-url}

Marketo需要組織服務URL才能與Dynamics執行個體同步。 以下說明如何在Dynamics中尋找。

1. 登入Dynamics。 按一下「設定」圖示並選取 **進階設定**.

   ![](assets/one.png)

1. 按一下 **設定** 並選取 **自訂**.

   ![](assets/two.png)

1. 按一下 **開發人員資源**.

   ![](assets/three.png)

1. 組織服務URL位於 **服務端點**.

   ![](assets/four.png)

1. 將該URL複製並貼到Marketo，然後享受其餘的同步作業。
