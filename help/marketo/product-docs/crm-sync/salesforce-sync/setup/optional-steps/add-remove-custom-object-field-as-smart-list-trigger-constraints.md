# 將自定義對象欄位添加/刪除為智慧清單/觸發器約束 {#add-remove-custom-object-field-as-smart-list-trigger-constraints}

Marketo可提供對Salesforce自訂物件同步的精細控制。 這可讓您選取自訂物件篩選條件中可用作限制的欄位，並在智慧型行銷活動中將其用作觸發器。

>[!NOTE]
>
>**需要管理權限**

1. 按一下 **管理員。**

   ![](assets/image2014-12-10-13-3a9-3a47.png)

1. 按一下 **管理** 然後 **Salesforce對象同步。**

   ![](assets/image2015-12-11-15-3a11-3a41.png)

1. **Salesforce對象同步** 顯示在左欄中。

   ![](assets/image2015-12-11-15-3a15-3a15.png)

1. 選擇要修改的對象。

   ![](assets/image2014-12-10-13-3a10-3a11.png)

1. 按一下 **編輯可見欄位**.

   >[!TIP]
   >
   >若 **編輯可見欄位** 按鈕會反灰，而物件目前正在智慧清單或智慧促銷活動中使用。 刪除所有關聯以繼續。

   ![](assets/image2014-12-10-13-3a10-3a25.png)

1. 如果已啟用全域同步，請按一下 **禁用全局同步**.

   ![](assets/image2014-12-10-13-3a10-3a36.png)

1. 勾選所需篩選/觸發限制旁的方塊，然後按一下 **儲存**.

   ![](assets/image2014-12-10-13-3a10-3a47.png)

   >[!NOTE]
   >
   >依預設，所有欄位都會選取為篩選條件限制。

1. 按一下 **欄位** 標籤來確認變更。

   ![](assets/image2014-12-10-13-3a10-3a56.png)

   >[!NOTE]
   >
   >別忘了重新啟用全局同步！

哇！ 現在您的智慧清單和智慧行銷活動擁有更強大的力量。
