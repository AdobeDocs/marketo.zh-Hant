# 讓Marketo同步使用者成為行銷使用者{#make-marketo-sync-user-a-marketing-user}

[Marketo同步使用者](/help/marketo/product-docs/crm-sync/salesforce-sync/setup/enterprise-unlimited-edition/step-2-of-3-create-a-salesforce-user-for-marketo-enterprise-unlimited.md)必須是Salesforce促銷活動同步的行銷使用者，才能正常運作。 以下是如何在Salesforce中讓使用者成為行銷使用者的方法。

>[!NOTE]
>
>**需要管理員權限**

1. 登入Salesforce。 在左搜索欄中搜索用戶，然後按一下「管理用戶」下的「**用戶**」。****

   ![](assets/image2015-7-8-14-3a25-3a49.png)

1. 尋找您的同步使用者，然後按一下其名稱。

   ![](assets/image2015-7-8-14-3a27-3a32.png)

1. 按一下&#x200B;**編輯**。

   ![](assets/image2015-7-8-14-3a29-3a7.png)

1. 勾選「**行銷使用者**」核取方塊，然後按一下「儲存&#x200B;**a3/>」。**

   ![](assets/image2015-7-8-14-3a30-3a16.png)

   太好了！ 這個Marketo同步使用者現在是行銷使用者。
