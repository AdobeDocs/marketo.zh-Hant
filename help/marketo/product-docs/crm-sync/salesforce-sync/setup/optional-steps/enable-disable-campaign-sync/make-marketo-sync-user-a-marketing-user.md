# 讓 Marketo 同步使用者成為行銷使用者 {#make-marketo-sync-user-a-marketing-user}

[Marketo同步處理使用者](/help/marketo/product-docs/crm-sync/salesforce-sync/setup/enterprise-unlimited-edition/step-2-of-3-create-a-salesforce-user-for-marketo-enterprise-unlimited.md){target="_blank"}必須是Salesforce行銷活動同步處理的行銷使用者，才能正常運作。 以下說明如何在Salesforce中將使用者設為行銷使用者。

>[!NOTE]
>
>**需要管理員許可權**

1. 登入Salesforce。 在左側搜尋列中搜尋使用者，然後按一下&#x200B;**[!UICONTROL Users]**&#x200B;下方的&#x200B;**[!UICONTROL Managing Users]**。

   ![](assets/image2015-7-8-14-3a25-3a49.png)

1. 尋找您的同步處理使用者，然後按一下其名稱。

   ![](assets/image2015-7-8-14-3a27-3a32.png)

1. 按一下「**[!UICONTROL Edit]**」。

   ![](assets/image2015-7-8-14-3a29-3a7.png)

1. 核取&#x200B;**[!UICONTROL Marketing User]**&#x200B;核取方塊並按一下&#x200B;**[!UICONTROL Save]**。

   ![](assets/image2015-7-8-14-3a30-3a16.png)

   太棒了！ 此Marketo同步使用者現在是行銷使用者。
