# 新增/移除自訂物件欄位做為智慧清單/觸發程序限制 {#add-remove-custom-object-field-as-smart-list-trigger-constraints}

Marketo Engage提供對[!DNL Veeva]自訂物件同步的特定控制。 這可讓您選取可在自訂物件篩選器中作為限制條件的欄位，並在Smart Campaigns中將其用作觸發器。

>[!NOTE]
>
>**需要管理員許可權**

1. 按一下&#x200B;**[!UICONTROL Admin]**，然後按&#x200B;**[!UICONTROL Veeva Objects Sync]**。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-1.png)

1. 選取您要修改的物件。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-2.png)

1. 按一下「**[!UICONTROL Edit Visible Fields]**」。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-3.png)

   >[!TIP]
   >
   >如果[!UICONTROL Edit Visible Fields]按鈕呈現灰色，表示物件目前正在智慧清單或智慧行銷活動中使用。 移除所有關聯以繼續。

1. 如果您的全域同步已啟用，請按一下&#x200B;**[!UICONTROL Disable Global Sync]**。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-4.png)

1. 核取所需篩選/觸發條件約束旁的方塊，然後按一下&#x200B;**[!UICONTROL Save]**。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-5.png)

   >[!NOTE]
   >
   >依預設，選取所有欄位作為篩選條件上的限制。

1. 按一下&#x200B;**[!UICONTROL Fields]**&#x200B;標籤以確認您的變更。

   ![](assets/add-remove-custom-object-field-as-smart-list-trigger-constraints-6.png)

>[!IMPORTANT]
>
>別忘了重新啟用您的全域同步處理！

現在，您的智慧列示和智慧行銷活動甚至擁有更多功能。
