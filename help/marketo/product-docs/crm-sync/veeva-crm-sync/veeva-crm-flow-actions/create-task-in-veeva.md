# 在[!DNL Veeva]中建立任務 {#create-task-in-veeva}

身為行銷人員，您擁有可協助銷售完成交易的資訊。 您可以建立任務，讓他們知道應該做什麼，以及何時應該做。

![](assets/create-task-in-veeva-1.png)

>[!NOTE]
>
>當Marketo Sync使用者建立任務時，**[!UICONTROL Due In]**&#x200B;是將在[!DNL Veeva]中建立之任務的必要欄位。 如果沒有值，Marketo預設會輸入5天。

根據預設，流程步驟將如下所示：

![](assets/create-task-in-veeva-2.png)

自訂所有欄位，以您想要的方式建立任務。

![](assets/create-task-in-veeva-3.png)

>[!TIP]
>
>您可以在[!UICONTROL Subject]和[!UICONTROL Description]中使用`{{lead.tokens}}`、`{{company.tokens}}`、`{{campaign.tokens}}`和`{{system.tokens}}`。 如需詳細資訊，請參閱流程步驟](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/use-tokens-in-flow-steps.md){target="_blank"}的[權杖。
