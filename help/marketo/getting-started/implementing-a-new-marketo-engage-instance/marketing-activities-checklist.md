# 新執行個體最佳實務：行銷活動檢查清單 {#new-instance-best-practices-marketing-activities-checklist}

行銷活動包含構成自動化行銷方案的所有資產和內容。 當您設定新的Marketo Engage執行個體時，乾淨和清晰的組織可確保所有使用者都可以輕鬆找到和管理各種計畫。

記得要[下載檢查清單](/help/marketo/getting-started/implementing-a-new-marketo-engage-instance/assets/adobe-marketo-engage-new-instance-admin-checklist.xlsx)並追蹤您的進度。
