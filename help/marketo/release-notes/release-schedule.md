# 發佈計畫 {#release-schedule}

**發佈**:每個版本的月/年。 發行說明一經提供，將連結在下表中。

**發佈日期**:部署通常從發佈周的星期二開始，大部分升級工作在週五晚上進行。 發送一封電子郵件，其中包含特定日期和發行詳細資訊。 列出的日期是星期五更新。

**狀態**:

* 按計畫 — 我們計畫在指定日期發佈季度功能和增強功能
* 正在進行中 — 我們正在開發和記錄內容，以在指定日期發佈
* 已發運 — 發放已完成

_發放日期可能更改_
