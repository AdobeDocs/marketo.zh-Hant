# 發行排程 {#release-schedule}

**版本**：每個版本的年/月。 發行說明一經發佈，即會連結至下表。

**發行日期**：推出通常從發行周的星期二開始，大部分升級在星期五晚上進行。 會傳送包含特定日期和發行詳細資訊的電子郵件。 所列日期為星期五的更新日期。

**狀態**:

* 如期推出 — 我們計畫於指定日期發行功能與增強功能
* 進行中 — 我們正在推出此版本的功能
* 已出貨 — 發行完成

_發行日期可能會有變動_
