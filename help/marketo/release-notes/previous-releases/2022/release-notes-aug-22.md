# 發行說明：2022年8月 {#release-notes-aug-22}

以下是』22年8月版本包含的所有功能。 查看您的Adobe Marketo Engage版本以了解功能可用性。

>[!AVAILABILITY]
>
>以星號表示的特徵(![星星](assets/yellow-star.png))是付費附加元件。 請連絡您的Marketo Engage代表以了解更多資訊。

下列功能開始在 **2022年8月26日**.

## 跨管道協調 {#cross-channel-orchestration}

* **[為動態聊天同時啟用/禁用所有發佈的對話框](/help/marketo/product-docs/demand-generation/dynamic-chat/dialogues/dialogue-overview.md#disable-enable-all-dialogues){target=&quot;_blank&quot;}**:按一下按鈕，從「配置」頁面一次全局啟用/禁用所有已發佈的對話框。

* **[動態聊天的自訂變身](/help/marketo/product-docs/demand-generation/dynamic-chat/configuration.md#agent-settings){target=&quot;_blank&quot;}**:上傳自訂聊天機器人頭像，使其可個人化至您的品牌。

* **動態聊天的聊天記錄**:檢視每次對話的聊天記錄，以深入了解每個網站訪客感興趣的內容。

## 新一代體驗

* **Adobe品牌推廣**:以新的Adobe Experience Cloud品牌更新編輯與人員詳細資料頁面的外觀與風格。

* **在「移動」對話框中顯示目標資料夾的資料夾層次結構**:檢視每個資料夾的資料夾階層可讓移動資產更輕鬆，並降低將資產放入錯誤資料夾的可能性。

* **[新一代體驗中的更新畫面](/help/marketo/product-docs/marketo-engage-modern-ux/toggle-switch.md){target=&quot;_blank&quot;}**:我們在新一代體驗中提供更多經過更新的畫面，提供可透過切換開關存取的更新設計和可用性增強功能：

   * 程式碼片段詳細資訊
   * 「影像和檔案」詳細資訊

>[!NOTE]
>
>例外是將資產移至行銷活動中方案內的資料夾。 此移動操作不會顯示資料夾層次結構，因為程式內的資料夾不能有重複的名稱。

## 體驗自動化 {#experience-automation}

* **[自助流程步驟 — 計畫導入增強功能](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/flow-step-service.md){target=&quot;_blank&quot;}**:改善對使用自訂流程步驟匯入程式的支援，您現在可以使用相同服務提供者的多個例項，並匯入具有與這些服務提供者相容之流程步驟的程式。

* **Munchkin — 擴充的連結追蹤**:延伸對追蹤的支援 `tel` 和 `mailto` 與Munchkin的連結，以追蹤擴充的網路行為集。

* **Webhook自訂標頭可見性**:Webhook自訂標頭現在會顯示在「管理員> Webhook」標籤中，以便更清楚顯示。

* **驗證碼**:評估表單提交的有效性 [使用reCAPTCHA v3](/help/marketo/product-docs/demand-generation/forms/using-captcha/enable-captcha-in-marketo-forms.md){target=&quot;_blank&quot;}對傳入的表單流量進行分數。 建立行銷工作流程以自動排除、隔離或刪除可疑的機器人流量。

* **核准表單的權限**:新權限，可控制哪些設計人員可以根據其他Design Studio資產批准對表單的更改。 這可防止其他設計人員在沒有其他人具有核准權限檢閱表單的情況下，將變更推送至表單。

* **匿名合併後一律執行促銷活動重播**:促銷活動重播前會發生匿名銷售機會合併，因此當匿名促銷活動重播完成時，自訂欄位篩選器會以可靠的方式運作。

## 行銷資料環境 {#marketing-data-environment}

* **修正「使用者」欄位自訂物件的UI截斷**:現在更容易識別「正在使用」的自訂物件欄位，以便您在必要時從自訂物件中刪除欄位。

## API增強功能 {#api-enhancements}

* **大量方案成員擷取API的新篩選功能**:依方案成員資格狀態、updatedAt、順序或用盡的內容篩選，以精簡擷取的資料集。

## Sales Insight {#sales-insight}

![（星號）](assets/yellow-star.png)

* **[Sales Insight與動態聊天的整合](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/dynamic-chat-integration.md){target=&quot;_blank&quot;}**:在「銷售分析」面板中，從動態聊天查看活動，並在您的勘察工作中利用此新資料點。
