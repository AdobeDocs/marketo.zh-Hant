# 發行說明： 2014年1月 {#release-notes-january}

2014年1月發行版本包含下列功能。 請檢查您的 [Marketo版本](https://www.marketo.com/pricing/) 以取得功能可用性。

## Forms 2.0 {#forms}

注意： Forms 2.0的檔案即將推出！

掌控表單建立程式，讓您的網頁開發人員稍作喘息。 Forms 2.0的設計目的是讓行銷人員能夠建立視覺和功能上強大的表單，而不需要程式設計知識。

**為您的Forms提供其應得的視覺化改觀：**

主題設計、按鈕自訂和彈性版面配置可讓您設計符合您網站外觀和感覺的現代外觀表單。

**條件式可見性與後續追蹤頁面邏輯：**

![](assets/image2014-9-22-10-3a30-3a52.png)

是否只想在使用者選取「美國」作為其「國家/地區」時顯示「州」？ 如何根據客戶在您的表格上回答問題的方式，向客戶展示不同的白皮書？ 直接從編輯器將條件式邏輯建置到您的表單中。 不需要javascript！

![](assets/image2014-9-22-10-3a31-3a54.png)

**輕鬆將Forms內嵌在您自己的登陸頁面上：**

從Marketo登陸頁面上的表單中提取html程式碼，並將它們拖放到iFrame中的日子已經一去不復返了。 只要抓取內嵌程式碼，並將其放置在您要呈現表單的登陸頁面上。 兩種模式（一般模式和燈箱）可讓您在網站上使用Marketo表單獲得更多彈性。

![](assets/image2014-9-22-10-3a38-3a2.png)

## 電子郵件程式的通訊限制 {#communication-limits-for-email-program}

[設定電子郵件程式的通訊限制](/help/marketo/product-docs/email-marketing/email-programs/email-program-actions/enable-disable-communication-limits-in-an-email-program.md) 以確保您不會過度通訊至資料庫。 如果人員超過定義的限制，她將不會收到電子郵件。

![](assets/image2014-9-22-10-3a38-3a31.png)

## 計畫會員資格分析中的其他欄位 {#additional-fields-in-program-membership-analysis}

現在您可以新增並分組您的 [計畫會籍分析](/help/marketo/product-docs/reporting/revenue-cycle-analytics/program-analytics/build-a-program-membership-analysis-report-that-lists-leads.md) 量度（依潛在客戶與公司屬性）。 例如，您可以新增產業欄位來檢視方案成員和成功的分割。

![](assets/image2014-9-22-10-3a39-3a1.png)
