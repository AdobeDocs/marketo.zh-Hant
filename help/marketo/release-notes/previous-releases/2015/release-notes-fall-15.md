# 發行說明：2015年秋 {#release-notes-fall}

秋季15發行版包含以下功能。 請查看您的Marketo版以瞭解功能可用性。

## 訂閱智慧清單 {#subscribe-to-a-smart-list}

[訂閱智慧清單](/help/marketo/product-docs/reporting/basic-reporting/report-subscriptions/subscribe-to-a-smart-list.md)

訂閱智慧清單使營銷人員能夠導出智慧清單，並將其通過電子郵件發送給不使用Marketo的利益相關方，例如銷售或電話營銷團隊。

導出可以安排為每日、每週或每月，可以具有結束交貨日期，並且可以自定義為共用有限數量的列。

![](assets/image2015-10-1-17-3a12-3a50.png)

可以在智慧清單上建立多個訂閱。 每個訂閱、跨工作區、每個Marketo實例的100個訂閱（每個訂閱10萬個銷售線索）有限制。

![](assets/image2015-10-1-17-3a11-3a50.png)

## Marketo對Google鉻的洞見 {#marketo-insights-for-google-chrome}

[Marketo對Google鉻的洞見](/help/marketo/product-docs/marketo-sales-insight/msi-chrome-plugin/using-marketo-insights-for-google-chrome.md)

我們很高興宣佈發佈Google郵件銷售真知灼見分機的更新！ 在  [Chrome商店](https://chrome.google.com/webstore/detail/marketo-insights-for-goog/jjkfbhajlmoeegbjgjipliamplidmbjb)。

此更新包括許多新功能：

* 在接洽之前，銷售人員可以直接在Google郵件中查看有關其潛在客戶的相關資訊，包括職務、twitter檔案、公司資訊、照片等。
* 銷售人員可以即時查看潛在客戶在跨渠道訪問的內容，如開啟或點擊的電子郵件、線上或現場活動、訪問的網頁、下載的電子書等。
* 通過Google郵件發送的電子郵件記錄在Marketo，並即時跟蹤。 這樣，銷售人員就可以看到潛在客戶查看其電子郵件的時間，以便他們能夠在適當的時間跟進。 MarketoSales Insight forGoogle郵件還使銷售人員能夠輕鬆地利用市場營銷建立的模板來發送漂亮的邀請、優惠和其他類型的內容。

## RTP — 內容分析和Recommendations {#rtp-content-analytics-and-recommendations}

[內容分析](/help/marketo/product-docs/web-personalization/understanding-web-personalization/understanding-content-analytics.md) 和Recommendations

RTP內容分析顯示定期Web訪問和RTP的內容建議引擎生成的訪問所產生的Web內容資產的效能。

* 查看哪些內容表現最佳，並引入最多線索
* 通過啟用RTP的預測性內容引擎中的內容自動向適當的訪問者推薦最佳內容，提高內容消費
* 深入查看每個內容資產，以查看更深入的度量、圖表和效能

RTP的「資產」頁現在被拆分為內容分析和內容Recommendations。

* **內容分析：** 顯示所有發現和定義的Web內容的視圖和直接線索，幫助您分析效能最佳的內容
* **內容Recommendations**:顯示RTP建議內容中的印象和點擊量以及相關的線索歸屬。 您還可以編輯和啟用此頁面中的內容建議 [欄](/help/marketo/product-docs/predictive-content/enabling-predictive-content/enable-the-content-recommendation-bar.md) 和 [富媒體](/help/marketo/product-docs/predictive-content/enabling-predictive-content/enable-predictive-content-for-web-rich-media.md) 建議。

* 自年初（2015年1月1日）以來，這兩頁的所有直接線索資料都已追溯更新。

## 富格文本編輯器改進 {#rich-text-editor-improvements}

我們對富格文本編輯器進行了幾項改進。 在7月份發佈更新的編輯後，我們收到了大量反饋，並能夠將這些更改應用到此升級中。 未來幾個月還有很多事情要做。 以下是第4季度新增功能清單：

* VML現在在您的HTML代碼中受支援：

```
<v:background xmlns:v="urn:schemas-microsoft-com:vml" fill="t">
<v:fill type="tile" src="<a href="https://i.imgur.com/YJOX1PC.png" rel="nofollow">https://i.imgur.com/YJOX1PC.png</a>" color="#7bceeb"/>
</v:background>
```

* 現在，任何內容都可以插入有效的HTML注釋中（以前刪除了下面所示的某些語法）:

`<!--[if gte mso 9]> <![endif]-->`

* 不將空表格單元格填充 `&nbsp;`

* 最大化/最小化添加到HTML源編輯器的按鈕
* 現在，將標識並顯示在「表屬性」對話框中
* 現在，預設情況下，兩行按鈕都顯示。
* 編輯器現在將接受任何元素（甚至不建議使用或非標準元素）:

`<myCustomElement>Hello World!</myCustomElement>`

* 編輯器現在將接受任何屬性（甚至不建議使用或非標準屬性）:

```
<myCustomElement myCustomAttribute="foo">Hello World!</myCustomElement>
<td background="someImage.png"> 
```
