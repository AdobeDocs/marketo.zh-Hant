# 發行說明：2023年3月 {#release-notes-mar-23}

在下面，您將找到2003年3月23日發行版中包含的所有功能。 查看您的Adobe Marketo Engage版功能的可用性。

>[!AVAILABILITY]
>
>用星表示的特徵(![星](assets/yellow-star.png))是付費附加項。 請聯繫您的Marketo Engage代表以瞭解詳細資訊。

## 標準發佈週期功能 {#standard-release-cycle-features}

以下功能屬於標準發佈週期，將在 **2023年3月31日**&#x200B;在隨後的幾週中分階段部署其餘功能。 版本功能和日期可能會更改。 請查看每個功能下面的狀態。

### Marketo Engage現代UX {#modern-ux}

</br>

* **元件用於登錄頁模板詳細資訊和表單詳細資訊的增強功能**:「新建由UI使用」頁籤在登錄頁模板詳細資訊和表單詳細資訊中提供附加資訊，包括有關已使用選定資產的資產的詳細資訊。 通過搜索、排序和篩選功能對資產的長清單進行分析。

## 敏捷發佈功能

以下功能遵循Agile格式，在標準發佈日期之前或之後的不同日期發佈。 請查看每個功能下面的狀態。

### 動態聊天 {#dynamic-chat}

</br>

* **用於動態聊天的對話項目報告**:提供隨時間推移的對話項目視圖，包括按訪問者類型和項目類型篩選功能，以為用戶提供更好的可見性。
