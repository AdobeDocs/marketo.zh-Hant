# 發行說明： 2023年1月 {#release-notes-jan-23}

以下是2023年1月發行版本包含的所有功能。 檢查您的Adobe Marketo Engage版本是否有功能可用。

>[!AVAILABILITY]
>
>以星號(![星形](assets/yellow-star.png))為付費附加元件。 請聯絡您的Marketo Engage代表以瞭解更多資訊。

## 標準發行週期功能 {#standard-release-cycle-features}

下列功能屬於標準發行週期，並將於以下日期開始發行 **2023年1月20日**，在接下來的幾週內分階段推出剩餘的功能。 發行功能和日期可能會有所變更。 請檢視每個功能下方的狀態。

### Marketo Engage新式UX {#modern-ux}

</br>

* **新版UX中的更新畫面**：在現代UX （先前稱為新一代體驗）中，我們提供額外的重新整理熒幕，透過切換開關提供更新的設計和可用性增強功能：

   * 程式碼片段清單檢視
   * 登陸頁面範本清單

## 敏捷發行功能

以下功能遵循敏捷格式，並在標準發行日期之前或之後的各種日期發行。 請檢視每個功能下方的狀態。

### Dynamic Chat {#dynamic-chat}

</br>

* **定義資訊擷取卡片中的挑選清單以供Dynamic Chat**：為協助確保資料整潔，您可以定義資訊擷取卡片的挑選清單，讓訪客只能從預先定義的選項清單中選取。
