# 發行說明：』17年4月增強ABM {#release-notes-april-abm-enhancements}

』17年4月ABM增強功能發行包含下列功能。 查看您的Marketo版本以了解功能可用性。

## 同步CRM對應的標準欄位 {#synching-of-crm-mapped-standard-fields}

Marketo ABM正在變更與CRM相關的行為。 今後，Marketo反彈道導彈系統在反彈道導彈系統賬戶和客戶關係管理賬戶之間建立並保持一對一關係。 這可讓Marketo保持對應的帳戶欄位與CRM同步。

## 用於CRM發現的自定義欄位 {#custom-fields-for-crm-discovery}

您現在可以新增自訂欄位至帳戶、將其對應至您的CRM，以及在Marketo中用於CRM帳戶探索。

![](assets/new-custom-field.png)

## 命名帳戶格線中的帳戶型篩選器 {#account-based-filters-in-the-named-account-grid}

您現在可以根據帳戶清單輕鬆篩選已命名的帳戶。

![](assets/named-account-filters.png)
